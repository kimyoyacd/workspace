### 2-A. 웨이크메이크 (WAKEMAKE)

#### 브랜드 정체성
- **론칭**: 2015년 (올리브영 최초 색조 PB)
- **포지션**: "컬러 전문 색조 브랜드", 퍼스널 컬러 기반 맞춤형 메이크업
- **핵심 제품**: 소프트 블러링 아이 팔레트 (3년 연속 어워즈 수상 2023~2025)
- **브랜드 무드**: 통통 튀는 컬러감, "코랄·캔디" 계열 생생한 톤
- **모델**: 르세라핌 허윤진 (코랄 컬렉션 화보)
- **해외 성과**: 일본 큐텐 아이섀도우·쿠션 카테고리 1위, 중국 티몰·와우컬러 입점

#### 현행 상세 페이지 추정 문제점

```
강점 (AS-IS)
├── 어워즈 수상 제품의 신뢰도 → 상세 페이지에 수상 배지 노출
├── 퍼스널 컬러 기반 컬러 구성 → 색상 정렬 차트 존재
└── 모델 화보 컷 확보 (르세라핌 허윤진)

약점 (AS-IS) — 설명회 발언 기반 추정
├── "노후화된 부분" (이선민 직접 발언)
│   → 2015년 론칭 이후 상세 페이지 포맷 전면 리뉴얼 미흡
│   → "컷 → 설명텍스트 → 컷 → 설명텍스트" 반복 구조
│   → 브랜드 무드와 상세 페이지 비주얼 온도 차이 발생
├── "획일적 구성"
│   → 모든 제품이 유사한 상세 페이지 포맷 사용
│   → 아이팔레트와 틴트의 상세 페이지가 구조적으로 동일
│   → 제품 특성별 맞춤 레이아웃 부재
└── 감각 연출 부족
    → 모델 컷은 있으나 발색/텍스처/레이어링 과정 시각화 미흡
    → AI 생성 연출 컷, 모션 GIF 등 신기술 활용 부재
```

#### TO-BE 방향 (S6 컨셉 인풋)
- "컬러가 말한다" 구조: 발색 클로즈업 → 퍼스널 컬러 매핑 → 룩 완성 시퀀스
- 16가지 섀도우 각각의 "텍스처 ID" 시각화 (매트/시머/글리터 구분)
- 르세라핌 허윤진 화보 컷을 도입부 풀블리드로 활용
- 모바일 세로 스크롤 기준 "한 섹션 = 한 감각 포인트" 설계

### 2-B. 컬러그램 (COLORGRAM)

#### 브랜드 정체성
- **론칭**: 2018년
- **포지션**: "플레이풀 메이크업", MZ세대 타겟 색조 맛집
- **핵심 강점**: IP 콜라보 (잔망루피 등 캐릭터 협업 한정판)
- **브랜드 무드**: 유쾌함 + 트렌디함, "일상에 특별한 뷰티 경험", 다채로운 색감
- **대표 키워드**: 땅콩 베이지부터 잔망 핑크까지, 개성 표현, 소장 가치

#### 현행 상세 페이지 추정 문제점

```
강점 (AS-IS)
├── IP 콜라보 제품은 캐릭터 비주얼 활용 → 차별화된 상세 페이지 가능
├── 다채로운 컬러 구성 → 컬러 칩 배열 자체가 비주얼
└── MZ 타겟 감성에 맞는 패키지 디자인

약점 (AS-IS)
├── 브랜드 일관성 부재 추정
│   → 일반 제품 vs IP 콜라보 제품 상세 페이지 퀄리티 격차
│   → 콜라보마다 상세 페이지 톤&매너 달라져 브랜드 신뢰도 하락
├── 설명회 발언: "트렌디·정리된 구성" 요구 (윤진희)
│   → 현재 너무 많은 요소가 나열되는 방식 추정
│   → "플레이풀"이 "정리 안 된 것"으로 읽힐 리스크
└── IP 콜라보 후 일반 라인 상세 페이지 품질 공백
    → 잔망루피 컬렉션은 감각적이나 일반 라인이 평균 이하
```

#### TO-BE 방향
- IP 콜라보 상세 페이지를 "기준 포맷"으로 격상 → 일반 라인도 동일 품질
- "컬러그램 = 색으로 오늘의 기분을 결정하는 곳" 서사 구조 통일
- 도입부: 기분/무드 선택 UI 감성 → 해당 무드에 맞는 컬러 큐레이션으로 연결
- 소장 가치 강조: 패키지 클로즈업 → 텍스처 디테일 → 컬러 발색 시퀀스

### 2-C. 필리밀리 (FILLIMILLI)

#### 브랜드 정체성
- **론칭**: 2018년
- **포지션**: "스마트 툴 전문 브랜드", 매일 만나는 뷰티 메이트
- **핵심 제품**: 메이크업 브러시, 뷰티 블렌더, 속눈썹 도구 등 미용 소품
- **어워즈**: 2025 올리브영 어워즈 미용소품 부문 1위
- **슬로건**: "Fill & Feel Millions Of Your Beauty"
- **무드**: 전문성 + 일상 편의성, 스마트하고 깔끔한 이미지

#### 현행 상세 페이지 추정 문제점

```
강점 (AS-IS)
├── 도구 브랜드 특성상 사용법 시퀀스 상세 페이지 구조화 가능
├── 어워즈 수상으로 제품 신뢰도 확보
└── 깔끔하고 정돈된 브랜드 무드 → 상세 페이지 설계에 유리

약점 (AS-IS)
├── 설명회 발언: "요즘 색조스럽게" 필요 (이선민)
│   → 도구 브랜드이나 결국 색조 메이크업과 연결되어야 함
│   → 현재 도구 스펙 나열 중심 → 메이크업 결과물 연결 미흡
├── 비주얼 온도 차이
│   → 웨이크메이크·컬러그램과 동일 플랫폼인데
│   → 필리밀리 상세 페이지만 "기능 카탈로그" 느낌
└── 색조 감성 부재
    → 브러시 털 클로즈업은 있으나 메이크업 완성 결과 연출 없음
    → "이 브러시로 어떤 메이크업이 가능한가" 감각 전달 부재
```

#### TO-BE 방향
- 도구 → 결과물 연결 구조: "이 브러시 + 이 섀도우 = 이 룩" 시퀀스
- 클로즈업 텍스처 컷(털의 밀도, 팁의 형태) + AI 메이크업 before/after
- "스마트한 도구는 감각적인 결과를 만든다" 서사 → 색조 감성 흡수

## 4. 현행 상세 페이지 공통 문제 — 3개 브랜드 Cross-Cut 분석

```
문제 구조:
┌─────────────────────────────────────────────────────────────┐
│  현재 올리브영 색조 PB 3개 브랜드 상세 페이지의 공통 패턴   │
│                                                              │
│  섹션 1: 대표 이미지 (제품 패키지 정면 컷)                  │
│  섹션 2: 브랜드 슬로건 텍스트                                │
│  섹션 3: 제품 상세 이미지 (제품 클로즈업)                   │
│  섹션 4: 컬러 스와치 나열                                    │
│  섹션 5: 성분 / 사용법 텍스트                                │
│  섹션 6: 모델 화보 컷 1~2장                                  │
│  섹션 7: 주의사항                                            │
│                                                              │
│  → 이 구조가 2015~2018년 론칭 당시 포맷에서 크게 바뀌지     │
│    않은 상태 = "노후화"의 실체                                │
└─────────────────────────────────────────────────────────────┘
```

### 문제 유형 5가지

| # | 문제 | 증거 | 영향 |
|---|------|------|------|
| **P1** | 감각이 설계되지 않았다 | 브랜드 무드와 상세 페이지 비주얼 온도 차이 | 브랜드 신뢰도 하락 |
| **P2** | 발색이 보이지 않는다 | 스와치만 있고 실제 발림감/텍스처 시각화 없음 | 구매 망설임 → 이탈 |
| **P3** | 스토리가 없다 | 컷-설명-컷-설명 나열 → 서사 없는 정보 덩어리 | 스크롤 이탈 가속 |
| **P4** | 브랜드끼리 구분이 안 된다 | 웨이크메이크·컬러그램·필리밀리가 유사한 포맷 사용 | 브랜드 정체성 희석 |
| **P5** | 모바일 최적화 미흡 | W1020 기준 PC용 레이아웃 → 모바일에서 컷이 작거나 텍스트 과다 | 모바일 이탈률 증가 |

## 6. S2 → S3 인계 정보

### S3 이데이션에서 반드시 다뤄야 할 질문

```
Q1. 웨이크메이크의 "퍼스널 컬러 전문성"을 상세 페이지에서 어떻게 보여줄 수 있나?
    (퍼스널 컬러 = 봄/여름/가을/겨울 타입 시각화)

Q2. 컬러그램의 "플레이풀"을 "정리된 구성"으로 만드는 방법은?
    (유쾌하되 어수선하지 않은 레이아웃의 해법)

Q3. 필리밀리 "도구 브랜드"가 "색조 감성"을 흡수하는 방법은?
    (도구의 전문성 + 결과물의 아름다움 연결)

Q4. 3개 브랜드 공통으로 쓸 수 있는 "감각 시각화 시스템"은 무엇인가?
    (발색 클로즈업, 텍스처 GIF, 무드 컷의 통합 포맷)

Q5. 글로벌(일본·미국) 상세 페이지와 한국판을 동시 설계하는 방법은?
    (텍스트만 교체 vs 비주얼도 현지화)
```

### S3 이데이션 출발점 3개 가설

```
가설 A: "발색이 곧 이야기다" (Color as Narrative)
→ 상세 페이지의 모든 섹션을 컬러 발색 여정으로 설계
→ 컬러명 → 무드 카피 → 발색 클로즈업 → 퍼스널 컬러 추천 → 완성 룩

가설 B: "도입부 3초가 전부다" (Hook First)
→ 상세 페이지 진입 후 3초 안에 브랜드 무드와 제품 포인트를 전달
→ 모바일 풀블리드 감각 이미지 + 한 줄 카피 = 즉각적인 감각 공명

가설 C: "체계가 감각을 만든다" (System as Sensation)
→ 텍스처 ID / 퍼스널 컬러 코드 / 무드 코드를 시각적 시스템으로 구축
→ 소비자가 스스로 자신의 제품을 찾아가는 구조
```
