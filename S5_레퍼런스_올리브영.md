# S5 Reference & Benchmark — 올리브영 색조 PB

> Stage: S5 Reference & Benchmark (Playbook v1.1)
> 목적: S6 컨셉 3안을 뒷받침할 레퍼런스 8건 선별 + 분석
> 기준: 감각 설계 / 서사 구조 / AI 활용 / 모바일 최적화 4축

---

## 레퍼런스 선별 기준

```
1. "감각이 설계된" 상세 페이지 → 브랜드 무드와 비주얼 일치
2. 발색·텍스처 시각화가 탁월한 사례
3. 서사 구조가 있는 상세 페이지 (나열이 아닌 여정)
4. AI 또는 디지털 기술 활용 사례
5. 글로벌 색조 브랜드의 모바일 최적화 사례
```

## 레퍼런스 8건

### REF-01. 롬앤 (rom&nd) — 글래스팅 컬러 글로스

| 항목 | 내용 |
|------|------|
| **URL** | romand.co.kr |
| **핵심 강점** | 컬러명이 무드 카피를 겸함. GIF 애니메이션으로 제품 질감·발색 직접 표현 |
| **레이아웃 특징** | 중립 배경(화이트/라이트그레이) + 강조색 최소화 → 제품 컬러 자체가 주인공 |
| **텍스트 스타일** | 격식+감성 혼합. "흰끼없이 쥬시한 느낌" 같은 소비자 언어를 공식 카피에 녹임 |
| **우리 적용 포인트** | GIF 텍스처 표현 + 컬러명=무드카피 구조를 웨이크메이크 팔레트에 적용 |
| **S6 연결** | Color Speaks 컨셉의 컬러 스토리텔링 방식 |

### REF-03. Charlotte Tilbury — Pillow Talk Lipstick

|------|------|
| **브랜드** | Charlotte Tilbury (글로벌 럭셔리) |

| **우리 적용 포인트** | 올리브영 브랜드 담당자(고주현·윤진희)의 "브랜드 담당자 추천" 섹션 삽입 |
| **S6 연결** | 브랜드 신뢰 구축 + 설명회 발언 활용 전략 |

| **S6 연결** | 퍼스널 컬러 매핑 서사 (B-8) 구현 방식 |
