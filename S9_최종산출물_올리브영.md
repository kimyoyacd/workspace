## 과제 시안 설계 — 디자이너 인계 브리프

### 과제 개요

```
브랜드: 웨이크메이크 (WAKEMAKE)
제품: 소프트 블러링 아이 팔레트 (대표 제품, 어워즈 3년 연속 수상)
범위: 도입부(헤더~브랜드 키비주얼) + 포인트 섹션 1개
규격: W1020px, 높이 자유
소스: RFP 제공 소스만 사용 (별도 촬영 없음)
AI 활용: 적극 허용 (연출컷, 무드 배경, 퍼스널 컬러 시뮬레이션)
컨셉: Color Speaks (컬러가 말한다) + Hook First 도입부
```

### 섹션별 설계 지시

```
[도입부 — Section 0: HOOK]
목적: 3초 안에 브랜드 무드 전달
레이아웃: W1020 풀블리드, 높이 약 600~800px

필수 요소:
  ① 배경: 팔레트 주조 컬러에서 추출한 그라데이션
     (코랄 → 와인 계열, 또는 AI 무드 배경 생성)
  ② 히어로 이미지: 팔레트 탑뷰 풀컷 (제공 소스 사용)
  ③ 타이포그래피:
     헤드라인: "컬러가 당신의 언어입니다" 또는 영문 "COLOR IS YOUR LANGUAGE"
     서브: 제품명 + 수상 배지 (2025 올리브영 어워즈)
  ④ 컬러 감각: 요즘 색조스러운 무드 = 선명하되 세련된, 과하지 않은

레퍼런스:
  - 3CE Mood Recipe: 극단적 여백 + 대형 타이포
  - 웨이크메이크 캠페인 "컬러는 웨이크메이크": 브랜드 키 컬러 활용

AI 활용 포인트:
  - 배경: AI 생성 컬러 그라데이션 또는 뷰티 무드 배경
  - 팔레트 탑뷰에 AI 라이팅 처리 (골드 림 라이팅 등)

[포인트 섹션 — Section 1: COLOR SPEAKS]
목적: 발색·텍스처가 직접 전달되는 감각 섹션
레이아웃: W1020, 높이 약 800~1200px

필수 요소:
  ① 팔레트 16구 탑뷰 + 각 컬러 넘버/이름 오버레이
  ② 텍스처 ID 시각화:
     매트 (M) / 시머 (S) / 글리터 (G) / 베이크드 (B)
     아이콘+컬러 코드 방식으로 4개 타입 구분
  ③ 퍼스널 컬러 매핑 바 (선택 추가):
     봄웜 / 여름쿨 / 가을웜 / 겨울쿨 타입별 추천 컬러 강조
     → AI 생성 퍼스널 컬러별 발색 시뮬레이션 이미지 포함
  ④ 클로즈업 컷 3개:
     매트 텍스처 클로즈업 / 글리터 텍스처 클로즈업 / 모델 눈 적용 컷

레이아웃 옵션:
  옵션 A: 팔레트 탑뷰 (풀폭) → 하단에 텍스처 ID 4개 카드
  옵션 B: 좌측 팔레트, 우측 텍스처 클로즈업 3개 그리드
  옵션 C: 팔레트 + 퍼스널 컬러 매핑 바 인터랙티브 느낌 정적 구현

레퍼런스:
  - NARS: 컬러가 직접 말하는 구조
  - Fenty Beauty: 다양성(타입별) 시각화
  - 롬앤: GIF 질감 느낌의 클로즈업
```

### 감각 검수 기준 (Section 0 + 1)

```
체크 1: "3초 안에 웨이크메이크 무드가 전달되는가?"
        → "힙하고 세련된 프로페셔널" (고주현 발언)
        → 올드하거나 진부한 뷰티 페이지 느낌은 탈락

체크 2: "발색이 직접 보이는가?"
        → 스와치 나열이 아닌 텍스처 감각이 느껴져야 함
        → "이 섀도우 어떤 느낌인지 손으로 느껴질 것 같다"

체크 3: "AI 연출컷이 부자연스럽지 않은가?"
        → AI임을 티 내지 않는 수준
        → 퀄리티 낮은 AI 이미지는 오히려 역효과

체크 4: "모바일로 봐도 감각이 전달되는가?"
        → 1020px 폭을 모바일 비율로 가상 확인
        → 텍스트가 너무 많지 않은지
```
