# S0 최적화 킥오프 팩 — 올리브영 색조 PB 상세 페이지

> 목적: 수주 성공률 극대화를 위한 사전 최적화 전략
> 기반: RFP 심층 분석 + 2차 설명회 녹취록(34분) + 제안사례 자산(17건)
> 작성: 2026.06.12 | 마감까지 D-7

## 1. RFP 기반 초격차 전략 — 킬러 로직

### 1-1. RFP 이면 읽기 — 왜 이 입찰이 열렸는가

**팩트 체인:**

```
① OY 색조 PB 3개 브랜드의 상세 페이지가 "노후화"되어 있다
   (이선민 직접 발언: "노후화된 부분을 요즘 색조스럽게 변경해 보는 감각이나 센스")

② 기존 협력사가 있었지만, 2차 설명회를 열어 신규 협력사를 물색 중이다
   → 기존 업체의 결과물에 불만족이 있다는 신호

③ "색조 브랜드다 보니까 각 업체에서 내재화되어 있는 상황이라고 많이 들었다"
   (이선민 발언) → 색조 전문 디자인 인력 확보가 시장에서 어렵다는 인식

④ 과제를 일주일 안에 요구 → 빠른 감각 검증이 목적, 장기 분석이 아니다

⑤ "우수의 협력사를 선정해서 업무를 진행" → 2개사 이상 선정 가능성
```

**이면의 진짜 니즈:**
올리브영 색조 마케팅팀은 **"색조를 아는 외부 크리에이티브 파트너"**가 필요하다. 단순 제작 외주가 아니라, 브랜드 무드를 이해하고 감각적 디자인을 자율적으로 뽑아낼 수 있는 팀. 기존 업체가 이것을 충족시키지 못했기에 재입찰이 열린 것이다.

### 1-2. 킬러 로직 — "감각 증명 선순환 구조"

경쟁사가 흉내 낼 수 없는 구조적 우위를 만드는 로직:

```
┌──────────────────────────────────────────────────────────────┐
│                                                               │
│  킬러 로직: "과제 시안이 곧 실무 첫 산출물이다"               │
│                                                               │
│  경쟁사 접근:                                                  │
│    RFP 읽기 → 제안서 작성 → 과제는 별도로 급하게 만들기       │
│    → 제안서와 과제의 품질 괴리 발생                            │
│                                                               │
│  우리 접근:                                                    │
│    RFP 디코딩 → 과제 시안을 제안서의 핵심 증거로 설계         │
│    → 제안서 Part 3에 과제 시안 프로세스를 전략의 증거로 삽입  │
│    → "이 과제가 실무에서 매달 반복됩니다"라는 메시지           │
│                                                               │
│  결과:                                                         │
│    제안서를 읽는 평가자가 과제 시안을 볼 때                    │
│    "아, 이 전략이 이렇게 나오는 거구나"가 연결된다             │
│    → 다른 업체는 제안서 따로, 과제 따로 = 분절                │
│    → 우리만 제안서-과제가 하나의 서사 = 초격차                │
│                                                               │
└──────────────────────────────────────────────────────────────┘
```

### 1-3. 킬러 로직 실행 — 3단 구조

| 단 | 내용 | 평가자 반응 목표 |
|----|------|-----------------|
| **1단: 공감** | P-1 The Real Question에서 "노후화"를 정확히 짚는다. 평가자가 속으로 "맞아"라고 고개 끄덕이게 한다. | "이 팀은 우리 문제를 안다" |
| **2단: 증거** | Part 3 Strategic Vision에서 3 Concept를 제시하고, 그 중 Hook First + Color Speaks를 **과제 시안으로 직접 증명**한다. 과제 시안 = 전략의 물리적 증거. | "말만 하는 게 아니라 보여준다" |
| **3단: 체계** | Part 4 Design Execution에서 이 과제 시안이 어떤 워크플로우에서 나왔는지, 매달 어떻게 반복되는지를 보여준다. | "이 품질이 매달 나오겠구나" |

**핵심: 제안서 35장 중 과제 시안과 연결되지 않는 슬라이드는 불필요하다.**

### 1-4. 경쟁사 대비 초격차 포인트 5가지

| # | 초격차 포인트 | 우리가 하는 것 | 경쟁사가 못 하는 이유 |
|---|-------------|--------------|---------------------|
| 1 | **과제-제안 통합 서사** | 과제 시안을 제안서 Part 3의 증거로 삽입 | 대부분 제안서와 과제를 별도 담당자가 작성 → 연결 불가 |
| 2 | **브랜드 담당자 발언 인용** | 설명회 녹취록에서 고주현·윤진희·이선민 발언을 직접 인용하여 과제 설계 근거 제시 | 녹취록을 이 수준으로 분석한 업체는 거의 없음 |
| 3 | **AI 연출 컷 실물 제시** | 과제 시안에 AI 생성 연출 컷 실제 삽입 (설명회 "너무 열려 있다" 근거) | "AI 활용 가능합니다"라고 말만 하는 업체 vs 실물을 보여주는 업체 |
| 4 | **RA 검수 선제 대응** | 화장품법·광고법 워딩 가이드를 Appendix에 미리 제시 → "이 팀은 RA 리턴을 줄여줄 것" | 대부분 RA를 몰라서 언급조차 안 함 |
| 5 | **글로벌 동시 설계** | 과제 시안을 국문·영문 동시 레이아웃으로 설계 가능함을 1장으로 보여줌 | 대부분 국문만 제출 → 영문은 "나중에 하겠다" |

## 2. 배점 항목 밀착형 R&R

### 2-1. 평가 구조 분석 (설명회 기반)

```
평가 = 정량(견적서) + 정성(포트폴리오 + 인력 투입 계획서 + 과제)
비중 = "비슷한 수준" (설명회 Q&A 참석자5 질문 → 참석자1 답변)
→ 정량:정성 ≈ 50:50 추정
```

**정성 평가 세부 배점 추정 (설명회 발언 기반):**

| 항목 | 추정 비중 | 근거 (설명회 발언) |
|------|----------|-------------------|
| **과제** | ★★★★★ 최고 | "색조다 보니까 감각적인 디자인이 많이 필요해서 부득이하게 과제를 드렸다" / "핏이 맞아서 잘 할 수 있는지 보고 싶다" |
| **포트폴리오** | ★★★★ | "포트폴리오 주실 때 AI 활용해서 나온 제작물도 있으시면 그것도 주시면 반갑게 검토" |
| **인력 투입 계획서** | ★★★ | "외주 인력에 대한 프로필 간단한 프로필과 적당한 포트폴리오까지도 준비 부탁" |

**결론: 과제 > 포트폴리오 > 인력. 리소스 배분도 이 순서.**

## 3. 리스크 선제 대응 — 허들을 기회로

### 3-1. 리스크 매트릭스

| # | 리스크 | 발생 확률 | 영향도 | 대응 전략 | 기회 전환 |
|---|--------|----------|--------|-----------|-----------|

### 3-2. 리스크 R1 상세 대응 — AI 연출 컷 파이프라인

과제 성패를 가르는 핵심 리스크이므로 상세 대응:

```
Step 1 · 소스 분석 (2h)
  제공 구글 드라이브에서 웨이크메이크 제품 컷·발색 컷 다운로드
  → 해상도·컬러 프로파일·배경 유형 분류
  → AI 활용 가능 소스 vs 불가 소스 분류

Step 2 · AI 연출 컷 생성 (4h)
  활용 도구: Midjourney / Stable Diffusion / Adobe Firefly
  생성 유형:
    a) 제품 연출 컷 — 소스 제품을 감각적 배경에 배치
    b) 텍스처 클로즈업 — 발색 소스를 확대·변형
    c) 무드 컷 — 브랜드 톤앤매너에 맞는 분위기 이미지
  주의: 발색 컷 자체를 AI로 생성하면 안 됨 (설명회 윤진희 "발색 컷은 좀 위험")
        → AI는 연출·배경·무드에만 사용, 실제 발색은 제공 소스 그대로

Step 3 · 품질 검수 (2h)
  체크리스트:
    □ 자연스러운가 (AI 티가 나면 역효과)
    □ 브랜드 톤앤매너에 맞는가
    □ 해상도 충분한가 (W1020 기준)
    □ 제품 형태가 왜곡되지 않았는가
```

```
Case C: 뷰티 경험 자체가 없는 경우
  → BX/BI + 이커머스 콘텐츠로 "디자인 역량" 증명

  → 과제 시안에 모든 것을 건다 — "과거보다 미래를 보세요"
  → 제안서 5-1에 "우리는 도메인을 학습하는 팀"이라는 메시지
```

## 5. 견적 최적화 전략

### 5-1. 단가 설계 원칙

```
원칙 1: 라이센스 내재화
  국내 온오프 + 글로벌 온라인 전체를 단가에 포함
  → 건별 협의 비용 제거 (이선민: "건마다 풀기에는 너무 어려움")
  → 경쟁사가 라이센스를 별도로 빼면 총액은 비슷하지만 관리 비용이 우리가 낮음

원칙 2: 임의 예가 범위 탐색
  올리브영 구매팀은 내부 예산 기준(임의 예가)을 가지고 있음
  → 시장 단가 리서치: 올리브영 상세 페이지 제작 단가 시장 기준 확인
  → 메인 상세 페이지 기획+촬영+디자인 풀패키지: 추정 150~300만원/건
  → 베리에이션: 메인의 40~60% 수준
  → 디자인 온리: 메인의 50~70% 수준
  주의: 추정치이며, 실제 단가는 팀 내부 원가 구조 기준으로 산출할 것

원칙 3: 하한선 사전 확정
  최저 단가 준용 대비 수용 가능 하한선을 디렉터와 사전 합의
  → "이 밑으로는 불가" 라인 명확화
  → 2개사 선정 후 단가 조정 요청 시 즉시 응답 가능하도록
```

### 5-2. 단가표 항목별 전략

| 항목 | 전략 | 이유 |
|------|------|------|
| A1. 메인 상세 페이지 (기획+촬영+디자인) | 핵심 단가 — 가장 공들여 산출 | 월 평균 제작 수량 × 단가 = 매출의 핵심 |
| A2. 베리에이션 | A1의 50~60% 수준 | 공수가 줄지만 너무 낮추면 "베리에이션도 대충 하겠다"는 인상 |
| A3/A4. 배너 (짧은/긴) | 상세 페이지 대비 20~40% | 기획+촬영 컷 1~2개 + 디자인 |
| B1~B4. 디자인 온리 | A 항목의 50~70% | 기획·촬영 없이 디자인만 → 명확히 낮춰야 |
| C1~C3. 촬영 | 촬영 유형별 차등 | 기획+촬영 > 촬영만 > 하우투 영상 |
| D. 번역 디자인 | 국문 디자인의 20~30% | 번역 제공, 레이아웃 적용만 |
