# S6 Concept Strategy — 컨셉 3안

> Stage: S6 Concept Strategy (Playbook v1.1)
> 기반: S2 진단 + S3 이데이션 + S4 리프레이밍 + S5 레퍼런스
> Bridge 워드: 감각(感覺) [디렉터 Gate 1 확정 전 초안]
> MECE 원칙: 3안 = "같은 방향(감각 설계)의 3가지 입구"

---

## 3안 개요

```
공통 방향: "감각이 설계된 상세 페이지"

1안 Color Speaks  → 컬러 자체에서 감각이 전달된다 (비주얼 중심)
2안 Mood First    → 기분이 감각의 출발점이다 (서사 중심)
3안 Sense System  → 감각을 만드는 체계가 차별점이다 (시스템 중심)
```

---

## 컨셉 1안: COLOR SPEAKS (컬러가 말한다)

### 한 줄 정의
> 상세 페이지에서 컬러가 먼저 말하고, 소비자는 그 감각을 체험한다.

### 핵심 아이디어
"설명하지 않고, 보여준다"

기존 상세 페이지: 제품 → 컬러 설명 텍스트 → 스와치
COLOR SPEAKS 방식: 컬러가 먼저 등장 → 감각 전달 → 텍스트는 확인용

### 상세 페이지 구조 (웨이크메이크 기준)

```
Section 0: HOOK (도입부)
  풀블리드 모델 화보 + 브랜드 키비주얼
  카피: "컬러가 당신의 언어입니다"
  → 3CE 방식: 극단적 여백 + 대형 타이포

Section 1: THE PALETTE
  아이팔레트 16구 펼침 컷 (탑뷰 풀블리드)
  각 컬러에 이름 라벨 오버레이
  → NARS 방식: 제품이 직접 말하는 구조

Section 2: TEXTURE ID
  16구 중 4개 선별하여 텍스처 클로즈업
  매트/시머/글리터/베이크드 = 4가지 텍스처 ID
  → 롬앤 GIF 방식: 질감이 보이는 클로즈업

Section 3: PERSONAL COLOR MAP
  봄웜/여름쿨/가을웜/겨울쿨 × 이 팔레트
  AI 생성 퍼스널 컬러별 발색 시뮬레이션
  → Fenty 방식: 다양성을 시각화

Section 4: THE MOMENT
  모델 완성 룩 화보 (르세라핌 허윤진)
  컬러명 + 무드 한 줄 카피
  예: "CORAL BURST — 빛이 나는 날의 색"

Section 5: LAYER BY LAYER
  메이크업 레이어 시퀀스 (Step 1→2→3→완성)
  각 스텝에서 이 팔레트가 쓰이는 순간

Section 6: BRAND + SPECS
  수상 내역 / 성분 인포그래픽 / 사용법
  (기능 정보는 마지막으로)
```

### 레이아웃 원칙
- 1020px 폭 기준, 섹션 0은 풀블리드(배경 확장)
- 여백 최소 60px 이상 확보
- 텍스트는 헤드라인 1줄 + 서브 2줄 이내
- 컬러 팔레트 배경색은 제품 주조색에서 추출

### 과제 시안에서의 우선 구현
```
Section 0 (HOOK) + Section 1 (THE PALETTE) + Section 3 (PERSONAL COLOR MAP)
→ 이 3섹션이 "감각 증명"의 핵심
→ AI 퍼스널 컬러 발색 시뮬레이션 = 경쟁사와의 초격차
```

### 브랜드별 적용
| 브랜드 | 컬러 언어 | 도입부 무드 |
|--------|---------|-----------|
| 웨이크메이크 | 퍼스널 컬러 스펙트럼 | "컬러가 당신을 완성한다" |
| 컬러그램 | 바이브 컬러 | "오늘의 컬러는 무엇인가요?" |
| 필리밀리 | 결과 컬러 | "이 브러시가 만드는 컬러" |

## 컨셉 2안: MOOD FIRST (기분이 먼저다)

### 한 줄 정의
> 상세 페이지가 제품 소개가 아닌, 오늘의 기분을 선택하는 경험이 된다.

### 핵심 아이디어
"소비자의 감각에서 출발한다"

기존 방식: 제품 → 소비자
MOOD FIRST 방식: 소비자 기분 → 제품이 그 기분을 완성

### 상세 페이지 구조 (컬러그램 기준)

```
Section 0: MOOD GATE (도입부)
  오늘의 기분 3가지 선택지 비주얼
  "Daily / Vibe-Up / Night"
  각 무드별 다른 컬러 팔레트 배경
  → 소비자가 "나"의 이야기로 진입

Section 1: TODAY'S COLOR
  선택된 무드에 맞는 제품 등장
  카피: "이 기분에는, 이 컬러"
  제품명 + 컬러명 + 무드 한 줄

Section 2: COLLECTION STORY
  이번 컬렉션의 배경 이야기
  (예: 잔망루피 콜라보 → 루피의 세계관 시각화)
  브랜드 세계관이 상세 페이지에 임베드

Section 3: COLOR PARADE
  해당 무드 라인업 전체 컬러 나열
  컬러 칩 + 이름 + 무드 키워드 1개
  예: "#14 잔망 핑크 — 오늘 한 번쯤은"

Section 4: HOW TO VIBE
  무드별 메이크업 적용 방법
  3스텝 시퀀스 (클리오 방식 차용)
  이 컬러로 "이 기분"을 만드는 과정

Section 5: COLLECT IT
  소장 가치 강조 섹션
  패키지 클로즈업 + 굿즈 정보
  "이번 컬렉션, 지금만 만날 수 있어요"

Section 6: BRAND + SPECS
  성분/사용법/주의사항 (마지막)
```

### 레이아웃 원칙
- 비대칭 그리드 허용 (플레이풀 감각)
- 무드 게이트(Section 0)는 강렬한 컬러 배경
- 섹션마다 배경색 전환 (무드 변화 시각화)
- 캐릭터 일러스트·패턴 요소 활용 가능

### IP 콜라보 상세 페이지 특화
```
잔망루피 × 컬러그램 적용:
Section 0: 루피 캐릭터 무드 비주얼 (일상/기분 좋은/신남)
Section 2: 루피 세계관 이야기 + 컬러 연결
Section 5: 아크릴 키링·틴트링 굿즈 소장 가치 강조
```

### 브랜드별 적용
| 브랜드 | 무드 언어 | 도입부 무드 |
|--------|---------|-----------|
| 컬러그램 | 플레이 무드 (3가지) | "오늘 어떤 기분인가요?" |
| 웨이크메이크 | 퍼스널 컬러 무드 | "당신의 컬러 계절은?" |
| 필리밀리 | 결과 무드 (어떤 룩을 원하나?) | "오늘 어떤 메이크업으로?" |

## 컨셉 3안: SENSE SYSTEM (감각의 체계)

### 한 줄 정의
> 매달 감각적인 상세 페이지를 만들어내는 "시스템"이 우리의 진짜 제안이다.

### 핵심 아이디어
"제품 하나가 아니라, 브랜드 전체의 감각을 체계로 설계한다"

기존 방식: 의뢰 → 제작 → 납품 (반복)
SENSE SYSTEM 방식: 시스템 구축 → 모듈 조립 → 감각 일관성 유지

### 시스템 3요소

```
감각 코드 시스템 (Sensation Code System)
├── 웨이크메이크 코드: SPECTRUM (스펙트럼)
│   수평 컬러 그라데이션이 핵심 비주얼 언어
│   퍼스널 컬러 매핑이 정체성
│
├── 컬러그램 코드: PLAY (플레이)
│   율동감 있는 비대칭 레이아웃
│   캐릭터·IP 요소 통합 시스템
│
└── 필리밀리 코드: PRECISION (프리시전)
    정밀하고 정돈된, 전문가의 도구 언어
    도구→결과 시퀀스 구조

AI 생산 시스템 (AI Production Pipeline)
├── 인풋: 제품명/컬러코드/텍스처타입/성분
├── 프로세스: AI 초안 자동 생성 (레이아웃+카피 초안)
├── 아웃풋: 디자이너 정제 후 납품
└── 결과: 월 납품 사이클 60% 단축

글로벌 모듈 시스템 (Global Module System)
├── 비주얼 블록: 한·일·영 공통 (이미지만)
├── 텍스트 블록: 언어별 모듈 교체
└── 결과: 3개 언어 동시 납품 가능
```

### 제안서에서의 전략적 역할
```
SENSE SYSTEM은 과제 시안의 "배경 전략"이다.

"우리가 만든 웨이크메이크 도입부 시안(과제)은
SPECTRUM 코드에 기반한 SENSE SYSTEM의 첫 번째 결과물입니다.
앞으로 매달 납품되는 상세 페이지도 같은 시스템으로 생산됩니다."

→ 과제 시안이 단순한 샘플이 아니라,
  "이 시스템의 증거"가 되는 구조
```

### 상세 페이지 구조 (3개 브랜드 공통 템플릿 기준)

```
Section 0: BRAND CODE 헤더
  브랜드별 감각 코드 그래픽 + 캠페인 메시지

Section 1: SENSATION INTRO
  제품의 핵심 감각 포인트 1가지
  풀블리드 비주얼 + 핵심 카피 1줄

Section 2: TEXTURE × COLOR
  텍스처 ID + 컬러 시각화 (브랜드별 다른 방식)
  웨이크메이크: SPECTRUM 그라데이션
  컬러그램: PLAY 그리드
  필리밀리: PRECISION 다이어그램

Section 3: APPLICATION STORY
  사용 과정 시퀀스 (3스텝)
  결과 룩 클로즈업

Section 4: GLOBAL READY
  한·일·영 텍스트 모듈 전환 가능 레이아웃

Section 5: BRAND + SPECS
  성분/사용법/수상 내역
```

### 브랜드별 감각 코드 시각 요약

| 브랜드 | 코드명 | 컬러 시스템 | 레이아웃 | 타이포 |
|--------|--------|-----------|--------|--------|
| 웨이크메이크 | SPECTRUM | 수평 그라데이션 | 대칭·정렬 | 세리프 헤드라인 |
| 컬러그램 | PLAY | 팝 컬러 블록 | 비대칭·역동 | 볼드 산세리프 |
| 필리밀리 | PRECISION | 뉴트럴+포인트 | 수직·정밀 | 모노스페이스 |
