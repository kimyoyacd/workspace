검증 게이트 (앰버)
최종 게이트 (레드)
에이전트 간 전달
Marketer
market-research
경쟁사 N개 분석 → XY 포지셔닝 맵 + 빈 자리 발견. 출처 필수.
GATE A
10항목 전부 통과해야 다음 단계로 간다. 최대 4라운드까지 재시도 가능.
design-trend
최근 3개월 글로벌 트렌드 + 약신호 포착. 출처 필수.
reference-curator
카테고리별 레퍼런스 큐레이션. 방향별로 3~5개씩 정리.
brainstorm
방향별 키워드 발산 + 이종업계 아날로지 → 비주얼·카피 3방향 도출.
Strategist
rfp-analyst
RFP → 명시·숨은·미정의·위험 4분류 + 확인 질문 목록.
concept-director
핵심 관점을 한 문장으로 정한다. 이 문장이 없으면 디자인도, 카피도 시작할 수 없다.
brand-pipeline
Manus 1 — 리서치 시장·경쟁·소비자 데이터로 브랜드 기반 잡기.
Manus 2 — 비주얼 아이덴티티 컬러·타이포·레이아웃 방향 3가지 제안.
Runable 3 — 시스템+덱 브랜드 가이드라인 + 제안 덱 최종 완성.
GATE B·C·D
각 Phase 끝마다 통과 기준 충족 확인. 통과 못 하면 해당 Phase 재실행.
review-panel
기획·크리에이터·사업·마케팅·B2B 고객 5관점 병렬 검토.
critic
비용·기간·역량 3항목 점수화(/30). 30 미만이면 진행 재검토.
Designer
moodboard
분위기·컬러·소재·타이포 등 6버킷으로 무드보드 구성. 방향별 1세트.
visual-generator
AI(Higgsfield)로 키비주얼·목업 이미지 생성. 프롬프트 → 사진급 결과물.
storyboard
영상·모션 콘티 제작. 컷별 장면·텍스트·카메라 움직임 명시.
figma-bridge
코드 ↔ Figma 양방향 연동. 컴포넌트 매핑 + 에셋 전달.
Director
design-critique
시안 6영역 검수. 🟢통과 · 🟡보완 · 🔴재작업 3단계로 판정.
creative-director
톤 최종 점검 + 온보딩 팩 생성. CD 관점에서 전체 일관성 확인.
fact-checker
사실·숫자·인용 반증 먼저. 의심스러우면 출처 확인 후 수정.
tone-guardian
발송 전 회신 톤 교정. 너무 딱딱하거나 너무 가볍지 않게.
design-system-guardian
사내 디자인 시스템 준수 점검. 색상·컴포넌트·간격 기준 체크.
delivery-gate
라이트모드·PII·글자 잘림 린트. 외부 발송 직전 마지막 점검.
GATE FINAL
전부 통과해야 외부 발송 가능. 하나라도 🔴이면 발송 중단.
Operator
project-manager
진행 현황·매출 달성율·가동률. 노션·구글시트 read-only 조회.
quote-accountant
원가표 + 마진 시나리오 산출. UI Design만 우리 실 매출로 집계.
account-radar
반복 클라 히스토리·리스크 정리. 미팅 전 브리핑으로 활용.
HR · Legal
hiring-radar
채용 파이프라인 관리. JD 작성 · 소싱 · 면접 · 오퍼 단계 추적.
vendor-radar
외주 풀 관리 + 외주비 산출. 적합한 프리랜서 매칭.
legal-compliance
계약·NDA 쟁점 플래그. 법적 자문 대체가 아닌 위험 사전 포착.
