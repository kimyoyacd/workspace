## 1. 문제 정의 (실장님 불편)

동희실장이 가져온 예시 리소스 시트는 **내부 리소스(실별 MD 배분)는 보이나**, 다음이 안 보임:

1. **스케줄상 어느 정도인지** — 언제 시작해서 언제 끝나는지
2. **병렬 진행 시 마감일 예상 업무**가 바로 안 보임
3. 리소스 시트 ↔ 스케줄 시트가 **분리·복잡** (기존 시트가 이미 많아 내부 저항 예상)

→ **하나의 시트에서 스케줄부터 예상치까지** 관리되는 버전 필요.

## 2. 예시 3종 분석 (장점만 흡수)

| 예시 | 흡수한 장점 | 버린 단점 |
|---|---|---|
| **동희실장 (VINA)** | 소재별 **실별(MAX/BXP/GXD/VINA) MD 배분** · MD환산(1h=0.125, 8h=1MD) | 마감·스케줄 부재 |
| **LINE 매직코인** | **일자별 가로 간트**(1~31일 셀에 MD 입력→자동합) · 요청일·완료일 | 실별 구분 없음(1TO 기준) |
| **밀리의 서재** | **유형별 표준 소요일**(슬림 2일/베이직 4일/스페셜 6일)→마감 예측 · 계약MM 대비 초과투입률% | 주차 단위라 데일리 마감 안 보임, 표 파편화 |

**핵심 통합 공식**:
> 동희실장의 **실별 MD 배분** + LINE의 **일자별 간트** + 밀리의 **유형별 표준소요일(마감 자동)** 을 업무 1행에 합친다. 공통 자산인 **MD 환산(1h=0.125)** 은 그대로 계승.

## 3. 시트 구조 (5탭, 전부 수식 연동)

### ① 대시보드
- 기준일·대상월(자동) / 핵심 KPI: 등록 업무·이번달 총MD·총MM·**오늘 병렬진행 건수**·🔴지연·🔴임박(D-2)·완료율
- 실·팀 가동률 요약(③ 링크) + 신호등(🔴과부하/🟡타이트/🟢여유)
- 전부 자동. 입력은 ②·④에서만.

### ② 업무마스터 (통합 간트) — 핵심
**1행 = 1업무.** 노란칸(시작일·유형·실별MD)만 입력하면 마감·D-day·간트 자동.

| 구간 | 컬럼 | 로직 |
|---|---|---|
| 업무정보 | 프로젝트·소재명·유형·규격·주관실·상태·진척% | 유형/주관실/상태 = 드롭다운 |
| 스케줄(자동) | 시작일 → **소요일**(유형별 자동 lookup) → **마감예상**(`WORKDAY`) → **D-day** → **신호**(🔴🟡🟢) | 밀리 표준소요일 계승 |
| 실별 배분 | 기획MD·디자인MD·MAX영상MD·VINA_MD → 총MD → MM | 동희실장 배분 계승 |
| 간트(자동) | 대상월 1~31일. 시작~마감 사이 `■` 자동 채움 + 파란 바 조건부서식 | LINE 간트 계승 |

**핵심 수식**
- 소요일: `=IFERROR(INDEX(설정!소요일,MATCH(유형,설정!유형,0)),"")`
- 마감예상: `=WORKDAY(시작일, 소요일-1)` (영업일 기준, 주말 제외)
- D-day: `=IF(상태="완료","완료", 마감-오늘)`
- 신호: 지연<0 / 임박≤2 / 주의≤5 / 여유
- 간트칸: `=IF(AND(날짜>=시작, 날짜<=마감),"■","")`

### ③ 실별 현황
- 실·팀별 **월 캐파(인원×1인캐파) vs 투입MD(SUMIFS) → 가동률 → 잔여 → 초과 신호**
- 진행중 건수·지연 건수 자동 카운트. 가동률 조건부서식(과부하 빨강).

### ④ 설정
- **노란칸만 입력**하면 전체 반영: 대상월·MD/MM 기준·MD환산표·**유형별 표준소요일**·실별 인원/캐파·상태값·**비나 시나리오 번호**

### ⑤ 비나 시뮬
비나 활용 4안을 금액/공수로 자동 배분:
1. **공수 %분배** — 기획→기획실, 디자인→디자인실, 영상→MAX실
2. **비나=베리만** — 영상 신규는 MAX, 사이즈베리는 디자인실 흡수
3. **이미지 TO 추가발주** — 1안 + 추가발주비 별도 라인
4. **본사형(협업)** — 비나 영상+이미지 협업, 이미지 TO 추가발주 X

## 4. 운영 방식 (데일리 관리)

- **입력은 딱 2곳**: `②업무마스터`(업무 추가) + `④설정`(기준값). 나머지는 열지 않음.
- 아침에 `①대시보드`만 보면 → 오늘 병렬 몇 건, 지연/임박 몇 건, 어느 실이 과부하인지 즉시 파악.
- 새 업무는 ②에 한 줄 추가 → 유형 선택 → 시작일·실별MD 입력 → **마감·D-day·간트 자동 생성**.
- 기존 시트 난립 문제 해소: 리소스+스케줄을 이 한 파일로 대체 (탭 5개로 압축).

## 5. 계승한 회사 표준
- MD 환산: **1시간 = 0.125MD, 8시간 = 1.0MD (근무 1일 = 1MD)**
- 월 환산: 기본 20 MD/MM (설정에서 조정 가능)
- 실 구성 기본값: 기획실 · 디자인실 · MAX실(영상) · VINA영상 · VINA이미지 (설정에서 자유 변경)
