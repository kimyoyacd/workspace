2. Form Language (형태 언어)

권장 방향

모래시계 전형의 비율(1:2:1)을 유지하되, 각 챔버의 벽 두께를 강조. 황동 링의 두께감으로 무게를 표현. 유리 내부의 광학 특성 활용(기포, 굴절) — 불완전함이 개성

피해야 할 방향

곡선 남용 (세련됨의 함정: 소재 진정성 상실). 장식 요소 추가 금지. 극단적 축소/확대 (사용 불편성 > 미감)

3. Material Selection (소재)

핵심 3가지

1. Brass (황동) — 온기, 내구성, 에이징 스토리의 정점. 매트 마감만.

2. Borosilicate Glass — 투명도 높음, 온도 변화에 강함. 기포는 불완전함이 증거.

3. Natural Sand — 입자성 강조, Slow/Fast/Heavy 시리즈별 물리 특성 다름

권장 원칙

모든 소재는 "사용할수록 변한다" 서사를 가져야 함. 제조 과정의 불규칙성을 마케팅 포인트로 (AI 완벽함 거부). 소재 투명성: 이 황동은 어디서, 이 모래는 어떤 특성.

4. Typography (타이포그래피)

서체

Display — Neue Helvetica / DM Sans

Body — IBM Plex Sans / Inter

단일 산세리프 고수. 저래그 정렬만. 자간 1.5배 이상 (호흡감).

타이포 규칙

크기 대비 최소. 색상: Deep Charcoal만. 텍스트는 "조용히 강렬하게" (침묵의 럭셔리). 자간의 여백이 호흡이 됨.

5. Photography Style

3가지 스타일 결합

A. Ambient Realism — 자연광(오후 4시), 창문 그림자, 필름 그레인, 3200K 이상

B. Through-the-Glass — 유리 내부 굴절, 반사, 매크로 기포·입자

C. Hand+Object — 손이 protagonist, 뒤집는 행위 포착

핵심 원칙

3가지 스타일을 조화. 제품 정물사진이 아니라 "행위의 기록". 불완전함이 증거 (지문, 그림자, 필름 그레인). 매크로 디테일 사용 (입자가 개별 보이는 수준).

6. Composition & Layout

여백 전략

제품 40%, 배경(여백) 60%, 손 30~40%.

광원: 한 개만 (창문). 각도: 45도 측광. 위치: 제품 왼쪽 또는 뒤쪽 (림라이트).

권장 원칙

여백이 침묵. 배경 정보 최소. 한 번에 하나의 이야기만. 시간 흐름이 보이게 (모래 낙하 중 포착). 완벽한 조명 금지 (불자연스러움).

7. Symbol System (3개 상징)

상징 정의

1. 모래 낙하 궤적 — 상부→하부 수직선. 시간의 흐름, 중력, 불가역성.

2. 비워진 공간 — Cloud Dancer 상부 구. 가능성, 새로운 시작, 선택.

3. 손의 행위 — 회전 제스처. 의도, 선택, 개인화.

디자인 규칙

기본: 24px (12px 배수). 선폭: 1.5px (가는 선). 라운드: 2px (각짐 금지). 손그려진 듯한 온기. 세 가지가 한 시스템 이루기 (일관성).

8. Motion & Behavior

모션 원칙

실시간 속도 — 슬로우모션 금지. 1배속 그대로 재생.

Loop 영상 — Seamless (첫=마지막). ASMR 콘텐츠.

전환 — Fade 3초. 스케일보다 opacity. 텍스트 애니메이션 없음.

콘텐츠 형식

Instagram Reel: 9:16, 15~30초. Website Hero: 16:9, Loop 5초. Email: GIF, 2~3초. 과다한 전환 효과 금지. 음악 오버레이 최소.

Competitive Visual Code Summary

이미 과밀한 코드

1. Minimal + White Background — HAY, Craighill 중복 (차별화 불가)

2. Productivity Language — 기능성 강조 → 럭셔리 불가

3. Heritage Craft — 과거 중심 → 젊은 세대 진입 어려움

아직 덜 쓰인 코드

1. Through-the-Glass POV — 유리 굴절 시그니처 가능

2. Sand as Texture — 입자성을 아이덴티티로

3. Hand + Ritual — 거의 없음, 감정 전달 최강

Brutalist Framework Evaluation

적용 수준: 부분 적용 (Selective Brutalism)

권장 용어: "Honest Minimalism" 또는 "Material Brutalism"

✅ 적용 가능

Exposed Structure — 황동 링 구조 노출, 유리 기포

Raw Surface — 매트 황동, 필film 그레인, 지문 자국

Typographic Force — 최소 타입, 최대 여백

❌ 비적용

Anti-Polish — 정제 필요 (거친 느낌 제외)

Functional Tension — 럭셔리 시장 가독성 우선

최종 평가: 물성 진정성만 강조, 시각적 완성도는 높게
