# GRAIN · P1 Product Design Brief
> Product Specification + Visual Direction Guide  
> 2026-06-30

---

## Overview

이 문서는 Manus 1/2의 브랜드 전략을 **구체적인 제품 설계와 비주얼 표현**으로 전환하는 P1 단계의 작업 기준이다.

**목표**: 
- 물리적 호글라스 설계 명확화
- Ritual Moment / Gravity Well 두 컨셉별 시각적 표현 방법 정립
- 제품 사진촬영 + 3D 렌더링 리소스 맵 작성

## Part 1. Product Specification (물리적 설계)

### 1.1 호글라스의 기본 사양

**제품명**: GRAIN Hourglass

| 항목 | 사양 | 근거 |
|------|------|------|
| **형태** | 대칭 구형 (또는 다각 구) | Form Language: 극소주의 기하학 |
| **높이** | 180-200mm | 손에 쥐기 적절한 스케일 |
| **지름** | 상부/하부 구 90-100mm | 모래 용량 확보 + 우아한 비율 |
| **협착점(waist)** | 8-12mm | 모래 낙하 속도 조절 지점 |
| **전체 무게** | 500-700g (모래 포함) | 럭셔리 물성감 표현 |

---

### 1.2 재질 사양

#### **A. 프레임 (황동 링 + 베이스)**

| 부위 | 재질 | 마감 | 상세 |
|------|------|------|------|
| **상부 링** | Brass (Cu 65% + Zn 35%) | Brushed (무광) | 두께 3mm, 패티나 에이징 의도 |
| **하부 링** | Brass (Cu 65% + Zn 35%) | Brushed (무광) | 상부와 동일 |
| **베이스** | 황동 또는 나무(오크/월넛) | 무광 마감 | 안정감 + 테이블 소음 최소화 |

**색상 지시**: Brushed Brass #B08D57 (사진 후보정 최소화, 실제 황동의 자연색)

#### **B. 유리 (상부 + 하부 챔버)**

| 항목 | 사양 | 근거 |
|------|------|------|
| **재질** | Borosilicate Glass (온도 내성) | Through-the-Glass POV 구현 |
| **두께** | 3mm 이상 | 무게감 표현 + 강도 |
| **투명도** | 90% 이상 (기포 자연 포함) | 불완전함의 진정성 |
| **가공** | 무광 또는 Frosted 옵션 | 완벽한 투명 금지 |
| **기포** | 제거하지 않음 | AI 완벽함 거부, 수작업 증거 |

#### **C. 모래 (3가지 시리즈)**

| 시리즈명 | 낙하속도 | 입자크기 | 색상 | 용도 |
|---------|--------|--------|------|------|
| **Slow Sand** | 45-60초 | 0.5-1mm | Amber #FDB833 | 명상, 집중 |
| **Fast Sand** | 15-20초 | 1-2mm | Warm Brown #8B4513 | 활동, 에너지 |
| **Heavy Sand** | 20-30초 | 0.5mm (밀도 높음) | Deep Amber | 의례적 무게감 |

**모래 출처**: 자연 모래 (착색 최소화, 산화성 미네랄 제거)

---

### 1.3 어셈블리 방식

1. **상부 챔버**: 유리 구 + 황동 링 가압(glue-free 또는 고급 에폭시)
2. **협착점**: 정밀한 구멍 뚫기 (CNC 가공)
3. **모래 충전**: 진공 밀폐 후 정밀 무게 계량
4. **하부 챔버**: 상부와 동일 방식
5. **베이스**: 4-point contact (테이블 안정성, 소음 최소화)

## Part 2. Visual Direction: 2개 컨셉의 사진·렌더링 전략

### 2.1 컨셉 1: Ritual Moment (전략 적합도 9.5/10)

#### **Visual System**

| 요소 | 방향 |
|------|------|
| **주인공** | 손 (손가락의 회전 행위) |
| **조연** | 호글라스 (배경 정도) |
| **배경** | 실제 사용자 책상 (office / home / café) |
| **색온도** | 3200K-4000K (오후 자연광) |
| **포토 스타일** | Ambient Realism + Hand+Object |

#### **샷 타입별 리스트**

**A. 영웅 샷 (Hero Shot)**
- 손이 호글라스를 뒤집는 순간 (정지 사진 또는 연속 프레임)
- 카메라 각도: 45도 상측면
- 초점: 손의 엄지/검지
- 배경: 책상 위 일상 물건 (연필, 책, 커피) 약간 보임
- 크롭: 손 + 호글라스 70%, 배경 30%

**B. 의례 시리즈 (Ritual Sequences)**
- 뒤집기 전 (손 위에 얹은 상태)
- 뒤집는 중 (회전 중)
- 뒤집은 후 (모래 낙하 관찰)
- 각 3-5장씩 총 9-15장

**C. 다양성 시리즈 (Diversity)**
- 손: 남성/여성, 동아시아/서구/아프리카 피부톤, 20-50대
- 환경: 회사 책상 / 홈오피스 / 카페 / 침실
- 시간대: 아침 / 점심 / 저녁 (자연광 각도 변화)

**D. 클로즈업 디테일**
- 손가락 끝과 호글라스 표면 접촉 부분 (매크로)
- 황동 표면의 미세 긁힘 (빛의 반사)
- 지문 자국 (불완전함의 증거)

**E. 과정 샷 (Process)**
- 호글라스를 집기 (손 아래)
- 들기 (손 옆면)
- 기울이기 (45도)
- 뒤집기 (역회전)

#### **촬영 기술 (Photography Technical)**

| 항목 | 스펙 |
|------|------|
| **카메라** | Full-frame (Canon 5D / Sony A7 / Nikon D850) |
| **렌즈** | 35-50mm (자연스러운 원근감) |
| **ISO** | 200-400 (자연광, 노이즈 최소) |
| **셔터** | 1/125-1/250초 (손 흔들림 방지) |
| **조리개** | f/4-f/8 (손 초점, 배경 미묘한 흐림) |
| **촉매** | 자연광만 (창문, 오후 4시 권장) |
| **반사판** | 최소화 또는 제거 (자연스러운 그림자 유지) |
| **필터** | 필름 그레인 추가 (Kodak Portra 시뮬레이션) |

#### **색상 보정 (Color Grading)**

- **white balance**: 3500K 고정 (따뜻함 유지)
- **contrast**: 중간 (명암 너무 크지 않게)
- **saturation**: -10% (자연스러운 톤)
- **shadows**: +15% (검정 너무 깊지 않게)
- **highlights**: -5% (백색 너무 밝지 않게)
- **color cast**: Warm (황색+적색 미묘하게)

#### **콘텐츠 용도**

- Instagram Feed: 1:1 정사각형 (영웅 샷, 시리즈)
- Instagram Reel: 9:16 세로 (의례 시퀀스 애니메이션)
- TikTok: 9:16 세로 (손의 행위 강조)
- Website: 16:9 가로 (배경 있는 환경 샷)
- Email: 1:1 정사각형 (단일 강렬한 이미지)

### 2.2 컨셉 2: Gravity Well (전략 적합도 9/10)

#### **Visual System**

| 요소 | 방향 |
|------|------|
| **주인공** | 호글라스 (제품 단독) |
| **조연** | 없음 (또는 최소) |
| **배경** | Cloud Dancer 무색 배경 |
| **색온도** | 4000K-5000K (스튜디오 중성 보정) |
| **포토 스타일** | Through-the-Glass + 스튜디오 조명 |

#### **샷 타입별 리스트**

**A. 정면 앙글 (Front View)**
- 카메라 각도: 수직 (90도)
- 호글라스 위치: 이미지 중앙
- 배경: 순백 또는 Cloud Dancer
- 촛점: 유리 내 모래 입자
- 용도: 제품 정의, 타이틀 뒤

**B. 측면 각도 (Side View)**
- 카메라 각도: 90도 측방향
- 초점: 협착점 + 모래 통로
- 라이팅: 림 라이트 (뒤에서 빛, 유리 테두리 강조)
- 용도: 제품의 깊이감 표현

**C. 매크로 / 클로즈업 (Macro Detail)**
- 모래 입자 개별 가시화 (macro lens 100mm+)
- 유리 표면의 기포 (bokeh와 함께)
- 황동 표면의 브러시 마크
- 광학 왜곡 (유리 굴절)

**D. Through-the-Glass POV (유리를 통한 시점)**
- 카메라를 유리 뒤에 배치
- 모래가 유리에 보이는 굴절 효과
- 배경이 유리 내에서 왜곡되어 보이는 효과
- 모래 낙하 중 포착

**E. 모래 패턴 시리즈**
- Slow Sand: 세밀한 층 (시간이 기록됨)
- Fast Sand: 큰 입자 (에너지)
- Heavy Sand: 집중된 무게 (압밀)
- 각 시리즈별 3-5장

**F. 색상 변형 (Color Variants)**
- 호글라스만 (기본)
- 호글라스 + 손 (소유의 증거)
- 호글라스 + 배경 물건 최소 1-2개 (문맥)

#### **촬영 기술 (Photography Technical)**

| 항목 | 스펙 |
|------|------|
| **카메라** | Studio 중형 또는 Full-frame (배경 균일성) |
| **렌즈** | 50-85mm (macro 옵션) |
| **ISO** | 100-200 (스튜디오 플래시) |
| **셔터** | 1/125초 (플래시 싱크) |
| **조리개** | f/5.6-f/11 (심도 깊이, 전체 초점) |
| **조명** | 3-point setup (메인+릴+백라이트) |
| **배경** | 순백 또는 Cloud Dancer 디지털 배경 |
| **삼각대** | 필수 (정확한 수직/수평) |
| **탈젠** | Lightbox 또는 Cyc 스튜디오 |

#### **조명 설정 (Lighting Setup)**

```
         ┌─ 백라이트 (Backlight)
         │  위치: 제품 뒤, 위
         │  강도: 메인의 70%
         │  목적: 유리의 광학 특성 강조
         │
[카메라] ◄──[호글라스]──┐
         │  위치: 메인 45도 좌측
         │  강도: 100% (메인)
         │  목적: 제품 형태 정의
         │
         └─ 릴 라이트 (Fill)
            위치: 카메라 우측 또는 위
            강도: 메인의 30-50%
            목적: 그림자 부드럽게
```

#### **색상 보정 (Color Grading)**

- **white balance**: 5000K-5500K (스튜디오 중성)
- **contrast**: 높음 (명확한 정의)
- **saturation**: 자연스러움 (brass 따뜻함 유지)
- **shadows**: -5% (그림자 검지 않게)
- **highlights**: 0% (유리 표면 적노출)
- **color cast**: 중성 (차갑지 않게, 따뜻하지 않게)

#### **3D 렌더링 옵션**

제품 사진이 완성되기 전, 빠른 프로토타입 비주얼로 3D 렌더링 활용:
- 렌더링 엔진: Blender Cycles 또는 Cinema 4D
- 매테리얼: Brass(brushed), Glass(realistic refraction), Sand texture
- 라이팅: 같은 3-point setup 활용
- 용도: 웹사이트 히어로, 이메일 헤더, SNS 프리뷰

#### **콘텐츠 용도**

- Website: 16:9 가로 (상세 페이지, 제품 쇼케이스)
- Instagram Feed: 1:1 정사각형 (제품 정의, 매크로)
- Instagram Stories: 9:16 세로 (모래 낙하 타임랩스)
- Email: 1:1 정사각형 (호글라스 정면)
- Print/Catalog: 3:2 또는 4:3 (상세 설명과 함께)

### 3.4 Phase 4: Video Content (선택 사항)

#### Ritual Moment 비디오

```
제목: "The Turn"
포맷: 15-30초 Reel
구성:
1. 손 closeup + 호글라스 배치
2. 천천히 회전하는 행동 (5-8초)
3. 모래 낙하 관찰 (3-5초)
4. 텍스트 오버레이: "You are choosing"
음향: 자연음 (호글라스 회전음 + 모래 미세음)
```

#### Gravity Well 비디오

```
제목: "What 30 Minutes Weighs"
포맷: 15-20초 Reel
구성:
1. 정면 호글라스 정적 (2-3초)
2. 모래 느리게 낙하 (8-10초)
3. 하부 모래 축적 (3-5초)

음향: 자연음 + 미묘한 앰비언트 톤
```

## Part 5. 품질 기준 (QC Checklist)

### 모든 이미지 필수 항목

- [ ] **색온도**: 3200-5000K 범위 (컨셉별 기준 준수)
- [ ] **초점**: 의도한 부분(손/모래/황동) 선명함
- [ ] **노출**: 백색 클리핑 없음, 그림자 정보 보존
- [ ] **필름 그레인**: 자연스러움 (과도하지 않음)
- [ ] **컬러 캐스트**: 의도한 톤 유지
- [ ] **시간**: 얼마나 오래 촬영하느냐보다 **한 장이 얼마나 강한가**

### Ritual Moment 추가 기준

- [ ] 손이 주인공으로 보임 (50% 이상 프레임)
- [ ] 호글라스와의 상호작용이 명확함
- [ ] 다양한 피부톤 포함
- [ ] 배경이 자연스럽지만 제품을 방해하지 않음

### Gravity Well 추가 기준

- [ ] 호글라스가 명확하게 정의됨
- [ ] 배경이 순수 (Cloud Dancer 또는 순백)
- [ ] 유리의 광학 특성이 표현됨
- [ ] 모래의 개별 입자가 보임 (최소 매크로 이미지)
