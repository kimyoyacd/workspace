## 1. Ritual Moment (전략 적합도 9.5/10)

**한 줄 정의**: 뒤집는 행위를 세상에서 가장 중요한 순간으로 재정의

### 무드 키워드
- 의도적 / 개인화된 / 순간적 / 신성한 / 선택

### 컬러 팔레트

| 컬러 | 코드 | 역할 |
|------|------|------|
| Brushed Brass | #B08D57 | 따뜻함, 황동의 정직함 |
| Cloud Dancer | #F0EDE6 | 중립성, 부드러운 배경 |
| Amber | #FDB833 to #8B4513 | 모래의 따뜻함 |
| Cream Angora | #F5F3F0 | 손의 피부톤과 조화 |

### 이미지 그리드 (7개 카테고리)

| # | 카테고리 | 설명 | 왜 필요한가? |
|---|---------|------|------------|
| 1 | 컬러 팔레트 샘플 | Brass + Cloud + Amber 스와치 | 따뜻한 톤의 통일성을 확인. 차가운 색 완전 배제. |
| 2 | 손 다양성 시리즈 | 나이/성별/톤이 다른 손들이 제품 잡기 | 개인화의 핵심. "누구나 주인공이 될 수 있다" 메시지. |
| 3 | 회전 중단 순간 | 손 + 모래가 함께 보이는 회전 장면 | 행위의 시간성 포착. 뒤집기의 의례화. |
| 4 | 책상 라이프스타일 | 책, 커피, 노트북 등과 함께 제품 | 개인의 일상 정황. 제품이 그곳에 자연스럽게. |
| 5 | 언박싱 순간 | 손잡이 + 패키징 개봉 장면 | 무게를 느끼는 첫 경험. 기대감과 확신. |
| 6 | 매크로 손 디테일 | 지문, 손때, 주름 | 인간의 흔적. 사용의 증거. AI 완벽함 거부. |
| 7 | 모래 낙하 흐름 | 연속 프레임 또는 시간 경과 표현 | 시간 흐름의 가시화. 영상 모션 기반. |

### 타이포그래피 방향

**서체**
- Display: Plex Sans Bold / Helvetica Neue
- Body: IBM Plex Sans / Inter

**스타일**
- 소문자 일관
- 행위 동사 강조
- 1.5배 자간 이상
- 예시: "turn it / choose / begin / your moment"

### 포토그래피 스타일

**3가지 언어의 결합:**

1. **Hand + Object**
   - 손이 주인공, 제품은 조연
   - 다양한 손의 피부톤/나이/제스처
   - 상호작용 명확함

2. **Ambient Realism**
   - 오후 4시 자연광
   - 창 그림자 자연 포함
   - 필름 그레인 유지
   - 색온도: 3200K 이상

3. **Stop-Motion Narrative**
   - 회전 중단의 순간들
   - 움직임 포착 (연속 프레임)
   - 시간의 가시화

**라이팅 원칙**
- 광원: 한 개만 (창문)
- 각도: 45도 또는 측광
- 위치: 왼쪽 또는 뒤쪽 (림라이트)
- 그림자: 자연 그림자만

### 공간/환경 참고

- **책상 스타일**: 목재/친환경 소재 선호
- **배경 요소**: 책, 커피, 노트북, 식물, 초 (명상적 요소)
- **시간대**: 오후/저녁 (따뜻한 시간)
- **국가·문화 다양성**: 아시아·북유럽·북아메리카의 다양한 책상

### 모션/시간성 암시

- **영상 속도**: 슬로우 비디오 0.7배속 (실시간보다 약간 느림)
- **효과음**: 회전음 + 모래음 (ASMR 품질)
- **Loop**: 하부 구가 상부 구로 돌아오는 순간 seamless cut
- **길이**: 15-30초 Reel 형식

### 웹·패키징·SNS 활용 방안

1. **SNS 콘텐츠 전략**
   - UGC 캠페인 #MyRitual
   - 사용자의 실제 손과 책상 사진
   - 지역별·문화별 다양한 의례 표현

2. **웹 히어로**
   - 영상 배경 (슬로우 회전 모션)
   - 텍스트 오버레이 카피
   - 여백 50% 이상 (호흡감)

3. **패키징**
   - Cloud Dancer 박스
   - 손잡이 디자인 강조
   - 구매자 이름 각인 옵션 (개인화)

4. **이메일 시리즈**
   - 매주 다른 사용자의 의례 소개
   - 컨텍스트 스토리 포함
   - 30-50대 타깃에 어필

5. **인플루언서 협업**
   - 각자의 ritual 보여주기
   - 짧은 영상 (15초)
   - 위치/시간/의례 다양성

## 2. Gravity Well (전략 적합도 9/10)

**한 줄 정의**: 물질의 무게감이 만드는 시각적 질서. 중력이 시간을 만든다.

### 무드 키워드
- 물리적 / 무거움 / 결정적 / 침묵 / 건축적

### 컬러 팔레트

| 컬러 | 코드 | 역할 |
|------|------|------|
| Brushed Brass | #B08D57 | 기본 톤, 황동의 따뜻함 |
| Charcoal | #2B2B2B | 깊이와 침묵 |
| Cloud Dancer | #F0EDE6 | 경계와 배경 |
| Deep Black | #1A1A1A | 옵션 악센트 |

### 이미지 그리드 (7개 카테고리)

| # | 카테고리 | 설명 | 왜 필요한가? |
|---|---------|------|------------|
| 1 | 컬러 팔레트 샘플 | Brass + Charcoal + Cloud 스와치 | 차가운 톤의 무게감. 관계성 표현. |
| 2 | 매크로 모래 극접사 | 개별 모래알의 기하학적 질서 | 개별 입자 = 개별 시간 단위의 증거. |
| 3 | Through-the-Glass 구도 | 유리 통한 굴절 + 기포 + 왜곡 | 투명도 vs 불완전함의 대비. |
| 4 | 제품 정면 기하학 | 건축적 그리드 + 수직 중심축 | 중력의 당김을 시각화. |
| 5 | 황동 표면 매크로 | 긁힘, 패티나, 사용의 이력 | 무거움이 시간을 새기는 증거. |
| 6 | 유리 내 기포 클로즈업 | 미세한 기포들의 집합 | 불완전함이 개성. 과학적 아름다움. |
| 7 | 모래층 단면 표현 | 수평 지층처럼 쌓인 모래 | 지질학적 시간의 비유. 침전의 구조. |

### 타이포그래피 방향

**서체**
- Display: Helvetica Neue Bold / DM Sans
- Body: IBM Plex Sans / Inter (최소 텍스트)

**스타일**
- 왼쪽 정렬만 사용
- 극대 자간 (2배 이상)
- 명사/형용사 중심
- 예시: "weight / proof / the sand / arrives / gravity"

### 포토그래피 스타일

**3가지 언어의 결합:**

1. **Through-the-Glass**
   - 유리 내부 굴절이 보이는 각도
   - 왜곡: 유리가 빛을 굴절시켜 배경이 왜곡됨
   - 반사: 유리 표면 반사 + 내부 투명도 대조
   - 매크로: 기포, 내부 모래 입자 개별 가시

2. **Macro Detail**
   - 극접사 렌즈 (100mm+)
   - 모래 입자 개별 화
   - 기포와 bokeh 효과
   - 황동 표면의 브러시 마크

3. **Architectural Precision**
   - 정면 또는 극측 구도
   - 기하학적 정밀성
   - 엄격한 음영 (soft shadow 최소)
   - 표면의 미세 변화 강조

**라이팅 원칙**
- 정면 또는 사이드라이트
- 엄격한 음영
- 반사광 최소
- 백라이트로 유리의 광학성 강조

### 공간/환경 참고

- **배경**: Cloud Dancer 또는 Deep Charcoal 무광 배경
- **배치**: 완벽한 대칭 또는 정면 중심
- **소품**: 최소화 (책상/손 배제, 제품만)
- **분위기**: 갤러리/박물관 느낌. 침묵.

### 모션/시간성 암시

- **영상 속도**: 1배속 실시간 (슬로우모션 금지)
- **효과음**: 모래음 + 중력감 (ASMR: 부드러운 낙하음)
- **구도**: 상부→하부 중력의 방향성을 강조하는 수직 카메라
- **길이**: 10-20초 minimal loop

### 웹·패키징·SNS 활용 방안

1. **SNS 콘텐츠 전략**
   - 매크로 이미지 라이브러리 (1000개+ 제작 가능)
   - 아트 필터 협업 (모래 입자를 미술관 전시처럼)
   - 과학 콘텐츠 (모래의 물리학, 중력 개념)

2. **웹 히어로**
   - 제품만 노출 (손 없음)
   - 여백 60% 이상 (침묵의 미학)
   - 정적 이미지 또는 매우 느린 루프

3. **패키징**
   - 검정 무광 박스 (무게감)
   - 검정 벨벳 내부
   - 제품 자체가 유일한 언어

4. **이메일**
   - 과학 콘텐츠 (모래 물리학)
   - 세계 여러 모래 비교
   - 뉴턴 법칙과의 연계

5. **프리미엄 채널**
   - 갤러리/박물관 전시 스타일
   - 미술 잡지 협업 (Wallpaper*, Kinfolk)
   - 학술 출판물 참고

## 3. 두 컨셉의 전략적 역할

### Unified Message (통합 메시지)

> **"We don't sell hourglasses. We sell the weight of a moment."**

### Dual Positioning

| 차원 | Ritual Moment | Gravity Well |
|------|--------------|-------------|
| **감정** | 개인화, 의도, 선택 | 물리, 증거, 질서 |
| **시각** | 손 + 책상 + 다양성 | 제품 + 극접사 + 정밀성 |
| **채널** | Instagram/TikTok, UGC | Website, Print, Magazines |
| **톤** | 따뜻함, 친근함 | 차가움, 침묵 |
| **고객** | 자기표현 추구층 | 오브제 수집가 |

### 함께하는 역할

1. **Ritual Moment**가 "당신의 선택"을 증명하면,
2. **Gravity Well**이 "모래가 증거"임을 증명한다.
3. 둘 다 **"시간의 무게"** 라는 핵심가치를 언어별로 표현한다.

## 4. 무드보드 구성을 위한 이미지 생성 프롬프트

### Ritual Moment 이미지 생성 프롬프트

```
1. Color palette - Brushed Brass, Cloud Dancer, Angora, Amber swatches
2. Woman's hands of different ages holding brass hourglass, turning gesture, warm afternoon light, natural wood desk
3. Close-up of hands flipping hourglass in stop-motion moment, sand particles clearly visible, warm studio light
4. Unboxing moment - hand opening cloud dancer box, brass hourglass visible inside, precious reveal
5. Minimal desk scene with hourglass, wooden surface, afternoon light from left window, warm film grain, books nearby
6. Multiple hands (diverse skin tones, ages) holding hourglass in different gestures, anthropological style
7. Abstract flowing sand visualization, particles falling through light, time concept, warm color grading
```

### Gravity Well 이미지 생성 프롬프트

```
1. Color palette - Brushed Brass, Charcoal, Cloud Dancer, Deep Black swatches
2. Macro photography of sand grains inside glass chamber, extreme close-up, geometric precision, minimal background
3. Hourglass shot through glass lens perspective, refracted view, distorted but transparent, light study
4. Architectural product photography - hourglass centered, geometric composition, strong shadows, studio lighting
5. Macro detail of brass surface, fine scratches and patina visible, tactile texture, raking light
6. Glass chamber with sand layers, through-the-glass perspective, bubble details, scientific precision
7. Overhead composition with geometric elements, hourglass as focal point, minimal palette, silent stillness
```
