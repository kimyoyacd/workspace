# GRAIN v2 · 폼·소재 확장 브레인스토밍 — 3단 적층·원목·캡슐형 변수 발산

## 1. 빠른 발산 — 변수별 아이디어 (각 8개 이상)

### 폼(Form) — 벌브 개수·형태
1. 2단 전통형 — 상하 2벌브, 잘록한 허리 (기존 유지안)
2. 3단 적층형 — 상·중·하 3벌브, 허리 2군데 (크리스탈 스택 램프 모티프)
3. 3단이지만 비대칭 — 위/아래는 크고 가운데만 작은 "구슬 하나 끼운" 느낌
4. 완전 알약형 — 벌브 자체가 캡슐 실루엣, 허리 없이 매끈하게 이어붙임
5. 4단 이상 — 아예 염주처럼 여러 개 이어 붙여 "무한 반복" 암시 (파괴 과함 주의)
6. 전통 원형 벌브 + 캡슐형 벌브 혼합 — 위는 둥근 전통형, 아래는 캡슐형으로 폼 자체가 "거짓말"
7. 벌브가 물성 모티프 그대로 왜곡된 실루엣 (Slow=말랑하게 처진 형태, Fast=표면장력으로 뾰족, Heavy=스파이크가 실루엣까지 침범)
8. 3단 스택인데 가운데 벌브만 회전 가능(키네틱) — 소리·움직임과 결합
9. 2단인데 허리를 극단적으로 얇게 — 거의 끊어질 듯한 실낱 허리
10. 벌브 크기 위아래 비대칭(위 작고 아래 큼, 혹은 반대) — 무게중심 왜곡

### 소재(Material) — 데코/캡
11. 원목(내추럴) — 월넛/오크/애쉬, 오일 피니시
12. 원목이지만 그을림·번트우드(burnt) 마감 — 헤비 라인과 결합 시 효과적
13. 브라스/스틸/크롬 (기존안, 유지 옵션)
14. 무광 블랙 스톤/세라믹 — 레퍼런스③(블랙볼+금박) 모티프
15. 스톤+금박 포인트 — 무광 블랙 스톤 캡에 얇은 금박 라인 하나만
16. 원목+메탈 인레이 — 나무 안에 얇은 금속 링 삽입 (자연+인공의 이중성)
17. 콘크리트/시멘트 캡 — 거칠고 무거운 산업적 대비
18. 반투명 레진 캡 (벌브와 동일 계열 소재로 캡까지 통일)

### 마감(Finish) — 벌브 표면
19. 유광 투명(기존안 유지)
20. 반투명 그라데이션(알약처럼) — 레퍼런스②, 색이 위→아래로 옅어짐
21. 완전 무광(matte) — 표면 반사 없이 촉각적 질감만 강조
22. 무광인데 내부는 훤히 비침(외부 매트 코팅 + 내부 투명) — "겉은 숨기고 속은 보여줌"
23. 프로스티드 글라스(서리) — 반투명이지만 그라데이션 없이 균일한 뿌연 톤
24. 마감이 라인마다 다름 — Slow=무광, Fast=고광택, Heavy=반유광(세미매트)

## 3. 확장 추천 3선

### 추천① "3단 적층 + 원목 캡 + 세미매트 벌브"
크리스탈 스택 램프의 3단 구조를 가져오되, 금속 대신 원목 캡으로 바꿔 "차가운 변형"과 "따뜻한 자연" 사이 긴장을 만든다. 벌브는 완전 유광도 완전 무광도 아닌 세미매트로 "빛을 다 주지도 다 숨기지도 않는" 애매함을 유지.
- **핵심 키워드**: 3단 리듬 / 원목 온기 / 반신뢰 표면 / 자연×초자연 대비 / 콜렉터 오브제감
- **한 줄 포지셔닝**: 경쟁 아트토이가 전부 플라스틱/레진 단일 소재일 때, GRAIN만 "손에 잡히는 원목 + 믿을 수 없는 내용물"의 이중 신뢰 구조를 판다.

### 추천② "캡슐형 반투명 그라데이션 벌브 + 무광 스톤 캡"
벌브 자체를 알약처럼 매끈한 캡슐 실루엣으로 바꾸고 내부 물성이 그라데이션으로 비치게 한다("500mg" 같은 각인 대신 라인별 물성 이름을 미세 각인 — 예: "JELLY 03"). 캡은 무광 블랙 스톤으로 무게감을 잡아 캡슐의 가벼움과 대비.
- **핵심 키워드**: 알약 실루엣 / 의약품 패러디 / 그라데이션 투명도 / 무광 스톤 무게감 / "복용하는 시간"
- **한 줄 포지셔닝**: "시간을 재는 도구"가 아니라 "시간을 처방받는 오브제"라는, 경쟁사 누구도 안 쓴 의약품 메타포를 정면으로 가져간다.

### 추천③ "비대칭 3단 + 원목+메탈 인레이 + 라인별 마감 분화"
가운데 벌브만 작게 낀 비대칭 3단 구조로 "완벽한 좌우대칭"이라는 기존 모래시계 문법 자체를 배신. 캡은 원목이되 안에 얇은 금속 링을 심어 자연과 인공이 물리적으로 결합. 마감은 라인마다 다르게(Slow=무광/Fast=고광택/Heavy=세미매트) 가져가 "같은 제품군인데 각 라인이 서로 다른 진실을 말한다"는 서사 강화.
- **핵심 키워드**: 비대칭 리듬 / 자연×금속 인레이 / 라인별 마감 분화 / 컬렉션 다양성 / 시각적 불균형
- **한 줄 포지셔닝**: 하나의 마감으로 통일하는 대신, 3개 라인이 서로 다른 표면 언어를 쓰게 함으로써 "컬렉션 전체가 하나의 진실을 말하지 않는다"는 브랜드 관점을 형태 자체에 새긴다.

## 4. 폼×소재×마감 조합 매트릭스 (6~10개)

## 5. 상위 추천 3개 (매트릭스 중)

1. **#7 "비대칭 3단(가운데 캡슐형) + 원목 캡 + 라인별 마감 분화"** — 폼·소재·마감 세 변수가 각각 "하나로 고정되지 않는다"는 메시지를 동시에 말해서 Gate1 관점과 가장 밀도 높게 결합. 다만 구조가 복잡해 촬영·양산 검토 필요.
2. **#1 "3단 적층 + 원목 캡 + 무광 젤리"** — 리스크 낮고 즉시 시각화 가능, Slow 라인 킥오프 샘플로 최적. 원목의 온기가 "젤리가 진짜 살아있는 유기물"이라는 착시를 가장 자연스럽게 보조.
3. **#4 "비대칭 3단 + 원목+금박 인레이 + 유광"** — 레퍼런스③의 금박 포인트를 그대로 살리면서 원목과 결합해 "내추럴 럭셔리"라는 새 포지션 확보, 콜렉터 타깃에게 소장가치 어필력 최고.

## 6. 약식 포지셔닝 맵

**X축**: 자연(원목·유기적 톤) ↔ 인공(메탈·크롬·합성)
**Y축**: 투명(내부가 다 보임) ↔ 은폐(무광·불투명, 내용물을 숨김)

```
          투명
           │
  #2 크롬+   │   #10 원목+
  그라데이션 │   메탈인레이+유광
           │
자연 ──────┼────────── 인공
           │
  #1 원목+  │   #6 스톤+
  무광젤리  │   무광(4단 과함)
           │
          은폐
```

**빈자리**: "자연×은폐" 사분면 안쪽 깊은 곳 — 원목이면서 완전 무광, 게다가 벌브까지 프로스티드로 내용물이 거의 안 보이는 극단 버전은 아직 아무도 안 채움. 지금까지 GRAIN은 "투명해서 물성이 보여야 배신감이 산다"는 전제가 있었는데, 반대로 "안 보여서 더 의심스럽다"는 버전은 미개척 — Gate1의 "불신(Uncanny Doubt)"을 오히려 형태 정보 차단으로 만드는 실험적 방향.

## 7. 기존 3라인 매핑과의 관계

- **강화되는 지점**: 소재 변수(원목)와 마감 변수(무광/그라데이션)는 3라인 컨셉 자체를 바꾸지 않고 "포장"만 바꾸는 변수라서 Slow/Fast/Heavy 정체성과 충돌 없음. 오히려 라인별 마감 분화(#7, #9 아이디어)는 "왜 이 세 개가 다른 라인인가"를 형태로도 설명해줘서 스토리텔링이 강화됨.
- **잠재 충돌 지점**: Fast 라인(리퀴드크롬=차갑고 금속적)에 원목 캡을 그대로 붙이면 톤 충돌 위험(조합#2 참고). Fast는 크롬/스틸 캡을 유지하고, 원목은 Slow·Heavy 쪽에 우선 배치하는 게 안전할 수 있음 — 즉 "원목 전면 적용"보다 "라인별로 소재를 다르게 배정"하는 쪽이 기존 3라인 논리와 더 잘 맞음.
- **구조 변수(2단→3단)는 3라인과 독립적** — 몇 단인지는 라인 구분과 무관하게 전체 컬렉션 공통 폼 언어로 가져가도 되고, 혹은 "Heavy만 3단(무게감 강조), Slow·Fast는 2단 유지"처럼 라인별로 다르게 갈 수도 있음 — 이 부분은 Director 판단 필요.

## 8. 다음 단계 질문

1. **원목 소재를 3라인 전체 공통 캡으로 갈지, 아니면 Slow·Heavy에만 적용하고 Fast는 기존 메탈 계열을 유지할지** — 톤 충돌 우려(위 #7 참고) 때문에 결정이 필요합니다.
2. **3단 구조를 컬렉션 전체 공통 폼으로 확정할지, 아니면 라인별로 단 수를 다르게(예: Heavy만 3단) 가져가서 "라인마다 형태 문법 자체가 다르다"는 메시지를 만들지** — 이건 이후 critic 검증 전에 방향을 잡아두면 좋겠습니다.
