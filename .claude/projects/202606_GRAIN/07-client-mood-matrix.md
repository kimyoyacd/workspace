## Step 7. Brutalist Framework 적용 판단

**적용 여부: 부분적용**

**적용 이유**: Instrument Grade의 핵심 언어(공차 표기, 소재 그대로 노출, 장식 제거)는 Brutalism의 "Exposed Structure"·"Raw Surface" 원칙과 철학적으로 겹침. 개스킷·나사산을 숨기지 않는 것은 "이 물건이 어떻게 작동하는지 증명한다"는 철학을 재료 차원에서 실행하는 방식.

**적용하지 말아야 할 이유**: 순수 Brutalism(노출 콘크리트, 거친 타이포, 비대칭 그리드, 의도적 조야함)은 "프리미엄 모래시계"의 촉각적 고급감과 정면 충돌. 원형적 태도를 그대로 가져오면 계측기가 아니라 산업 폐기물처럼 읽힐 위험.

**사용 가능한 요소**
- Exposed Structure → 공차 표기, 조립선 시각화, 나사산·개스킷 디테일 노출
- Raw Surface → 스틸그레이 무광 텍스처, 브러시드 메탈 마감선
- Grid Honesty → Suisse Mono 공차 타이포 정렬선을 오히려 드러내는 레이아웃

**피해야 할 요소**
- 거친 콘크리트 질감 — 스모크글라스·브라스와 상극
- 의도적 비대칭·불완전 정렬 — "계측기"는 정밀함이 생명
- 채도 낮은 무채색 단일 팔레트로의 전면 이동 — 브라스의 온기가 죽음

**브랜드 전략과의 연결**: Brutalism을 "스타일"이 아니라 "증명의 태도"로 재정의해서 차용. "거칠게 보이기 위해 거칠다"가 아니라 "숨길 이유가 없어서 노출한다"는 논리 안에서만 허용.

**패키지 적용 가능성**: 가능. 조립 공차선·측정 기준선을 프린트/엠보싱으로 노출. 단, 소재는 러프 콘크리트가 아니라 스틸그레이 무광 카드보드/메탈릭 포일 한정.

**웹/SNS 적용 가능성**: 웹은 공차 타이포·그리드선 노출이 UI 레이어에서 자연스럽게 확장. SNS(Gravity Cut)는 "증거 샷"의 즉물성만 차용하고 장식적 거침은 배제.

**리스크**: 제작 파트너가 "Brutalism"을 문자 그대로(거친 콘크리트 등) 해석해 프리미엄 톤을 훼손할 위험 — 브리프에서 "태도 차용, 스타일 차용 아님" 명문화 필수.

**보완 장치**: 무드보드에 "Do/Don't" 페어 이미지 명시(노출 O / 거침 X). 소재 위계 고정(스틸그레이·스모크글라스·브라스 3종 외 질감 도입 금지).

**최종 판단**: Brutalism은 "노출의 논리"로만 제한적 차용. Exposed Structure/Raw Surface 두 원칙만 흡수, 거친 질감·비대칭·반정제 타이포는 전면 배제.

## Step 8. Concept Shortlist 최종 확정

### 안전 — Instrument Grade
- **한줄정의**: 캘리브레이션된 도구로서의 모래시계 — 정밀함이 곧 신뢰의 증거
- **선택이유**: Gate 1 철학을 가장 손실 없이 번역, 정적 매체에서 즉시 스케일, 실행 리스크 최저 (Total 28/30)
- **전략연결**: "정밀"을 메인 축으로 신뢰도의 근거
- **비주얼방향**: 스틸그레이+스모크글라스, 랩 라이트박스 촬영, Suisse Mono 공차 타이포(그래픽 요소로 승격)
- **패키지방향**: 외함에 조립 공차선·측정 기준선 노출(엠보싱/포일), 스틸그레이 무광 카드보드 한정
- **카피방향**: 서술형 배제, 수치·공차·기준값 중심의 "측정 언어" ("0.02mm의 오차도 허용하지 않는다")
- **리스크**: 지나치게 차갑고 산업적 — 럭셔리 온기 부족
- **보완장치**: 브라스 디테일(손잡이·캡) 온기 포인트 1곳 고정, 라이트박스 조명 색온도 미세 웜톤 보정

### 차별화 — Gravity Cut
- **한줄정의**: 중력이 작동하는 찰나를 포착한 증거 샷 — 행위 자체가 브랜드
- **선택이유**: SNS 숏폼 시각 임팩트·차별화력 최고, Instrument Grade의 정적 신뢰감을 보완하는 동적 증거
- **전략연결**: "행위"를 전면화 — 모래 낙하를 "증명의 행위"로 재프레이밍
- **비주얼방향**: 브라스+다크차콜, Grotesk 대형 넘버링, 하이스피드 크롭, 스트로보 편집
- **패키지방향**: 메인 패키지엔 미적용(포맷 종속성) — 언박싱 영상·SNS 전용 모션그래픽으로 한정
- **카피방향**: 짧고 즉각적 임퍼러티브("떨어지는 순간, 증명된다" 4~6단어 훅)
- **리스크**: 정지 매체 전환 시 임팩트 약화
- **보완장치**: 정지 이미지 파생 시 "정점의 순간"(모래가 공중에 뜬 프레임)만 선별 사용하는 컷 선정 룰 문서화

### 실험적 대안(3순위 보류) — Field Notes
- **한줄정의**: 검증된 기록물로서의 모래시계 — 매 순간의 측정이 기록되고 인증된다
- **선택이유**: 안정적 완성도+리스크 최저, 예산·일정 문제 시 대체 가능한 보험
- **리스크**: 항해일지·인증서 모티프가 프리미엄 오브제 시장에서 이미 흔한 클리셰
- **채택 조건**: 시리얼넘버를 실제 낙하시간 오차와 연동시켜 "실측 기반 인증"으로 차별화 지점 보강 필요
