CONCEPT C01
오늘의 선택

감성 진입 → 즉각 전환 주력 컨셉 · PDP 시뮬레이션

퍼스널컬러 진단 프레임을 걷어내고 직관과 감성으로 선택하게 만드는 첫 번째 컨셉. 이유를 묻지 않는 것이 전략이다.
오전 창가 자연광·코랄 브릭의 온기가 "오늘 기분"을 건드리는 감각 언어로 구성. 올리브영 채널 온리 포지셔닝으로 전환 마찰을 최소화한다.
Strategic Fit 9/10 — 세 컨셉 중 가장 빠른 감성 연결과 전환률을 기대할 수 있는 구성. Hero~CTA 섹션 전체가 하나의 감정 곡선으로 설계되어 있다.

S1 — Hero 섹션 분석
다크 배경 풀블리드 히어로의
역할과 카피 전략
히어로는 PDP의 첫 감정 게이트다. C01은 #111 배경으로 코랄이 최대 채도로 보이는 환경을 만들고, 브랜드 레이블을 brass italic으로 작게 올려 "선택된 컬러" 감각을 준다.

메인 카피 "오늘 기분을 먼저 골라도 돼"는 이유와 근거를 먼저 요구하는 퍼스널컬러 피로에 반응한 언어다. 서브 카피 "이유는 나중에 찾아도 된다"는 구매 허락 메시지로 기능한다.
coral 강조는 브릭 톤으로 조정 — 형광 오렌지가 아닌 "오전 창가의 따뜻한 빛"으로 읽혀야 한다.
S2 — Color Story 팔레트
코랄 브릭 + 피치 + 크림

S3~S5 — 텍스처·스킨톤·사용법
미들 섹션: 감각 → 확인 → 신뢰
3단계 설득 구조
텍스처 클로즈업(S3)은 제품 발색의 감각을 CSS 그라디언트로 시각화. 립을 바르는 순간의 감각적 묘사로 상상력을 자극한다.

스킨톤 슬롯(S4)은 진단이 아닌 확인의 역할 — "내 톤에도 어울리겠다"는 확신을 주는 장치로 작동. 쿨·뉴트럴·웜 3그룹을 CSS 그라디언트로 표현해 실제 이미지 없이도 톤 차이를 명확히 전달한다.

3스텝 사용법(S5)은 아이콘 + 짧은 텍스트로 진입 장벽을 낮춘다.
S3 텍스처 그라디언트 블록 — "발리는 순간의 감각" 카피 오버레이
S4 스킨톤 슬롯 3개 (쿨/뉴트럴/웜) — CSS 그라디언트로 피부톤 표현
S5 3스텝 사용법 — 아이콘 + 텍스트 카드 구성
S6~S8 — 채널 온리 · 리뷰 · CTA
전환 설계: 독점성 → 사회적 증거
→ 감성 CTA
S6 OliveYoung Only 섹션은 다크 배경에 brass 배지로 희소성과 채널 온리 가치를 부각시킨다. "이 컬러 조합은 올리브영에서만 고를 수 있도록 만들었다"는 카피는 경쟁 채널 이탈을 막는 앵커 역할을 한다.

S7 리뷰는 별점 4.8과 2개의 실제 구매자 목소리로 사회적 증거를 쌓는다. S8 CTA는 coral 배경으로 감정을 최고조로 끌어올리고 흰 버튼으로 클릭 유도한다.
CTA 버튼 카피 "오늘의 색, 담기"는 담다(장바구니) + 오늘의 색(감성)의 이중 의미. 행동 유도와 감성 언어를 동시에 충족.
Decision Rationale
왜 C01을
추천하는가
세 컨셉 중 C01은 가장 낮은 인지적 진입 장벽을 가진다. 퍼스널컬러 피로가 쌓인 20~30대 여성에게 "분석 없이 골라도 된다"는 메시지는 강력한 허락 언어로 기능한다.

올리브영 채널 온리 포지셔닝은 타 채널 비교 쇼핑을 차단하고 즉각 구매로 연결되는 경로를 만든다. 감성 언어 + 채널 독점 조합이 Strategic Fit 9/10의 근거다.
