# 03 목표와 문제 재정의 — Boomi

## 4관점 비교

| 관점 | 내용 |
|---|---|
| `Brand Says` | "던지면, 돌아온다" — 무전원 프리미엄 야외 놀이 (설계값) |
| `Brand Does` | 바이오 소재 3-blade 안전 부메랑 — 아직 제품·경험 실체 없음 (컨셉) |
| `Customer Sees` | 아직 없음. 대리 지표: 부메랑=저가 기념품 or 마니아 스포츠라는 기존 인식 (INFERENCE) |
| `Market Compares` | 장인 공방(성능·투박) vs 저가 노벨티 vs 컨셔스 프리미엄(실내·정적) — 프리미엄 야외 플라잉 브랜드 부재 (FACT, 01-research) |

## 상태 정의

- **현재 상태**: '부메랑'은 브랜드 없는 카테고리다. 장인 성능 아니면 관광 기념품. 보호자 언어(안심·발달·소재)로 말하는 플레이어가 없다.
- **희망 상태**: "화면 없는 야외 놀이의 프리미엄 기준"으로 인식되는 첫 브랜드. 보호자가 안심하고, 아이가 조르는 브랜드.
- **활용 자산**: 회귀 서사(고유 물리 마법) · 바이오 소재 · 캐릭터 세계관 · 17년 디자인 역량 (실행 주체)
- **브랜드 격차**: 인지 제로 · 수요 미검증 · 습득 난이도라는 카테고리 태생 장벽
- **미검증 가설**: ① 프리미엄 부메랑 절대 수요 존재 ② 보호자가 '안심+디자인'에 프리미엄 지불 ③ 습득 난이도가 안전 형상 설계로 극복 가능

## 문제 재정의

```
표면적 요청
어린이 부메랑 브랜드의 컨셉과 비주얼이 필요하다.

현재 현상
부메랑 시장은 성인 마니아 성능 경쟁과 저가 기념품으로 양분되어,
보호자가 살 이유도 아이가 조를 이유도 만들지 못하고 있다.

원인 가설
카테고리가 '기능(비행 성능)'의 언어로만 말해 왔고,
구매 결정자(보호자)의 언어 — 안심·소재·발달·세계관 — 로 번역된 적이 없다.

진짜 해결 문제
'던지면 돌아온다'는 부메랑 고유의 물리적 마법을,
보호자가 프리미엄 가격으로 안심하고 사고
아이가 스스로 원해서 조르는 브랜드 경험으로
어떻게 번역·증명할 것인가?

내려야 할 의사결정
부메랑 단일 코어로 갈 것인가, 플라잉 토이 라인으로 확장 설계할 것인가.
```

## 성공 기준
1. 시장 빈자리 논증이 출처 있는 데이터로 성립
2. 보호자 안심 게이트(무전자·소재·안전 형상)를 브랜드 언어로 통과
3. 아이 견인 장치(캐릭터·수집·성취)가 전략에 내장
4. 확장 의사결정(부메랑 vs 플라잉 토이)의 근거 확보

## 리서치 질문
1. 경쟁·대체 브랜드 구도에서 '프리미엄 야외 플라잉 토이' 자리는 실제로 비어 있는가, 왜 비어 있는가?
2. 보호자 안심을 브랜드 자산으로 만든 브랜드들은 무엇을 증명 장치로 썼는가? (이종업계)
3. 아날로그 가치를 프리미엄으로 번역한 브랜드들의 공통 원리는? (이종업계)
4. 아이 주도 수집·성취 리텐션을 소비주의 반감 없이 설계한 사례는? (이종업계)
