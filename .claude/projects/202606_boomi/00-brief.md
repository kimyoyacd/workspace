# 00 브랜드 브리프 — Boomi 어린이 부메랑

- **클라이언트**: 미정 (인테이크 단계)
- **작성**: 2026.06
- **업무 범위**: AI 활용 브랜드·비주얼 컨셉 제안 (양산 아님 / 컨셉 덱)

## 브랜드 기본 정보

| 항목 | 내용 |
|---|---|
| **브랜드명** | Boomi (부미) |
| **카테고리** | 어린이 야외 놀이 — 부메랑 |
| **주요 공간/채널** | 아웃도어 완구 / 온라인 D2C |
| **핵심 목적** | 던지고 돌아오는 신체 야외 놀이를 프리미엄 브랜드로 재정의 |
| **타겟층** | 어린이(5~12세) + 구매결정자 보호자 |
| **브랜드 이름 의미** | Boomerang + 돌아오는 관계(回歸) |
| **핵심 기능** | 바이오 소재 무전원 부메랑 / 폰이 비행 관찰 (아이는 화면 안 봄) |
| **브랜드 스토리** | 던진 만큼 되돌아오는 놀이. 씨앗이 자라는 것처럼, 아이의 몸과 감각이 자란다. |

## 제약 조건
- AI는 **비주얼 제작 도구**로만 사용 (제품에 AI 미탑재, 브랜드 카피에서 AI 언급 없음)
- 어린이제품안전특별법·KC 인증 필수 (양산 시)
- 보호자 '화면 의존' 우려 해소가 핵심 구매 게이트
