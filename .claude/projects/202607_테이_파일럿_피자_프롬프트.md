## 🔒 STYLE LOCK (매 컷 동일)
```
Stylized 3D art-toy render, Blythe-style oversized head with large glossy round eyes,
soft matte vinyl-figure surfaces, gentle subsurface scattering, clean soft studio lighting.
Muted editorial palette — warm cream, sage green, dusty floral tones, low saturation.
NOT photoreal, NOT 2D anime, no glossy CGI plastic, no candy-bright colors.
Storybook-calm, cozy craft mood.
```

## 🔒 CONTINUITY (매 컷 동일)
```
tei — 7-year-old girl. Black chin-length bob with blunt fringe, big round blue eyes with
strong catchlights, chubby cheeks, small nose. Signature outfit: green floral puff-sleeve
shirt + sage floral pinafore jumper-skirt, cream socks, mint-green sneakers.
Stays visually identical every shot (same face, hair, outfit, proportions as the character sheet).
Props are felt / fabric food pieces (not real food). Seamless warm-neutral / pastel-sage
backdrop, low saturation — never candy-bright.
```

## 🔒 DIRECTION (톤 A — 매 컷 동일)
```
Front bust shot. Gentle static or slow push-in — calm, no whip-pans, no dutch angles.
Beats land softly on a 4-count with smooth transitions, not hard cuts.
Sound is ASMR felt-texture (soft pat, gentle thump), not loud SFX.
Expression restrained, sincere, warm — micro-changes, never exaggerated comedy.
```
