# 202607 · 피지컬 AI·스마트팩토리 — 디자인×AI 업력 기반 사업 확장

- **구분**: 신사업 기획 (아이디어 인박스 1호)
- **출처**: 유튜브 https://youtu.be/3078JD6gqxU (피지컬 AI·첨단 공장 턴키 수출 국가 전략)
- **상태**: 확장완료 (brainstormer 발산 + critic 평가 완료, 실행 판단 대기)
- **관련 클라**: 코웨이 · 아모레퍼시픽 · 웨이크메이크 · 올리브영

## 한 줄 요약
게임 UI(고밀도 실시간 화면) + 제조 클라 접점 + AI 에이전트 운영력을 묶어,
**관제·디지털 트윈 대시보드 UI(B안)를 쐐기**로 "제조업 AI 전환 크리에이티브 파트너"로 확장.

## 3안 평가 (critic, 50점 만점)

| 안 | 점수 | 판정 | 6개월 내 첫 계약 |
|---|---|---|---|
| **B. 관제센터·디지털 트윈 대시보드 UI** | **33** | 조건부 추진 | 조건부 가능 (코웨이 경유 파일럿 2~4천만 or 솔루션사 파트너십) |
| A. 피지컬 AI 데이터 수집 UI/UX 컨설팅 | 22 | 보류 → B의 위성 상품 | 앱 UI 소액 건만 가능, 컨설팅 불가 |
| C. 제조 혁신 콘텐츠 에이전시 (SOP→AI 자산) | 22 | 보류 → 격하 버전만 | 소비자향 매뉴얼·CS 콘텐츠로 낮추면 가능 |

**B가 이기는 이유**: 관제화면 = 실시간 전략게임 HUD (정보 밀도·긴급 알람·상태 시인성 전부 게임 UI 문법). 사내 대시보드 v7 실적이 즉시 증빙 포트폴리오. HTML 라이브 데모를 자체 제작할 수 있는 디자인사는 드물다.

**공통 제약**: 가동률 이미 정상~과부하 → 신사업 BD는 실장 1인 몫. 개발 인력 0명 → 구축형 상품은 파트너 의존.

## 확장 로드맵 (3단계)

### 1단계 (0~6개월) — B 쐐기
- 게임 UI 포트폴리오를 "고밀도 실시간 정보 설계"로 재프레이밍한 관제 대시보드 스펙 목업 1세트를 **HTML 라이브 데모**로 제작 (외주비 0원)
- 코웨이 접점 경유 생산/DX 파일럿 제안: 화면 5~10개, 2~4천만 원, **운전원 인터뷰 2회를 계약 범위에 명시** ("예쁜 대시보드" 함정 방지)
- 병행: 디지털 트윈/스마트팩토리 **솔루션 제품사** 2~3곳에 디자인 파트너 제안 — SI 하청보다 이 경로 우선
- KPI 단 하나: **6개월 내 파일럿 1건 수주**

### 2단계 (6~18개월) — 위성 상품 확장
- B-1 Control Room 디자인 시스템 (알람 위계·다크모드·위젯 라이브러리) → 리테이너 전환
- B-4 수출 영업용 데모 관제화면 (전시회·바이어 미팅용 시뮬레이션) → 턴키 수출 컨소시엄 대상
- A 격하 버전: "피지컬 AI 스타트업용 작업자 앱 UI 패키지" 정찰제 (컨설팅 단어는 실적 3건 전까지 금지)
- C 격하 버전: 기존 클라 소비자향 매뉴얼·CS 콘텐츠에 AI 파이프라인 파일럿

### 3단계 (18개월~) — 컨설팅 단가 구간 이동
- 포지셔닝 빈자리: **제조 × 컨설팅** 구간 (경쟁자는 대형 컨설팅펌뿐인데 화면을 직접 못 만든다)
- 경로: B-3 UX 감사 → B-1 시스템 구축 → A 데이터 수집 UX 컨설팅
- 보너스: D. 산업 트레이닝 게이미피케이션 (안전교육→게임 튜토리얼 UX, 코웨이 서비스 기사 교육 첫 타깃), E. AI 스튜디오 셋업 컨설팅 (에이전트 24종 운영 경험 자체를 상품화)

- [ ] 영업 경로 우선순위: 솔루션 제품사 > 파일럿 예산 > SI 하청 (제조 대기업 직발주는 기본값 아님)

## 진입 시나리오 (기존 클라)
- **코웨이**: 마케팅 라인 → 생산/DX 소개 → 공장 품질검사 화면 파일럿. 서비스 기사 교육 게임화(D)도 카드
- **아모레퍼시픽**: 오산 뷰티파크 견학·관제 화면 UX 감사 파일럿 — 견학 동선 화면이라 마케팅 예산으로도 집행 가능
- **웨이크메이크**: 신제품 공정 촬영분을 SOP+브랜드 콘텐츠 이중 활용 (C 무마찰 진입)
- **올리브영**: 배너·상세페이지 에이전트 파이프라인 워크숍 → 리테이너 (E 진입)
