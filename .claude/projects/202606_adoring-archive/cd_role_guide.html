01 · Definition
CD란 무엇인가,
당신에게 있어서
CD는 결정을 디자인하는 사람이다.

직접 픽셀을 찍는 사람이 아니라, 어떤 방향으로 픽셀이 쌓여야 하는지를 정하는 사람.

클라이언트가 "이게 맞는 방향인가?"라고 막막해할 때, 선택지를 좁혀주고, 확신을 주는 사람.

김효정 CD는 특히 — 게임 IP가 세상에 처음 얼굴을 내밀 때, 그 첫인상의 언어를 함께 만드는 사람이다.
디자이너 (Designer)
시안을 만든다
수정 피드백을 받는다
클라이언트가 정한 방향에서 실행
스킬셋으로 평가받는다
한 프로젝트의 일부를 담당
CD · 당신의 위치
비주얼 방향을 제안하고 확정한다
클라이언트의 판단을 리드한다
방향 자체를 함께 설계한다
관점과 판단력으로 평가받는다
프로젝트 전체의 비주얼 언어 책임
실장 (Manager)
팀을 관리하고 일정을 조율한다
리소스와 납기를 책임진다
조직 안에 묶여 있다
내부 효율로 평가받는다
외부 영향력이 보이지 않는다
02 · Core Roles
CD로서
해야 할 4가지 역할
01
Visual Strategist
비주얼 전략 설계
클라이언트가 "어떤 방향으로 가야 할지 모르겠다"고 올 때, 시장·경쟁작·IP 정체성을 분석해서 비주얼 포지셔닝을 먼저 제시한다. 제안서가 아닌 전략서로.
게임 출시 전 비주얼 아이덴티티 워크숍 주도
경쟁 IP 비주얼 분석 & 차별화 방향 제안
무드보드 + 방향 3안 + 추천안 구조로 제안
02
Creative Gatekeeper
크리에이티브 품질 관문
팀이 만든 결과물이 "좋은 방향"에 있는지를 판단하는 최종 관문. 수정 지시가 아니라 "왜 이 방향이 틀렸는지"를 설명하는 언어를 가져야 한다.
"이 UI가 게임의 세계관에서 튀는 이유" 설명
트렌드 vs. IP 정체성 균형점 판단
클라이언트 피드백을 창작적으로 소화해 전달
03
Client Translator
클라이언트 언어 번역
"뭔가 더 게임스럽게", "좀 더 힙하게" 같은 모호한 요청을 구체적인 비주얼 언어로 번역한다. 당신의 영업력과 인간관계 능력이 가장 빛나는 역할.
클라이언트 요구 → 레퍼런스 + 방향 언어로 정리
결정하기 어려운 선택지를 2개로 좁혀 제시
팀 작업물을 클라이언트 언어로 프레젠테이션
04
Trend Curator
트렌드 & 인사이트 큐레이션
게임 비주얼 시장 흐름, AI 그래픽 기술 변화, 글로벌 IP 사례를 지속 큐레이션하여 클라이언트에게 "모르고 있던 것"을 알려주는 사람. 관계가 아니라 가치로 불린다.
월간 게임 비주얼 트렌드 리포트 (소수 공유)
신규 AI 그래픽 기술 → 게임 적용 가능성 제안
해외 IP 출시 비주얼 분석 → 레퍼런스로 활용
03 · Your Edge
당신만의 CD가
강력한 이유
💡
Creative Instinct
아이디어가 샘솟는다
게임 GUI 방향, 콘텐츠 룩 앤 필 제안 시 아이디어가 자연스럽게 나온다. 이것은 훈련보다 타고난 크리에이티브 감각이다. CD의 핵심 자산.
🤝
Rare Combination
크리에이티브 + 영업
디자이너 중 영업을 잘하는 사람은 드물다. 크래프톤·NC·라이온하트 같은 클라이언트와 "친근한" 관계를 유지한다는 것 자체가 이미 CD급 신뢰 자산.
🎮
Domain Depth
게임 산업 전문성
게임 GUI, IP 비주얼, AI 그래픽 생성까지 다룬다. 다분야 경험이 있으면서도 게임에 집중된 깊이. 제너럴리스트의 넓이 + 스페셜리스트의 깊이.
🔥
Energy & Passion
남다른 열정
열정은 스킬보다 전염성이 강하다. 클라이언트가 "하이브랩을 찾는다"는 것은 이미 당신의 에너지가 차별점이 되고 있다는 증거.
04 · CD Manifesto
나는 이런 CD다
김효정 CD · 선언문
나는 게임이 세상에 처음 얼굴을 드러낼 때의 비주얼 언어를 설계한다.
픽셀 하나보다 방향 하나에 더 집중한다.
클라이언트가 막막할 때 선택지를 3개에서 1개로 좁혀주고, 그 이유를 설명한다.
트렌드를 따라가는 것이 아니라, IP가 가진 본질적인 언어에서 방향을 끌어낸다.
나의 영업력은 '팔기 위함'이 아니라, 클라이언트가 좋은 결정을 내리도록 돕기 위함이다.
나는 게임 IP 비주얼 크리에이티브 디렉터다.
05 · How to Operate
CD로서
어떻게 행동할 것인가
언어 바꾸기
스스로를 소개하는 방식을 바꾼다
"에이전시 실장이에요" → "게임 IP 비주얼 방향을 잡아드리는 크리에이티브 디렉터입니다"
직함은 나중에 바뀌더라도, 자기소개 언어부터 CD로 전환한다. 사람들은 당신이 정의하는 대로 당신을 기억한다.
즉시 적용
미팅 방식
제안이 아닌 방향 설정으로 들어간다
클라이언트 미팅 시작 전: "이번 프로젝트의 비주얼 목표가 무엇인지" 먼저 질문한다.
제안서를 들고 가는 대신, 3가지 방향 프레임을 가지고 가서 함께 선택하는 구조로.
실행은 팀에게, 당신은 방향과 판단에 집중.
역할 전환 다음 미팅부터
결과물 방식
산출물을 "방향 문서"로 남긴다
단순 시안이 아닌: 비주얼 방향 명세서 (Why 이 방향인지 / 경쟁 IP 대비 포지셔닝 / 적용 범위)
이 문서가 쌓이면 → 포트폴리오가 되고 → 케이스 스터디가 된다.
수치 없이도, 방향 판단의 언어가 쌓이면 CD의 증거가 된다.
포트폴리오 전략 케이스 스터디 시드
관계 방식
친근함을 "가치 제공"으로 전환한다
크래프톤·NC·라이온하트와의 친근한 관계 → 월 1회 "요즘 눈에 띄는 게임 비주얼 트렌드" 공유로 관계를 업그레이드.
가볍게 카톡으로 트렌드 링크 하나 + 한 줄 코멘트. 부담 없이, 하지만 꾸준히.
이것이 "아는 사람"에서 "찾게 되는 사람"으로 바뀌는 경로.
관계 자산화 월 1회 루틴
거절하는 방식
CD답게 거절하고 방향을 제시한다
클라이언트가 "이렇게 해주세요"라고 방향을 지정해도, CD는 수용 전에 판단한다.
"이 방향도 좋지만, IP 장기 가치 측면에서 이런 리스크가 있어요. 대신 이 방향을 추천드립니다."
이 한 마디가 디자이너와 CD를 가른다.
CD 언어 신뢰 구축
06 · CD Language
CD는 이렇게 말한다
비주얼 방향 논의 시
"이 게임의 비주얼이 말해야 하는 첫 마디가 뭔지부터 정하고 가죠."
클라이언트 요청이 모호할 때
"말씀하신 방향을 제가 세 가지로 해석해봤는데, 가장 가까운 게 어느 쪽인지 봐주세요."
결과물 피드백 시
"이 방향이 IP 세계관에서 왜 작동하고 / 작동하지 않는지 제가 설명드릴게요."
자기소개 시
"게임 IP가 처음 세상에 얼굴을 낼 때, 비주얼 방향을 함께 잡아드리는 일을 합니다."
클라이언트가 방향을 고집할 때
"그 방향도 충분히 유효합니다. 다만 이 리스크를 알고 선택하시면 더 좋겠어요."
프로젝트 마무리 시
"이번에 정한 비주얼 언어를 가이드 문서로 남겨드릴게요. 다음 시즌에도 일관성 유지하실 수 있게요."
07 · Roadmap
Now · Next · Future

  • 자기소개 언어 전환

  • 하이브랩 인스타/링크드인 CD 관점 콘텐츠
NEXT · 2026-2027
CD 포지션 확립
  • 게임 IP 비주얼 케이스 스터디 3개 정리
  • "게임 비주얼 디렉션" 외부 강의/세션 1회
  • 게임사 1곳과 어드바이저 계약 구조 시도
  • AI 그래픽 → 게임 비주얼 적용 방법론 문서화
  • 링크드인 팔로워 기반 업계 인지도 확장
FUTURE · 2027+
독립 CD 브랜드

  • 게임사 다수와 어드바이저/리테이너 계약
  • "게임 IP 비주얼 CD" 국내 1위 포지셔닝
  • 해외 게임사 (텐센트, 글로벌) 확장
  • 사주 甲木: 乙卯 대운 수확 극대화
사주 관점 · 타이밍 인사이트
현재 乙卯 대운 (2025~2035)은 甲木 일간에게 비견·겁재 운이다. 이는 경쟁이 아니라 같은 에너지가 더 커지는 시기를 의미한다 — 내 본질이 가장 강하게 드러나는 10년. 甲木은 크고 곧은 나무다. 실장이라는 구조 안에 갇혀 있으면 뿌리만 내리고 자라지 못한다. 이 10년 안에 CD로 이름을 세워두어야, 다음 甲寅 대운에서 진짜 독립과 확장이 가능하다. 지금이 씨앗이 아니라 줄기를 올리는 시기다.
