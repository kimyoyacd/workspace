### P1 — AI 제품 폼 스터디 (3–5일)
**상태**: 🔲 미시작

- Higgsfield Nano Banana Pro로 제품 형태 스케치 → 렌더링 반복
- 소재(매트 플라스틱 / 유리 / 금속 / 세라믹), 색상, 비율 탐색
- Alun Prompt Method — 손 스케치 → "Turn this drawing into a render" 루프
- 최종 3–5개 방향 선정 → FAVORITE 마킹 → 1개 주력 방향 결정

**툴**: Nano Banana Pro, Alun Method, ComfyUI(선택)

---

### P2 — 비주얼 아이덴티티 (3–5일)
**상태**: 🔲 미시작

- **네이밍**: Claude로 10–20개 후보 → 편집 감각으로 1개 선택 (한국어 감각 우선)
- **로고**: AI로 스케치 다방향 → 유망한 것 선정 후 타입 리파인
- **컬러**: 3 iteration — 초안 → 보완 → 최종 팔레트
- **패키징**: 벡터 다이컷 → AI 렌더 → 3D 전개도 탐색

**툴**: Claude, Nano Banana Pro, Figma

**선정 브랜드명**: _(미정)_

---

### P3 — AI 포토그래피 + 영상 캠페인 (3–4일)
**상태**: 🔲 미시작

- **스튜디오 컷**: 화이트 배경 미니멀 제품샷, 소재·조명 탐색
- **서리얼 컷**: YOYOYO NYC 빌딩 방식 — 예상치 못한 스케일·배경
- **라이프스타일**: 인-사이추에이션 — 사람과 함께, 공간 속 제품
- **영상**: Higgsfield Video로 cinematic 모션 3–5클립

**툴**: Higgsfield(이미지/영상), Seedance, Kling, Soul Model

---

### P4 — 케이스스터디 사이트 + 공개 (2–3일)
**상태**: 🔲 미시작

- **HTML 사이트**: YOYOYO 방식 단일 스크롤 — Intro → 제품 → 아이덴티티 → 포토 → 영상 → 요약
- **차별점**: 각 섹션에 AI 프롬프트 원문 일부 공개 (투명성 = 신뢰)
- **공개 채널**: 효정 LinkedIn (Solo) 또는 MAX Lab 계정 (팀)
- **활용**: 피칭 레퍼런스 / AI 크리에이티브 역량 증명 자산

**툴**: Claude HTML 제작
