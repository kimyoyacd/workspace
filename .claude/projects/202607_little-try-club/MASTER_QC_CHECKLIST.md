## 2️⃣ 세계관 & 배경 검증

### 시각 스타일 (Clean Surreal Fashion Playground)

**배경 원칙** — 깨끗하고 단순하지만, 전체 화면은 컬러풀하고 감각적

- [ ] 배경이 단순하고 깨끗한가? (디테일 최소)
- [ ] 배경이 파스텔 톤 또는 단일 컬러인가? (Cream, Peach, Sage Green, Sky Blue, Soft Beige 등)
- [ ] 배경에 불필요한 패턴이나 디테일이 없는가?
- [ ] 배경이 실제 방처럼 현실적이지 않은가?
- [ ] 배경이 교실이나 설명적인 교육 환경이 아닌가?

**컬러 분위기** — 배경은 깔끔하되, 캐릭터·오브제·패턴으로 감각적 임팩트 창출

- [ ] 캐릭터의 의상이 강렬한 포인트 컬러(코발트, 토마토 레드, 코랄, 그라스 그린 등)를 사용하는가?
- [ ] 의상의 패턴(스트라이프, 체크, 컬러 블록)이 그래픽적으로 강렬한가?
- [ ] 오브제의 색상이 시각적으로 눈에 띄는가?
- [ ] 전체적으로 8-12세 아이들이 보기에 "재미있고 흥미로운" 분위기인가?
- [ ] 유치원 느낌(과하게 귀엽고 단순한)은 아닌가? (패션 에디토리얼처럼 세련된 느낌)

### 오브제 배치

- [ ] 오브제가 1-3개만 사용되었는가?
- [ ] 오브제가 배경을 채우기보다는 후킹 포인트로 작용하는가?
- [ ] 오브제가 캐릭터보다 크거나 비슷한 크기인가? (초현실감)
- [ ] 오브제가 니트, 퍼, 펠트, 클레이 같은 촉각적 질감인가?
- [ ] 오브제 색상이 캐릭터 의상과 대비되거나 조화로운가?

### 컬러 팔레트

- [ ] 배경: 60% 깨끗한 파스텔? (Cream, Beige, Peach, Sage Green, Sky Blue)
- [ ] 캐릭터: 25% 따뜻한 톤?
- [ ] 패턴: 10% 대담한 포인트 컬러?
- [ ] 악센트: 5% 생생한 색? (Cobalt, Tomato, Coral, Butter Yellow, Grass Green)
- [ ] 포인트 컬러가 화면에서 2-3개 이하인가?
- [ ] 베이지와 카키만 반복되지 않았는가?

### 질감 정의

- [ ] 캐릭터가 매트한 클레이 같은 질감인가?
- [ ] 오브제가 손으로 만지고 싶은 느낌인가?
- [ ] 전체 렌더가 광택 있는 플라스틱처럼 보이지 않는가?
- [ ] 피부 표현이 극사실적이지 않은가?
- [ ] 눈이 과하게 반짝이지 않는가?

### 카메라 & 구성

- [ ] 캐릭터가 화면 중심 또는 3분할 지점에 배치되었는가?
- [ ] 배경에 넓은 여백이 있는가?
- [ ] 카메라 무브먼트가 자연스러운가? (빠르지 않음)
- [ ] 첫 2초 안에 문제와 캐릭터 표정이 보이는가?
- [ ] 조명이 따뜻하고 부드러운가?

- [ ] 마지막이 따뜻하고 긍정적인가?
- [ ] 교육이 직접 설명되지 않고 행동으로 보여지는가?

### 캐릭터 감정 흐름

- [ ] 캐릭터의 감정 변화가 자연스러운가?
- [ ] 처음: 놀람/답답함 → 중간: 생각/결심 → 끝: 성취감/평온함?
- [ ] 표정이 각 단계마다 적절히 변하는가?

### 8-12세 시청 적절성

**나이대 타겟팅** — 어린이용이지만 세련되고 성숙한 느낌

- [ ] 아이들이 공감할 수 있는 현실적인 상황인가?
- [ ] 너무 복잡하지 않고 명확한가?
- [ ] 캐릭터가 능동적이고 주체적으로 문제를 해결하는가?
- [ ] 무섭거나 불안감을 주지 않는가?
- [ ] 희망적이고 따뜻한 메시지가 있는가?

**시각적 성숙도** — 유치원틱하지 않으면서도 친근한 느낌

- [ ] 과하게 귀엽고 단순한 유치원 느낌이 아닌가?
- [ ] 캐릭터의 의상이 현대적이고 세련되었는가? (시대 뒤떨어지지 않음)
- [ ] 패턴과 색상이 그래픽적으로 강렬하고 기억에 남는가?
- [ ] 전체 분위기가 "키즈 패션 에디토리얼"처럼 세련되었는가?
- [ ] 유아 장난감 같은 단순함이 아니라, 8-12세가 즐길 수 있는 감각적 디자인인가?
