## CONTINUITY 블록 (여기서 한 번만 정하고 모든 키프레임에 고정 재사용)

**캐릭터 외형**: `01_Character_Bible/<이름>/<이름>-bible.md`에서 그대로 복사 — 재서술 금지
```
[여기에 바이블 문구 붙여넣기]
```

**공간 지오메트리** (가구·문·창문 배치, 컷마다 불변):
```
[예: 나무 바닥, 왼쪽에 낮은 책장, 오른쪽 창문, 벽지는 장난감 패턴]
```

**조명·시간대** (에피소드 전체에서 고정 — 장면마다 절대 바꾸지 않는다):
```
[예: warm yellow lamp lighting, dusk, consistent throughout]
```

> STYLE_LOCK(그림체·렌더 스타일)은 시리즈 공통이라 여기서 다시 안 적는다 — `episode-director.md`
> §3의 STYLE_LOCK 문구를 모든 키프레임에 그대로 쓴다.
