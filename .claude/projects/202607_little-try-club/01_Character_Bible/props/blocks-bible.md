# 블록 소품 바이블 (Juni 시리즈 공용)

## 기준 이미지

- 파일명: `block_reference_01.png`
- 생성일: 2026.07.09
- 용도: 컷마다 텍스트 설명만으로 블록을 새로 생성하면 큐브 비율/크기가 미세하게 달라지는 문제를 막기 위한 고정 기준 이미지. 캐릭터 기준 이미지와 항상 함께 참조한다.

## 설명

동일한 라운드 큐브 비율의 니트 블록 3개, 색상만 다름:
- 코발트 블루
- 토마토 레드
- 버터 옐로우

질감: chunky knitted fabric, visible yarn stitching, matte finish, rounded edges. 세 블록 모두 동일 비율 유지.

## 사용 규칙

- 스틸 이미지 생성 시 `medias`에 캐릭터 기준 이미지 + `block_reference_01.png`를 **함께** role: image로 전달할 것.
- 프롬프트에도 "blocks matching the exact cube proportions and knit texture of the reference image" 문구를 추가해 재차 고정할 것.
- 색상 조합을 바꾸고 싶을 때(다른 에피소드 등)는 이 기준 이미지를 재사용하되 컬러만 프롬프트로 지정하지 말고, 필요하면 새 기준 이미지를 다시 한 번(2크레딧) 생성해 별도 파일로 고정한다 (`block_reference_02.png` 등).
