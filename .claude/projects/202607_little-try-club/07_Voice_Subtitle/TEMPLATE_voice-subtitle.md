## 자막 스타일 가이드 (편집 툴에서 적용)

- 폰트: 둥글둥글한 산세리프 (Pretendard, 여기어때 잘난체 등 키즈 친화적)
- 위치: 화면 하단 1/6 지점, 배경 오브제와 겹치지 않게
- 색상: 크림 화이트 텍스트 + 소프트 그림자 (배경 컬러에 따라 가독성 확보)
- 타이밍: 컷당 최대 1줄, 2-3초 노출
- **원칙**: 자막은 있어도 되지만 없어도 스토리가 이해되게 — 보조 수단일 뿐

## 효과음(SFX) 체크리스트

- [ ] 블록 쌓기 소리 (톡, 탁)
- [ ] 블록 무너지는 소리 (와르르, 통통통)
- [ ] 캐릭터 숨소리/작은 리액션 (선택, 과하지 않게)
- [ ] 완성 순간 챠임/반짝임 소리
- [ ] 배경 앰비언스 (아주 옅게, 있는 듯 없는 듯)

**주의**: 효과음이 너무 만화적(과장된 "뿅!", "삐용삐용")이면 브랜드 톤(빈티지 그림책+패션 에디토리얼)과 어긋남. 자연스럽고 촉감 있는 소리를 우선한다.

## 제작 방법 옵션

### A. Higgsfield로 생성
- `generate_audio` 툴로 BGM/SFX 초안 생성 가능
- 장점: 빠름, 브랜드 무드 프롬프트로 직접 컨트롤
- 단점: 크레딧 소모, 사람이 만든 것 같은 미묘한 뉘앙스는 부족할 수 있음

### B. 무료 라이브러리 활용
- YouTube 오디오 라이브러리 (완전 무료, 상업적 이용 가능)
- Epidemic Sound / Artlist (유료 구독, 퀄리티 높음)

### C. 직접 녹음/제작
- 실제 블록 소리 녹음 → 가장 자연스러운 촉감
