# 썸네일 디자인 가이드 (Little Try Club)

> **목적**: 에피소드마다 썸네일 톤이 흔들리지 않도록, 레이아웃/타이포/색을 고정하고
> 에피소드별로 바뀌는 부분(문구·이미지)만 갈아끼우는 방식으로 제작한다.
> **참조 원본**: EP001 "다시, 블록" 썸네일 (2026.07.09 확정, 첨부해주신 빈티지 코믹북 커버 무드 기반)

---

## 1) 캔버스 규격

- **16:9, 1920×1080px**
- 배경색: 크림/오프화이트 `#F3ECD8` (빈티지 종이 느낌)
- 전체에 은은한 halftone 도트 텍스처 (`opacity: 0.06`, 네이비 `#1a2340` 도트) — 인쇄물 같은 질감

## 2) 고정 레이아웃 구조 (7개 존)

에피소드가 바뀌어도 이 배치는 그대로 유지한다:

| 존 | 위치 | 내용 | 에피소드마다 바뀌는가 |
|---|---|---|---|
| 브랜드 리본 | 좌상단 | 로고 + "LITTLE TRY CLUB" 워드마크 | ❌ 고정 |
| 코너 배지 | 우상단 원형 | "EP.XX / SHORTS" | ✅ 회차 번호만 |
| 타이틀 블록 | 좌측 중단 | 아이브로우 태그 + 2줄 헤드라인(에피소드 제목) + 서브헤드(훅 문구) | ✅ 문구 |
| 코믹 패널 그리드 | 우하단, 배경 | 그 에피소드의 6패널 스토리보드(또는 대표 컷들) | ✅ 이미지 |
| 히어로 버스트 | 우상단, 별 모양 패널 | 그 컷의 감정이 가장 큰 순간(놀람/기쁨 등) 얼굴 클로즈업 | ✅ 이미지 |
| 말풍선 | 히어로 버스트 옆 | 짧은 감탄사 (1-3음절) | ✅ 문구 |
| 하단 태그 | 좌하단 | 시리즈 태그라인 | ❌ 거의 고정 (필요시만 수정) |

## 3) 컬러 팔레트 (브랜드 고정)

- 크림 배경: `#F3ECD8`
- 잉크/아웃라인: `#1a1a1a` (모든 요소에 두꺼운 블랙 아웃라인 — 이게 "빈티지 코믹" 느낌의 핵심)
- 코발트 블루: `#2A4BD7`
- 토마토 레드: `#D9432E`
- 버터 옐로우(헤드라인 채우기): `#F6D33C`

이 5색 외에 새 색을 추가하지 않는다. 오브제/배경 컬러는 브랜드 팔레트(비주얼 톤 바이블)와 이미 일치하므로 자동으로 맞음.

## 4) 타이포그래피

- **영문/숫자**: Fredoka Bold — 워드마크, 배지 숫자용. `00_Brand_Bible/fonts/fredoka-bold.woff2`에 이미 저장되어 있어 **매번 새로 받을 필요 없음**.
- **한글**: Black Han Sans — 헤드라인, 서브헤드, 아이브로우 태그, 말풍선. 코믹 포스터용 임팩트 서체.
  - ⚠️ 한글 폰트는 매 에피소드 **그 회차에 실제로 쓰는 텍스트만** 서브셋으로 새로 받아야 함 (아래 5-2 참고). 폰트 파일을 통째로 저장해두지 않는 이유: 전체 한글 커버 시 파일이 너무 커짐. 텍스트 서브셋 방식이 가볍고 빠름.

## 5) 제작 레시피 (에피소드별로 반복하는 절차)

### 5-1. 소스 이미지 고르기
- **히어로 버스트용**: 그 에피소드에서 감정이 가장 크게 드러나는 순간의 승인된 최종 스틸 1장 (놀람/집중/기쁨 등 — "훅"이 되는 표정). 정사각형으로 얼굴 위주 크롭.
- **패널 그리드용**: 그 에피소드의 6패널 통합 스토리보드 이미지 (`storyboard_*.png`), 없으면 6컷 최종 스틸을 나란히.

### 5-2. 얼굴 크롭 좌표 잡기 (크레딧 불필요)
```python
from PIL import Image
im = Image.open('cutN_9x16_final.png')
crop = im.crop((x1, y1, x2, y2))  # 정사각형이 되도록, 얼굴이 중앙에 오도록 시행착오로 조정
crop.save('face_crop.png')
```
결과를 Read 툴로 직접 확인하면서 얼굴이 프레임 중앙에 잘 들어올 때까지 좌표 조정.

### 5-3. 한글 폰트 서브셋 받기
그 에피소드 썸네일에 실제로 쓸 한글 텍스트(헤드라인+서브헤드+말풍선+태그)를 모아서:
```bash
python3 -c "import urllib.parse; print(urllib.parse.quote('여기에 실제 텍스트 전부'))"
curl -sL -A "Mozilla/5.0 ..." \
  "https://fonts.googleapis.com/css2?family=Black+Han+Sans&text=<위 인코딩값>&display=swap" \
  -o subset.css
# subset.css 안의 https://fonts.gstatic.com/l/font?... URL을 다시 curl로 받으면 소용량 woff2 완성
```

### 5-4. 템플릿에 값 채우기
`TEMPLATE_thumbnail.html`을 읽고, 아래 플레이스홀더를 실제 base64/텍스트로 치환한 사본을 생성 (원본 템플릿 파일은 건드리지 않음):

| 플레이스홀더 | 채울 값 |
|---|---|
| `__BLACKHAN_FONT_B64__` | 5-3에서 받은 서브셋 폰트 base64 |
| `__HERO_IMG_B64__` | 5-2 얼굴 크롭 이미지 base64 (JPEG) |
| `__HERO_BG_SIZE__` / `__HERO_BG_POSITION__` | 보통 `118%` / `46% 40%`부터 시작, 렌더 후 눈으로 보고 미세조정 |
| `__GRID_IMG_B64__` | 6패널 스토리보드 base64 (JPEG, 너비 1700px 정도로 리사이즈) |
| `__LOGO_IMG_B64__` | 로고 (고정, `00_Brand_Bible/logo.svg`를 렌더링한 PNG) |
| `__EP_NUMBER__` | "02", "03" 등 |
| `__EYEBROW_TAG__` | 짧은 태그 (예: "오늘의 도전!") |
| `__HEADLINE_LINE1__` / `__HEADLINE_LINE2__` | 에피소드 제목 2줄 분할 |
| `__SUBHEAD__` | 한 줄 훅 문구 |
| `__BUBBLE_TEXT__` | 감탄사 1-3음절 |
| `__BOTTOM_TAG__` | 보통 고정: "30초 챌린지 · 리틀 트라이 클럽" |

### 5-5. 렌더 & 검수
Playwright로 1920×1080 스크린샷 → Read 툴로 직접 확인 → 어색하면 5-2/5-4 좌표만 조정해서 재렌더 (크레딧 소모 없음, 전부 로컬 합성).

### 5-6. 저장
`06_Thumbnail/EP0XX_[에피소드명]/thumbnail_16x9_v1.png`로 저장, git 커밋.
