S1 · Hero 섹션 설계 의도
Design Rationale
다크 배경(#111111)으로 기술적 무게감을 먼저 확보. 제품 비주얼 대신 포뮬러 텍스처 그라디언트를 전면에 배치해 "이 제품은 기술이다"를 첫인상으로 전달. Brass italic 레이블이 고급스러운 기술 브랜드 코드를 심는다.
헤드라인 구조: 색 선택의 자유(감성) + 포뮬러의 적응력(기술)을 한 문장으로 압축.

가격 정보를 Hero 하단에 배치해 신뢰가 형성된 직후 구매 의향을 자극.
#다크배경 #텍스처비주얼 #가격앵커링
S2 · Formula Layer
Data Card 전략
3개의 수치 카드(밀착력·지속력·발색도)는 Mono 폰트 + Brass 컬러로 데이터 신뢰감을 극대화. 숫자 자체가 설득의 핵심 — 경쟁사 대비 우위 수치로 채울 것을 권장 (현재 Placeholder).
성분 배지(Formula Badge)는 #F5F5F5 배경으로 부드럽게, Coral/Brass 강조 배지로 핵심 기술 성분을 부각. 과학적 신뢰감과 가독성을 동시에 달성.

S3 · Color True 비교 블록
CSS 컬러 블록으로 "바르기 전(피부 본연색) vs 바른 후(정확한 발색)"을 시각화. 실제 Before/After 사진 없이도 컬러 대비만으로 발색 신뢰감을 전달하는 접근.

"피부 위에서 맞춰주는" 카피를 다크 박스 안에 배치해 기억 각인 효과.
S4·S5 · Texture + Skin Test
신뢰 레이어 강화
텍스처 시각블록은 Grid 패턴 + Radial Gradient로 "실사 클로즈업"을 CSS로 재현. 건성/지성/복합성 3종 피부 테스트 카드는 구체적 수치 점수로 객관성을 확보.
이 두 섹션이 C05의 핵심 차별점: 감성이 아닌 데이터와 검증으로 설득하는 구조.
S6 · OliveYoung ONLY 큐레이션
Brass 배지 + 다크 배경으로 독점 큐레이션의 희소성을 극대화.

큐레이터 코멘트 형식으로 올리브영 MD의 선택 근거를 서술 — 브랜드 일방 주장이 아닌 채널 큐레이션 신뢰를 활용.
S7·S8 · 리뷰 + CTA 전략
리뷰 카드는 성분·지속력 특화 리뷰만 노출 — 컨셉 C05의 포뮬러 신뢰 메시지와 사용자 UGC가 일치하도록 큐레이션.

CTA 서브카피 "이 색이 나한테 어울리는지는, 담고 나서 알게 된다."는 구매 망설임을 인정하면서 역설적으로 클릭을 유도하는 심리적 장치.
