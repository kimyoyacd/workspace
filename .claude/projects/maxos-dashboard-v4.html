사용법

트리거를 말하면 알맞은 에이전트가 작동합니다.

① 제안 풀세트 → "○○ 제안 풀세트 돌려"

RFP→시장→관점→발산→견적→검수가 순서대로. 실장님은 셀렉·OK만.

② 관점부터 → "이 제품 관점 한 문장으로"

concept-director가 관점 고정 → 모든 디자인 상속.

③ 견적 → "코웨이 UI Design 고급1 3.5MM 100%, 견적"

quote-accountant가 원가표+마진+우리실 매출(UI분). 최종가는 실장님.

④ 현황 → "이번 주 현황" / "가동률"

project-manager가 노션·시트 읽어 진행보드+달성율+가동률+오류.

⑤ 생성 → "키비주얼 뽑아" / "무드보드" / "Figma 보내"

visual-generator·moodboard-builder·figma-bridge·storyboard-maker.

⑥ 검수 → "시안 봐줘" / "계약서 검토" / "발송 전 점검"

design-critique·legal-compliance·delivery-gate 등.

핵심: 말하기 → 에이전트가 일함 → 실장님 최종 결정.
