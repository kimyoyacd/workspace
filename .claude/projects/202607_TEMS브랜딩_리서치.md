## 3. 그룹 분류 (발산 3방향)
- **G1 캐주얼 취준생 플랫폼** (잡다·캐치·링커리어·코멘토) — 비비드 원색, B2C 커뮤니티 감성. 대학 채널은 잡았으나 '기관 신뢰' 톤 아님.
- **G2 사이언스·엔터프라이즈 어세스먼트** (SHL·HireVue·Eightfold·Harver·뷰인터) — 네이비/블루, 데이터 권위. '평가받는 자'에겐 차갑고 위협적.
- **G3 경험·판단 성장 시스템** (Forage·Socra·Aera) — TEMS의 본적지. 이 그룹에서 '판단 데이터'를 정체성으로 시각화한 브랜드는 아직 없음.

## 4. XY 포지셔닝 맵

### (a) 시장 축 — X: 평가·선별 ↔ 경험·성장 / Y: B2C 개인 ↔ B2B 기관
```
                        B2B·기관 신뢰 (Y+)
                              │
   SHL ● HireVue ● Eightfold  │            Aera ●
   뷰인터 ●        Harver ●    │      ★ TEMS 목표 지점
                              │         (Forage △ 절반)
 평가·선별 (X-) ───────────────┼─────────────── 경험·성장 (X+)
                              │
        잡다 ●                 │   코멘토 ● Forage ●
        캐치 ●                 │   링커리어 ●  뤼이드/Socra ●
                              │
                        B2C·개인 캐주얼 (Y-)
```
- 좌상단(기관용 평가)·하단(개인용 취준 콘텐츠) 과밀. **공백 = 우상단 "기관(대학)에 공급되는 성장·판단 시스템"**.

### (b) 비주얼 톤 축 — X: 테크·차가움 ↔ 휴먼·따뜻함 / Y: 스타트업 캐주얼 ↔ 기관 신뢰
```
                       기관 신뢰·격식 (Y+)
                              │
  Eightfold ● Aera ●          │
  HireVue ● 뷰인터 ●           │     ★ 화이트스페이스
        SHL ●(반보)            │    "따뜻한 신뢰" — 비어 있음
                              │
 테크·차가움 (X-) ──────────────┼────────────── 휴먼·따뜻함 (X+)
                              │
   뤼이드(구) ●                 │  잡다 ● 캐치 ● 링커리어 ●
                              │  코멘토 ● Forage ● Harver ●
                              │
                       스타트업 캐주얼 (Y-)
```
- **우상단 "신뢰의 격식 + 사람의 온기"는 전 세계적으로 희박** — HireVue·뤼이드의 리브랜딩 이동 방향이 수요의 방증.

## 5. 화이트스페이스 결론
1. **본명당 — "따뜻한 신뢰" (기관 격식 × 휴먼 온기)**: 딥 네이비/딥 그린 급 신뢰 베이스 + 웜 액센트 1색. "AI가 평가하는 게 아니라 판단을 함께 길러준다"는 톤. G1은 B2C DNA라 못 가고 G2는 엔터프라이즈 관성이라 안 가는 자리.
2. **"판단의 구조" 심볼화 — HR/교육 DI 최초 선점**: 케이스×판단축·플라이휠·노드-연결 등 판단 데이터 구조 자체를 심볼로. SHL '연결의 점'이 유일한 유사 선례. 로고 심볼 방향 직결.
3. **함정 주의 — 순수 시뮬레이션·게임 톤**: 학생에겐 먹히나 신생 기업이 대학 조달·심사 관문에서 '검증 안 된 게임 업체'로 격하될 위험. 시뮬레이션은 제품 언어로만, 브랜드 톤은 1번으로.
