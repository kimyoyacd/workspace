# 프로젝트 폴더 규약

각 클라이언트/프로젝트는 `.claude/projects/[클라명]/` 아래에 둔다.

## 구조
```
.claude/projects/[클라명]/
  STATUS.md      # 현황 요약 (read 기준, 원본은 Notion)
  [산출물]       # 시안·제안·견적 등
```

## 규칙
- **새 폴더 생성 전 유사 폴더 중복 점검 필수.**
- STATUS.md는 Notion 2026 Project DB의 미러 — 쓰기는 사용자 직접.
- 민감정보(단가·개인정보)는 외부 노출 금지.

## STATUS.md 템플릿
```
# [클라명] — STATUS
- 진행 프로젝트:
- 기일:
- 최근 커뮤니케이션:
- 리스크:
- 결정 대기:
```
