## 🛡️ 크레딧 절약 전략

### 프롬프트 최적화 원칙

1. **첫 생성 전 프롬프트 검증**
   - [ ] Higgsfield 프롬프트 템플릿 100% 준수
   - [ ] 네거티브 프롬프트 완벽히 작성
   - [ ] 캐릭터 외형 설명 3번 이상 검토
   - [ ] 배경/오브제 설명이 명확한가 확인

2. **생성 재시도 최소화**
   - 한 번에 원하는 결과를 얻도록 프롬프트 정확성 극대화
   - 재생성 필요 시 프롬프트 수정 사항 명확히 기록

3. **배치 처리 고려**
   - 같은 캐릭터/배경의 컷들을 연속 생성할 때는 프롬프트 템플릿 활용
   - 컷 간 일관성을 유지하면서 생성 횟수 줄이기
