# 202607 결 (gyeol) 프로토타입

## 개요
- **클라이언트**: 내부 (MX 스튜디오 자체 툴)
- **업무 범위**: 디자인 키워드 → 멀티모달 프롬프트 번역기 웹 프로토타입 v0.1
- **일정**: 2026-07 프로토타입 → 피드백 후 v0.2 방향 결정
- **견적**: 해당 없음 (내부 R&D)

## 컨셉
한국어 디자인 키워드('풍')를 입력하면:
1. **확장 키워드** — 연상어 + 의성어·의태어, 영문 병기
2. **모달별 프롬프트** — 사진 / 타이포 / 영상 각각 영문 생성 프롬프트
3. **태그** — 발상의 원천(#원본-키워드) + 톤·렌즈·서체계열 등 역추적 태그
4. **기술 스펙** — 카메라·렌즈·조명 / 서체 분류·자간 / 프레임레이트·컷 길이

## 산출물
- `prototypes/gyeol/index.html` — 단일 파일 프로토타입 (의존성 없음)
- 수록 데이터: 키워드 10종 × 3모달 = 프롬프트 카드 30장 수동 큐레이션
  - 우아한 · 몽환적인 · 강렬한 · 아늑한 · 차가운 · 쓸쓸한 · 반짝반짝 · 폭신폭신 · 아련한 · 쨍한

## 기능 (v0.5)
- 키워드 검색 (부분 일치 + 확장 키워드 역검색)
- 모달 필터 탭 (전체 / 사진 / 타이포 / 영상)
- 프롬프트 원클릭 복사
- 보관함 (localStorage 저장 — 팀 라이브러리의 씨앗)
- **AI 툴 궁합** — 카드마다 이 무드가 잘 나오는 툴 표기 (Midjourney / Seedream / Seedance / Veo 3 / Kling / Runway / FLUX / Ideogram / Recraft, 최적·적합 2단계 + 이유)
- **이미지 첨부** — 무드 영역에 업로드(캔버스 자동 압축 후 localStorage 저장)·드래그&드롭·URL 붙여넣기, 미첨부 시 키워드별 그라디언트 스와치
- **한글 번역 토글** — 영문 프롬프트 아래 '한국어 번역 ▾' 클릭 시 병기 (30종 전체 수동 번역)
