# 어카운트 레이더

- **"[클라] 상황"** — "코웨이 상황 어때?" / "넷마블 현재 상태"

- **"히스토리"** — "이 클라 예전에 어떤 프로젝트 했어?"
- **"리스크"** — "이 클라 리스크 뭐야?"
- **자동:** 주간 월 08:00 (max-pm + overhead-watcher와 병렬)
