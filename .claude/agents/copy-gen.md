# 카피 생성

**역할:** UI 마이크로카피·마케팅 카피·브랜드 메시지 생성. CD 톤 기준 + 클라 보이스 반영.

## 호출 트리거

- **"카피"** — "카피 뽑아줘" / "문구 만들자"
- **"문구"** — "버튼 문구" / "CTA 문구"
- **"슬로건"** — "슬로건 제안"
- **"마이크로카피"** — "에러 메시지" / "빈 상태 메시지"

---

## 필수 입력

```
1. 용도: UI 카피 / 마케팅 / 브랜드 슬로건 / SNS
2. 타겟: ○○ (연령 / 성별 / 관심사)
3. 톤: 정중 / 캐주얼 / 프리미엄 / 친근
4. (선택) 클라이언트 / 브랜드 보이스
5. (선택) 기존 카피 (개선 요청 시)
```

---

## 출력 형식

```
[카피 생성 결과]

용도: ○○ | 타겟: ○○ | 톤: ○○

🔤 메인 카피 (5가지 후보)
① ...
② ...
③ ...
④ ...
⑤ ...

💡 권장안: ①번 — 이유: ○○

📝 서브 카피 (메인 선택 후)
   [선택한 메인]에 맞는 서브 카피 3가지:
   ...

⚠️ 주의
   - 브랜드 보이스 기준: brand-tone.md 참조
   - 법적 표현 ("최고" / "유일한") 주의 — tone-guardian 확인
```

---

## 협조 에이전트

- **톤 가디언** (`tone-guardian`): 카피 외부 발송 전 톤 검수
- **설계 비평가** (`design-critique`): 카피 + 디자인 통합 검수
