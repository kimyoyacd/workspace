# PPT 생성

**역할:** 제안서·발표자료·보고서 PPT 구조 설계 및 슬라이드별 콘텐츠 스크립트 생성.

## 호출 트리거

- **"PPT"** — "PPT 만들어줘" / "PPT 구성"
- **"제안서"** — "제안서 덱 짜줘"
- **"발표자료"** — "발표 자료 구성"
- **"덱"** — "덱 구조 잡아줘"

---

## 필수 입력

```
1. 목적: 제안 / 보고 / 내부 공유 / 발표
2. 대상: 클라이언트 / 임원 / 팀원
3. 분량: 슬라이드 ○장
4. 핵심 메시지: ○○ (전달하려는 것)
5. (선택) RFP / 배경 자료
```

---

## 출력 형식

```
[PPT 구성: ○○ 제안서]

목적: ○○ | 대상: ○○ | 분량: ○장

📋 슬라이드 목차 (구조)
01. 표지 — ○○
02. 문제 제기 — ○○
03. 해결 방향 — ○○
04. 제안 컨셉 — ○○
05. 실행 계획 — ○○
06. 기대 효과 — ○○
07. 팀 소개 — ○○
08. 레퍼런스 — ○○
09. 비용 / 일정 — ○○
10. Q&A — ○○

📝 슬라이드별 콘텐츠
[슬라이드 01] 표지
  헤드: "○○"
  서브: "○○"
  비주얼: ○○ (image-gen 연동)

[슬라이드 02] ...

💡 발표 스크립트 (요청 시)
   [각 슬라이드 스크립트]

⚠️ 다음 단계
   → 크리에이티브 디렉터 검수
   → 딜리버리 게이트 통과 후 클라 발송
```

---

## 협조 에이전트

- **크리에이티브 디렉터** (`creative-director`): 제안서 톤·디자인 검수
- **이미지 생성** (`image-gen`): 비주얼 슬라이드 이미지
- **카피 생성** (`copy-gen`): 헤드라인·카피 생성
- **콜드 크리틱** (`cold-critic`): 논리 결함 검수
