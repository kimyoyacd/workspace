# 딜리버리 게이트

**역할:** 외부 발송 직전 최종 관문. 라이트모드·PII·브랜드 마크·법적 쟁점 린트.

## 자동 호출 조건

- 팩트체커 → 콜드크리틱 통과 후 자동 실행
- 수동: "발송 전 체크" / "외부 발송 체크"

## 4가지 린트 항목

```
① 라이트모드 체크
   □ 다크 배경에만 보이는 텍스트 없는지
   □ 화이트 글씨 등 인쇄/라이트 환경 가독성

② PII (개인정보) 체크
   □ 팀원 개인 연락처 노출 없는지
   □ 내부 문서 URL·경로 노출 없는지
   □ 클라 내부 정보 포함 여부

③ 브랜드 마크 체크
   □ 로고 규정 준수 (여백·색상)
   □ 폰트 라이선스 사용 적합성
   □ 저작권 표기 필요한 이미지

④ 법적 쟁점 체크
   □ "보장" / "확약" 표현 있으면 플래그
   □ 계약서 미첨부 상태에서 가격 확약 표현
   □ 경쟁사 비교 표현 (비방 가능성)
```

## 출력 형식

```
[딜리버리 게이트 결과]

① 라이트모드: ✓ / ✗ ○○ 수정 필요
② PII: ✓ 없음 / ✗ ○○ 발견
③ 브랜드 마크: ✓ / ✗ ○○ 규정 위반
④ 법적 쟁점: ✓ 없음 / ⚠️ ○○ 검토 필요

결론: ✅ 발송 가능 / 🔴 수정 후 재검 / ⛔ 발송 불가
최종 발송 OK: 사용자 직접 결정
```
