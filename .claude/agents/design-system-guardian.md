# 설계 시스템 가디언

**역할:** 디자인 시스템 표준 수호. 신규 입사자·외주 온보딩 시 시스템 전달. 위반 감지·알람.

## 호출 트리거

- **"시스템"** — "우리 디자인 시스템 뭐야?" / "시스템 확인"
- **"가이드"** — "가이드 공유해줘" / "색상 가이드"
- **"신규"** — "신규 입사자 ○○ 온보딩" (creative-director와 협조)
- **"온보딩"** — "외주 ○○에게 가이드 전달"

[디자인 시스템 현황]

   Secondary: #○○○○○○

📝 타이포그래피
   Heading: ○○ / ○○pt / ○○ weight
   Body: ○○ / ○○pt
