# Hyojung Guided Agent · Question Flow Runbook

## 첫 실행 프롬프트

```md
Hyojung Guided Brand Proposal Agent로 시작합니다.
먼저 Stage 0 Intake만 진행하세요.
한 번에 많은 질문을 하지 말고, 지금 필요한 질문 3개만 해주세요.
아직 리서치, 컨셉, 이미지 프롬프트, 슬라이드 HTML은 생성하지 마세요.
```

## Stage 0 Intake 질문 템플릿

```md
현재 단계는 Stage 0 · Intake입니다.
지금은 프로젝트의 출발 정보를 확인하는 단계입니다.

질문은 3개만 드리겠습니다.

1. 브랜드명/제품명과 카테고리는 무엇인가요?
2. 목표 고객과 원하는 인상은 무엇인가요?
3. 최종 산출물은 무엇인가요? 예: 브랜드 제안서, BX 보드, 패키지 시안, AI 이미지, HTML 슬라이드
```

## Stage 1 Research & Positioning 질문 템플릿

```md
현재 단계는 Stage 1 · Research & Positioning입니다.
이 단계에서는 경쟁 구도와 포지셔닝 빈자리를 찾습니다.

1. 경쟁 브랜드를 사용자가 제공하나요, 아니면 제가 30개 후보를 구성할까요?
2. 시장 범위는 국내/글로벌/특정 지역 중 어디인가요?
3. 포지셔닝에서 가장 중요한 축은 무엇인가요? 가격, 감성, 기능, 프리미엄, 문화, 헤리티지 중 선택해주세요.
```

## Gate 1 검토 질문

```md
Stage 1 결과를 검토하겠습니다.
아래 3가지만 확인해주세요.

1. 추천 포지셔닝 축이 설득력 있나요?
2. 경쟁 브랜드 30개 중 반드시 추가/삭제할 브랜드가 있나요?
3. 시장의 빈자리가 실제 브랜드 목표와 맞나요?

선택:
A. 그대로 다음 단계 진행
B. 경쟁사 목록 수정
C. 포지셔닝 축 다시 제안
```

## Stage 1.5 Strategic POV 질문 템플릿

```md
현재 단계는 Stage 1.5 · Strategic Point of View입니다.
비주얼로 가기 전에 전략 관점 한 문장을 잠가야 합니다.

1. 이 브랜드가 시장에 던질 주장은 기능 중심인가요, 감정 중심인가요, 문화 중심인가요?
2. 과감한 차별화와 안정적 설득 중 어느 쪽을 더 우선할까요?
3. 한 문장 관점의 톤은 단단하게, 시적으로, 직관적으로 중 어디에 가까워야 하나요?
```

## Gate 2 검토 질문

```md
Strategic POV 후보를 검토하겠습니다.

1. 가장 마음이 가는 POV는 A/B/C 중 무엇인가요?
2. 너무 추상적이거나 평범한 문장이 있나요?
3. 3개 키워드 중 바꾸고 싶은 단어가 있나요?

선택:
A. POV 확정
B. 더 시적으로 다시
C. 더 B2B 설득형으로 다시
```

## Stage 2 Visual Identity 질문 템플릿

```md
현재 단계는 Stage 2 · Visual Identity & Concept System입니다.
이제 관점을 비주얼 시스템으로 바꿉니다.

1. 이 브랜드에서 보존해야 할 자산은 무엇인가요? 로고, 컬러, 헤리티지, 고객 인식 등.
2. 반대로 과감하게 바꿔도 되는 부분은 무엇인가요?
3. AI 이미지에 남길 휴먼터치 레이어는 낮음/중간/높음 중 어느 정도가 적절한가요?
```

## Gate 3 검토 질문

```md
Visual Identity 결과를 검토하겠습니다.

1. 6개 컨셉 중 브랜드와 가장 잘 맞는 방향은 무엇인가요?
2. 너무 유행처럼 보이거나 클라이언트가 부담스러워할 방향은 무엇인가요?
3. 모션/가변 타입/반응형 규칙으로 확장할 방향이 있나요?

선택:
A. 2~3개로 압축
B. 컨셉 6개 다시 발산
C. 휴먼터치 레이어만 조정
```

## Stage 3 Brand System & Slide HTML 질문 템플릿

```md
현재 단계는 Stage 3 · Brand System & Slide HTML Export입니다.
최종 산출물 생성을 준비합니다.

1. 최종 슬라이드는 몇 장 정도가 적절한가요? 15장, 30장, 45장 중 선택해주세요.
2. 이미지 프롬프트까지 생성할까요? 생성한다면 비율은 1:1, 3:4, 16:9 중 무엇인가요?
3. HTML은 발표용, 내부 검토용, Figma 이식용 중 무엇으로 만들까요?
```

## Final Review 질문 템플릿

```md
최종 생성 전 마지막으로 확인합니다.

1. 최종 POV를 그대로 표지와 첫 장에 사용할까요?
2. A/B/Alt 컨셉 구조로 보여줄까요, 단일 최종안으로 보여줄까요?
3. HTML 슬라이드 생성까지 진행할까요?

선택:
A. HTML 슬라이드 생성
B. 이미지 프롬프트만 먼저 생성
C. 브랜드 시스템 표만 먼저 생성
```

## 사용자가 “계속”이라고 할 때의 규칙

```md
사용자가 세부 답변 없이 “계속”이라고 하면 아래 원칙으로 진행합니다.

1. 가장 일반적인 B2B 제안 기준으로 가정합니다.
2. 가정은 반드시 Assumptions에 표시합니다.
3. 다음 Gate에서 사용자가 수정할 수 있게 합니다.
4. 최종 발송/제출/확정은 절대 자동으로 하지 않습니다.
```
