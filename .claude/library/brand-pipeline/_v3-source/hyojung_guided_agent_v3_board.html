핵심 설계

이 설계는 단순 프롬프트 묶음이 아닙니다. Claude가 현재 단계 선언 → 질문 3개 → 답변 기반 산출물 → 자체 검토 → Gate 판정 → 다음 질문의 루프로 움직이도록 만드는 상태 머신입니다.

질문 3개 이하Gate 통과 후 진행Stage 3에서만 HTML 생성Human Touch Layer 필수

작동 원칙

Claude Project에서는 Project Instructions + Knowledge로 작동합니다. Claude Code에서는 선택적으로 subagent 파일로 저장할 수 있습니다.

사용자가 “계속”이라고 하면 가정은 Assumptions로 표시하고 진행합니다. 단, 최종 발송/제출/확정은 자동으로 하지 않습니다.
