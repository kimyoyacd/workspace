# Hyojung Guided Agent · State Machine & Review Gates

## State Machine Overview

```text
START
  ↓
Stage 0 · Intake
  ↓ Gate 0: Brief Completeness
Stage 1 · Research & Positioning
  ↓ Gate 1: Research Logic / Source / Gap
Stage 1.5 · Strategic Point of View
  ↓ Gate 2: POV One Sentence / 3 Keywords
Stage 2 · Visual Identity & Concept System
  ↓ Gate 3: Visual DNA / Behavior / Human Touch
Stage 2.5 · Concept Selection
  ↓ Gate 4: Distinction / Risk / Preserve-Destroy
Stage 3 · Brand System & Slide HTML Export
  ↓ Gate 5: Deck Readiness
Final Review
  ↓
EXPORT HTML + IMAGE PROMPT PACK
```

## Stage 0 · Intake

### 목적
브랜드 제안의 출발 정보를 확보합니다.

### 질문 3개
1. 이 브랜드/제품은 어떤 카테고리이며, 최종 산출물은 무엇인가요?
2. 주요 타깃과 원하는 인상은 무엇인가요?
3. 피해야 할 인상, 금지 스타일, 클라이언트 민감 이슈가 있나요?

### Gate 0
- 브랜드명 또는 프로젝트명이 있는가?
- 카테고리와 산출물이 명확한가?
- 타깃과 피해야 할 방향이 최소한 정의됐는가?

Green: 다음 단계 진행
Yellow: 1~2개 질문 보완
Red: Intake 재작성

## Stage 1 · Research & Positioning

### 목적
와이드 리서치, 경쟁 구도, 포지셔닝 빈자리를 찾습니다.

### 질문 3개
1. 경쟁 브랜드를 이미 알고 있나요, 아니면 AI가 30개 후보를 구성할까요?
2. 시장 범위는 국내, 글로벌, 특정 지역 중 어디인가요?
3. 포지셔닝은 가격/감성/기능/문화/프리미엄 중 어떤 축이 중요해 보이나요?

### 산출물
- 경쟁 브랜드 30개 리서치 매트릭스
- 포지셔닝 맵 축 3안
- 추천 포지셔닝 맵 1안
- 선호도 분석
- 시장 빈자리

### Gate 1
- 경쟁군이 너무 좁지 않은가?
- 포지셔닝 축이 전략적으로 의미 있는가?
- 빈자리가 실제 고객 욕구와 연결되는가?
- 출처 없는 수치가 단정으로 들어가지 않았는가?

## Stage 1.5 · Strategic Point of View

### 목적
리서치에서 바로 비주얼로 가지 않고, 한 문장 관점을 잠급니다.

### 질문 3개
1. 이 브랜드가 시장에서 새롭게 주장할 한 문장은 어떤 방향이어야 하나요?
2. 기능/감정/문화/헤리티지/미래성 중 무엇을 중심에 둘까요?
3. 과감한 차별화와 안정적 설득 중 어느 쪽이 더 중요한가요?

### 산출물
- Strategic POV 후보 5개
- 최종 POV 후보 2개
- 3 Core Keywords
- THE ANSWER

### Gate 2
- 관점이 한 문장으로 말 되는가?
- 경쟁사도 똑같이 말할 수 있는 문장인가?
- 3개 키워드가 톤, 컬러, 카피, 화면으로 확장 가능한가?

## Stage 2 · Visual Identity & Concept System

### 목적
관점을 비주얼 DNA와 컨셉 시스템으로 변환합니다.

### 질문 3개
1. 이 브랜드에서 보존해야 할 자산은 무엇인가요?
2. 과감하게 바꿔도 되는 자산은 무엇인가요?
3. AI 이미지에 남길 휴먼터치 레이어는 어느 정도가 적절한가요? 낮음 / 중간 / 높음

### 산출물
- Visual Identity DNA
- Concept Territories 6개
- Client × Mood Matrix
- Behavior System 초안
- Human Touch Layer
- Preserve vs Destroy Map

### Gate 3
- 컨셉들이 서로 충분히 다른가?
- 모든 컨셉이 같은 전략 관점으로 회수되는가?
- 정적 이미지가 아니라 행동하는 시스템으로 확장되는가?
- 휴먼터치가 장식이 아니라 브랜드 이유와 연결되는가?

## Stage 2.5 · Concept Selection

### 목적
6개 컨셉에서 2~3개를 추리고, 최종 시스템 후보를 정합니다.

### 질문 3개
1. 가장 설득력 있는 컨셉은 무엇인가요?
2. 클라이언트가 받아들이기 어려운 리스크는 무엇인가요?
3. A/B/Alt 구조로 보여줄까요, 단일 방향으로 압축할까요?

### Gate 4
- A/B가 진짜 다른가?
- Alt가 안전망 역할을 하는가?
- 리스크 정렬 체크를 통과했는가?

## Stage 3 · Brand System & Slide HTML Export

### 목적
브랜드 시스템, 이미지 프롬프트, 최종 슬라이드 HTML을 생성합니다.

### 질문 3개
1. 최종 슬라이드 분량은 어느 정도인가요? 15장 / 30장 / 45장
2. AI Studio에서 이미지까지 생성할 프롬프트가 필요한가요?
3. 최종 HTML은 발표용, 검토용, Figma 이식용 중 무엇인가요?

### 산출물
- Brand System Summary
- Value Cards
- Keyword Map
- Copy Headline System
- Symbol Association
- Image Prompt Pack
- Slide HTML Structure
- Final HTML

### Gate 5
- 모든 슬라이드가 전략 관점 한 문장으로 환원되는가?
- 각 슬라이드에 Key Message가 하나만 있는가?
- 이미지 프롬프트가 브랜드 시스템과 연결되는가?
- HTML이 라이트모드, 텍스트 가독성, 구조적 위계를 만족하는가?
