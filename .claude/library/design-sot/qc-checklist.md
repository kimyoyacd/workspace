# QC 체크리스트 SoT (6영역)

> design-critique 에이전트의 상세 기준입니다.
> 외부 발송 전 모든 산출물은 이 체크리스트 통과 필수.

## 영역 ① UI (유저 인터페이스)

```
필수 항목:
□ 버튼·링크 클릭 영역 최소 44×44px (모바일 기준)
□ 폼 요소 레이블 명확히 표기
□ 상태 표시: normal / hover / active / disabled
□ 여백·정렬 일관성 (8px 그리드 기준)
□ 반응형: 360px / 768px / 1280px 확인

권장 항목:
□ 스켈레톤 로딩 설계 여부
□ 에러 상태 디자인
```

## 영역 ② 콘텐츠 (텍스트·카피)

```
필수 항목:
□ 오탈자·문법 오류 없음
□ 국문/영문 혼용 일관성
□ 줄임말·약어 정의 표기
□ CTA 문구 명확성 (행동 유발)

권장 항목:
□ 읽기 쉬운 문장 길이 (1문장 50자 이내)
□ 능동형 표현 사용
```

## 영역 ③ GUI (그래픽 유저 인터페이스)

```
필수 항목:
□ 아이콘 스타일 통일 (라인 굵기·크기)
□ 색상 시스템 준수 (design-system.md 기준)
□ 이미지 해상도 최소 2× (Retina)
□ 그래픽 요소 정렬 (픽셀 퍼펙트)

권장 항목:
□ 모션 일관성 (이징·타이밍)
□ 다크모드 대응 (있을 경우)
```

## 영역 ④ UA (유저 어피어런스)

```
필수 항목:
□ 시각적 위계 명확 (F/Z 패턴)
□ 포커스 포인트 1개 이상 명확
□ 정보 밀도 적절 (여백 30% 이상 유지)
□ 색상 대비 4.5:1 이상 (본문)

권장 항목:
□ 첫 화면 3초 내 메시지 전달
□ 스크롤 없이 핵심 CTA 노출 (above the fold)
```

## 영역 ⑤ 상세페이지

```
(쇼핑·제품·서비스 상세 한정)

필수 항목:
□ 제품/서비스 핵심 이미지 3장 이상
□ 가격·구매 CTA 명확 위치
□ 상품 설명 계층 (특장점→상세→스펙)

권장 항목:
□ 신뢰 요소 (리뷰·배지·수상) 포함
□ 관련 상품 추천 섹션
```

## 영역 ⑥ 브랜딩

```
필수 항목:
□ 로고 여백 규정 준수 (클리어스페이스)
□ 브랜드 컬러 팔레트 준수
□ 지정 폰트 패밀리만 사용
□ 비공인 로고 변형 없음

권장 항목:
□ 브랜드 보이스 톤 일치 (brand-tone.md)
□ 경쟁사 유사 표현 없음
```
