# 6영역 QC 체크리스트 SoT (design-sot)

> 참조 에이전트: `design-critique`(6영역 비평), `creative-director`, `delivery-gate`
> ⚠️ 항목은 기본 골격. 실무 기준은 실장이 보강.

## ① UI
- [ ] 정보 위계가 명확한가
- [ ] 가독성(대비·크기·여백) 충분한가
- [ ] 인터랙션/상태(hover·active·disabled) 일관적인가

## ② 콘텐츠
- [ ] 카피가 메시지를 정확히 전달하는가
- [ ] 오탈자·맞춤법
- [ ] 정보 구조가 논리적인가

## ③ GUI (게임)
- [ ] 세계관·몰입과 일치하는가
- [ ] 상태(체력·재화·진행) 표현이 직관적인가
- [ ] 연출·이펙트 과/부족

## ④ UA (유저 획득·광고·배너)
- [ ] 3초 내 후킹되는가
- [ ] CTA가 명확한가
- [ ] 전환 유도 흐름

## ⑤ 상세페이지
- [ ] 스크롤 설계(도입→설득→마무리)
- [ ] 구매/행동 설득 근거
- [ ] 모바일 가독성

## ⑥ 브랜딩
- [ ] 브랜드 아이덴티티 정합
- [ ] 톤·매너(brand-tone.md) 일치
- [ ] 브랜드 자산(로고·컬러·폰트) 규정 준수

## 공통 게이트 (delivery-gate 연계)
- [ ] HTML 라이트모드 3종
- [ ] PII·민감정보 노출 없음
- [ ] 텍스트 잘림·깨진 마크업 없음
