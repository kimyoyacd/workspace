# 브랜드 톤 & 매너 SoT

> creative-director, design-critique, tone-guardian 에이전트의 판단 기준입니다.

## MAX실 크리에이티브 디렉션 톤

### 핵심 키워드
```
전문적 · 간결 · 자신감 · 진정성
```

### 비주얼 방향성

```
✅ 우리가 추구하는 것
- 절제된 레이아웃 (공백 활용)
- 타이포그래피 중심 위계
- 브랜드 아이덴티티 일관성
- 글로벌 감각의 로컬 적용
- 모션: 목적 중심, 과도한 효과 지양

❌ 피해야 할 것
- 트렌드만 좇는 표면적 스타일링
- 정보 과부하 (과도한 텍스트·요소)
- 클라 브랜드 무시한 개인 취향 표현
- 저해상도·저품질 에셋 사용
```

### 커뮤니케이션 톤

```
클라이언트에게:
- 전문적이지만 딱딱하지 않게
- 자신감 있지만 강요 없이
- 결과 지향적 제안 (문제→해결 구조)

팀원에게:
- 직관적 피드백 (구체적으로)
- 방향 제시 후 자율성 부여
- "안 됨"보다 "이렇게 해줘" 방식
```
