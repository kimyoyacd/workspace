# 디자인 시스템 SoT

> design-system-guardian 에이전트의 표준 기준입니다.
> 클라별 시스템이 아닌 MAX실 내부 제작 기준입니다.

## 컬러 시스템

```
Primary:    [직접 입력] #○○○○○○
Secondary:  [직접 입력] #○○○○○○
Accent:     [직접 입력] #○○○○○○
Neutral:    [직접 입력] (Gray scale)
Error:      [직접 입력] #○○○○○○

접근성 기준:
- 배경 대비 최소 4.5:1 (WCAG AA)
- 대형 텍스트 최소 3:1
```

## 타이포그래피

```
국문:
  Heading: [폰트명] / [크기범위] / [웨이트]
  Body:    [폰트명] / [크기] / Regular
  Caption: [폰트명] / [크기] / Regular

영문:
  Heading: [폰트명]
  Body:    [폰트명]

줄간격: 1.5× (기본) / 1.8× (장문)
자간: 국문 -2% / 영문 기본값
```

## 레이아웃 & 그리드

```
기본 그리드: 12컬럼
최대 너비: 1280px (웹), 390px (모바일)
거터: 24px (데스크탑) / 16px (모바일)
여백: 80px (섹션) / 40px (블록) / 16px (요소)
```

## 컴포넌트 표준

```
버튼:
  Primary: 배경 [Primary색] / 텍스트 White / 모서리 ○px
  Secondary: 테두리 [Primary색] / 배경 없음
  크기: Large 48px / Medium 40px / Small 32px

아이콘:
  스타일: [라인 / 필드 / 혼합]
  굵기: [직접 입력]
  크기: 24px (기본) / 16px (소)

카드:
  그림자: [직접 입력]
  모서리: [직접 입력]px
```
