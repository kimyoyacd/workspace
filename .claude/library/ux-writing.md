# UX 라이팅 레퍼런스 라이브러리

## 목적
제안서·랜딩·배너·UI 문구 작성 시 참조하는 톤·원칙 레퍼런스.  
`copy-writer` 에이전트 및 모든 HTML 산출물 작성 시 자동 참조.

---

## 1. 토스 UX 라이팅 원칙
> 출처: https://developers-apps-in-toss.toss.im/design/ux-writing.html

### 핵심 원칙
- **명확하게**: 사용자가 한 번 읽고 이해할 수 있어야 한다. 중의적 표현 금지.
- **간결하게**: 필요한 정보만 담는다. 장황한 설명보다 핵심 한 문장.
- **친근하게**: 딱딱한 경어가 아닌, 대화하듯 자연스러운 존댓말.

### 토스 톤 특징
| 좋은 예 | 나쁜 예 |
|--------|--------|
| "바로 시작해요" | "이용을 시작하시겠습니까?" |
| "잠깐 기다려 주세요" | "처리 중입니다. 완료 시까지 대기해 주시기 바랍니다." |
| "연결이 끊겼어요" | "네트워크 오류가 발생하였습니다." |
| "얼마나 보내실 건가요?" | "송금 금액을 입력하십시오." |
| "완료!" | "처리가 완료되었습니다." |

### 실무 적용 원칙
1. **동사 중심** — "조회하기" 대신 "확인해요", "신청하기" 대신 "바로 신청"
2. **능동태** — "결제가 완료됐어요" (수동: "결제가 완료 처리되었습니다")
3. **숫자는 아라비아 숫자** — "삼만 원" → "30,000원"
4. **오류 메시지에도 친절** — 원인 + 해결책 동시 제공
5. **버튼 레이블은 동작 동사** — "확인" → "결제하기", "취소" → "다시 고를게요"

## 2. 브랜딩 + 감성 라이팅 레퍼런스
> 출처: https://refeeel.com/brandingclass00

### 편안한 표현의 원칙
- **숨겨진 필요를 먼저 말한다** — 제품 기능 전에 사용자의 감정·상황을 먼저 인정
- **백공간을 두려워하지 않는다** — 짧은 문장, 여백, 호흡이 브랜드의 자신감
- **과장 없이 진심** — "세상에서 제일"이 아닌 "당신의 하루에 딱 맞는"
- **단어 하나로 분위기 전환** — 조사·어미 선택이 브랜드 페르소나를 결정

### 감성 카피 패턴
```
[상황 인정] → [공감] → [제안]
"바쁜 하루 끝에도" → "잠깐의 쉼은 필요하니까" → "딱 한 잔의 여유"
```

```
[행동 동사 시작] → [구체적 결과] → [감정 연결]
"열어보세요" → "오늘 뭐가 달라졌는지" → "직접 눈으로 확인해요"
```

## 3. 카테고리별 톤 가이드

### 핀테크·앱 (카카오페이·토스 계열)
- 존댓말 구어체: "~해요", "~에요", "~네요"
- 숫자+단위 조합: "3초 만에", "1만 원 환급"
- 동사 시작 CTA: "바로 받기", "지금 확인해요"
- 금지: "~하시기 바랍니다", "~을 입력하십시오", 복잡한 명사형

### 게임·엔터
- 에너지와 임팩트: "지금 바로!", "놓치지 마세요"
- 감탄 어미 허용: "드디어 왔다!", "이게 바로 그것!"
- 숫자+기간 강조: "24시간 한정", "선착순 100명"

### 제조·B2B 격식체
- 간결한 경어: "~합니다", "~드립니다"
- 수치 기반 신뢰: "17년 업력", "누적 납품 500건+"
- 전문 용어 절제: 업계 용어는 한 번만 설명 후 사용
- CTA: "문의하기", "자료 요청", "상담 신청"

### 브랜딩·감성
- 형용사보다 명사·동사: "우아한" 대신 "조용히 빛나는"
- 시제 섞기: 현재형으로 생생하게 ("지금 이 순간")
- 여백 지향: 한 줄 카피 → 행간 → 서브 카피

## 4. 심리 트리거 패턴

| 트리거 | 패턴 | 예시 |
|--------|------|------|
| **희소성** | 수량·기간 한정 | "오늘 밤 12시까지", "50명 한정" |
| **사회적 증거** | 숫자+행동 | "이미 3만 명이 시작했어요" |
| **손실 회피** | 놓칠 것 강조 | "지금 안 하면 다음 달부터 조건이 바뀌어요" |
| **호기심 갭** | 결론 보류 | "아무도 몰랐던 이유가 있어요" |
| **구체성** | 숫자·이름·날짜 | "3일 만에 5kg", "6월 한 달간" |
| **공감 선행** | 감정 먼저 인정 | "매번 작심삼일로 끝났다면" |

## 5. AI 라이팅 금지 패턴 (anti-ai-writing 연동)

절대 쓰지 않을 표현:
- "혁신적인", "게임체인저", "패러다임", "시너지"
- "X가 아닙니다. Y입니다." 형태의 부정 병렬
- "오늘날 급변하는 환경에서…" 식 도입부
- "함께 알아보겠습니다", "떠나보겠습니다"
- 뒷받침 없는 과장: "세상에서 가장", "압도적인"

## 6. 레이아웃·타이포 위계 참조 (HTML 산출물용)

ElevenLabs 브랜드 디자인 참조 기준:
- **Display**: Waldenburg 300 (얇고 우아한 대형 제목)
- **Body**: Inter (명료한 본문)
- **배경 톤**: Warm Stone `#f5f2ef` 계열 (차갑지 않게)
- **버튼**: pill 형태 (border-radius: 9999px)
- **그림자**: 다층 shadow, opacity 0.1 이하 (무겁지 않게)
- **색상 위계**: 모노크롬 베이스 + 시그니처 포인트 컬러 1개

### 타이포 위계 원칙
```
H1 (Display) — 감성 키워드, 1~5단어
H2 (Bold/Medium) — 섹션 제목, 7단어 이내
H3 (Regular) — 소제목, 문장형 가능
Body — 16px Inter, 1.6 행간
Caption — 12px, ink3 컬러
```
