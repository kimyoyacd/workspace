# 견적 템플릿 SoT (quote-sot)

> 🔒 **내부 전용 · 외부 산출물 노출 금지**. 참조 에이전트: `quote-accountant`
> 표준 양식: ㈜하이브랩 견적서 (KOSA 2026 기준). 단가는 `unit-rates.md`.
> ⚠️ 최종가·마진(할인율)·날인은 **사용자 직접**(절대 위임 금지).

## 하이브랩 견적서 구조 (실제 양식)
- 헤더: 견적명칭 / 수신(㈜OOO 담당자님 귀중) / 견적일자(유효 15일) / 견적금액(VAT별도)
- 발행사: ㈜하이브랩 · 144-81-01176 · 대표 서종혁 · 경기 성남 분당 대왕판교로 670 A동 8층
- 직접인건비 표 → 제경비 → 기술료 → 합계(개발비) → 할인율 → 최종견적(만원 절사)
- 하단: 참고사항(컨셉시안 등 단위/단가) + 산출근거(KOSA 고지)

## 직접인건비 표 (역할 × 등급 × MM × 참여율)
| 분류(역할) | 등급 | 인원 | M/M | 참여율 | 직접인건비 | 매출귀속 |
|---|---|---:|---:|---:|---:|---|
| Project Manager | 특급 | | | | | 타부서 |
| UX Design / Content Planning | 고급 | | | | | 타부서 |
| **UI Design** | **고급** | | | | | **★MAX실** |
| Front-end (HTML/CSS/JS) | 중급 | | | | | 타부서 |
| Back-end | 초급 | | | | | 타부서 |
| QA | (별도) | | | | | 케이스별 |

- 직접인건비 = 월노임단가 × 인원 × M/M × 참여율 (단가는 unit-rates.md)

## 계산 흐름
1. 직접인건비 합계
2. 제경비 = 직접인건비 × 제경비율(코웨이 110%)
3. 기술료 = (직접인건비 + 제경비) × 20%
4. 개발비 = 1+2+3
5. **최종견적 = 개발비 × (1 − 할인율)** → 만원단위 절사 (VAT별도)

## 🚨 MAX실 매출 인식
견적 전체 금액 ≠ MAX실 매출. **UI Design 라인 분(직접인건비→개발비→할인 적용분)만** 우리 매출로 집계.
quote-accountant는 ① 통합 견적가, ② 그 중 UI Design 라인 매출 — **두 숫자를 분리 제시**한다.

## 코웨이 BEREX 워크드 예시 (🔒 실데이터)
| 견적 | 일자 | 직접인건비 | 제경비(110%) | 기술료(20%) | 개발비 | 할인 66% → 최종 |
|---|---|---:|---:|---:|---:|---:|
| 초안 (260326_1) | 2026.3.26 | 93,373,046 | 102,710,350 | 39,216,679 | 235,300,075 | ×0.34 = 80,002,025 → **₩80,000,000** |
| 개정 (260515_2) | 2026.5.15 | 26,996,927 | 29,696,620 | 11,338,709 | 68,032,257 | ×0.34 = 23,130,967 → **₩23,100,000** |

- 발주: ㈜코웨이 주혜란님 / 수행: ㈜하이브랩 / 프로젝트: 코웨이 BEREX GUI/UX
- 역할 구성: PM(특급) · UX(고급) · UI(고급) · FE(중급) ··· 중 **UI Design 라인이 MAX실 매출**
- 개정에서 스코프 축소(80M→23.1M). 참고사항: 컨셉시안 2type(각 3종, 총 6종), 컨셉개발 인력 고급1/중급1/초급0.25

## 출력 시 고정 문구
