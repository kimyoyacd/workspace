# TO 템플릿 SoT

> quote-accountant 에이전트가 TO 제안 시 참고하는 표준 양식입니다.

## TO-01: UI/UX 웹사이트 구축

```
기간: 4~8주 기준

[VM] 시니어 1명: 80% (디자인 리드)
[VM] 주니어 1명: 60% (화면 제작)
[VX] 시니어 1명: 40% (UX 설계·QA)
[실장] 1명: 20% (PM·클라 미팅)

외주 필요 시:
[외주/개발] 프론트엔드: 별도 계약
```

## TO-02: 모션 그래픽 / 영상

```

[외주/3D]: 복잡도에 따라 별도 계약
[외주/사운드]: 별도 계약
```
