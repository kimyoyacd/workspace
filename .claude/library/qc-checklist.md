# 시안 QC 체크리스트 (design-critique 기준)

> 시안 검수 시 기본 점검 항목. 영역별로 해당되는 것만 본다. 실장님 기준으로 자유롭게 보강 가능.

## 공통 (모든 시안)
- [ ] 텍스트 잘림·넘침 없음 / 줄바꿈 깨짐 없음
- [ ] 저해상·깨진 이미지 없음 / 비율 안 깨짐
- [ ] 오타·맞춤법 / 숫자·단위 정확
- [ ] 민감정보(단가·개인정보·클라 내부정보) 노출 없음
- [ ] (HTML이면) 라이트모드 정상 — 다크모드에서 흰 배경 깨지지 않음

## UI/UX
- [ ] 시각 위계 명확(제목>본문) / 정렬·그리드 일관
- [ ] 여백 리듬 일관 / 요소 간 간격 규칙적
- [ ] 터치/클릭 영역 충분 / 상태(hover·active) 정의
- [ ] 색 대비 가독성(접근성)

## 콘텐츠
- [ ] 카피 톤이 브랜드와 일치 / 군더더기 없음
- [ ] 정보 정확·최신 / 과장·미검증 표현 없음

## GUI (게임)
- [ ] 아이콘·HUD 가독 / 세계관 톤 일관 / 폰트 가독

## UA / 퍼포먼스
- [ ] 첫 1초 후킹 존재 / CTA 명료 / 전환 동선 짧음

## 상세페이지
- [ ] 정보 위계(키비주얼→핵심→근거→팩트)
- [ ] 발색/제형 표현 충실 / 스크롤 리듬 있음
- [ ] "컷 하나 + 설명 하나" 식 심심한 구도 아님

## 브랜딩
- [ ] 관점(한 문장)이 시안 전체에 일관 적용
- [ ] 헤리티지(지킬 자산) 보존 / 타입·컬러 시스템 일관
