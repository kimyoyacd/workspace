# Designer Lens Insertion Map — 4개 게이트 삽입 지도

**역할:** 기존 Manus 1·2·Runable 3 프로세스에 디자이너 관점을 어디에 꽂아야 하는지 안내하는 삽입 지도  
**원칙:** 기존 흐름은 유지. 별도 부록이 아니라 **판단 게이트**로 삽입.

---

## 0. 핵심 정리

```text
Manus 1: 리서치·포지셔닝 단계
→ 디자이너 관점은 '관점 한 문장'과 '시장 빈자리 해석'에 삽입

Manus 2: 비주얼 아이덴티티·컨셉 단계
→ 디자이너 관점은 '자기 렌즈'와 '행동하는 비주얼 시스템'에 삽입

Runable 3: 브랜드 시스템·슬라이드 생성 단계
→ 디자이너 관점은 '시스템 토큰', '휴먼터치 레이어', '리스크 정렬 체크'에 삽입
```

---

## 게이트 4개

### Gate 1. Strategic POV Gate
**위치:** Manus 1 포지셔닝 전략 초안 직후

```md
이 브랜드는 어떤 시장 빈자리를 보는가?
이 브랜드는 무엇을 다르게 믿는가?
이 관점은 한 문장으로 말할 수 있는가?

통과 기준:
관점 한 문장:
이 관점이 필요한 이유:
경쟁사와 다른 믿음:
타깃이 반응할 감정:
```

---

### Gate 2. Lens Translation Gate
**위치:** Manus 2 Visual Identity DNA 추출 전

```md
핵심가치가 어떤 렌즈로 세상을 해석하는가?
그 렌즈가 색, 타입, 소재, 움직임으로 번역되는가?

통과 기준:
Core Value:
Designer Lens:
Visual Proof:
Motion / Behavior Proof:
Human Touch Proof:
```

---

### Gate 3. Living System Gate
**위치:** Manus 2 Concept Territories 직후

```md
이 컨셉은 정적 이미지에서 끝나지 않는가?
모션, 가변폰트, 반응형 규칙, 생성 규칙으로 확장되는가?

통과 기준:
Static Identity:

Generative Rule:
```

### Gate 4. Human Touch & Risk Gate
**위치:** Runable 3 브랜드 시스템 완성 직전

AI가 만든 것처럼 매끈하기만 하지 않은가?
어디에 인간의 흔적을 남길 것인가?
과감한 방향이 핵심가치와 정렬되어 있는가?

통과 기준:

## 삽입 위치 전체 맵

| 기존 단계 | 추가할 디자이너 관점 | 삽입 위치 |
|---|---|---|
| Manus 1 · 경쟁 브랜드 30개 | Blanding 탈출 / 동질화 진단 | 경쟁사 매트릭스 필드에 추가 |
| Manus 1 · 클러스터링 | 8개 비주얼 트렌드 태그(T1~T8) | 클러스터명 옆에 태그 |
| Manus 1 · 포지셔닝 맵 | 핵심가치 → 관점 한 문장 | 추천 포지셔닝 맵 직후 (Gate 1) |
| Manus 1 · 선호도 분석 | 헤리티지 보존 vs 파괴 스펙트럼 | 선호도 분석 후 |
| Manus 1 · 제품 비주얼 프롬프트 | 휴먼터치 레이어·물성·불완전성 | Physics & Blending 뒤 |
| Manus 2 · Visual Identity DNA | 핵심가치 → 자기 렌즈 → 비주얼 시스템 | DNA 추출 첫머리 (Gate 2) |
| Manus 2 · Competitive Visual Code | 이미 과밀한·덜 쓰인·금지 코드 | 기존 코드 요약과 통합 |
| Manus 2 · Concept Territories | 행동하는 시스템·모션·가변 규칙 | 각 컨셉 필드에 추가 |
| Manus 2 · Client×Mood Matrix | 리스크 정렬 체크 | 매트릭스 평가 항목 추가 (Gate 3) |
| Runable 3 · Brand Board | Behavior System / Human Touch System | Visual System 뒤 (Gate 4) |
| Runable 3 · Value Cards | 가치별 시각 증거·행동 증거 | Visual Expression 뒤 |
| Runable 3 · Copy & Symbol | 커스텀 타입·상징·반복 기호 | Symbol Association에 추가 |
| Runable 3 · Final Deck | 관점 한 문장 → 시스템 → 이미지 → 슬라이드 | Executive Summary 앞뒤 |
