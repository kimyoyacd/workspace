# 프롬프트 원형 — 브랜드 브리프 템플릿

> 출처: Hyojung guide 협업 자료의 브랜드 기획 브리프 구조. 브랜딩 프로젝트 시작 시 `concept-director` 입력으로 사용.
> 괄호 변수만 실제 값으로 채우면 한 장짜리 브리프가 완성된다.

## 한 줄 정의 (헤더)

```
(브랜드의 공간/장소)에서 (브랜드의 목적)하는 (타겟층) 전용 '(브랜드 이름)'을(를)
브랜딩하는 프로젝트.
```

## 5개 필드

```
1. 브랜딩 프로젝트의 목표
   - (이 브랜딩으로 무엇을 이루려는가 — 1~2줄)

2. 타겟층
   - (누구인가 + 그들의 행동/소비 특성 3~4줄)

3. 브랜드의 기능
   - (사용자가 경험하는 핵심 기능/장면)
   - 브랜드 모티브: (브랜드를 관통하는 시각/정서적 모티브)

4. 브랜드 이름의 의미
   - (이름의 어원·발음·이중 의미·담긴 철학)

5. 브랜드 스토리
   - (브랜드가 건네는 한 편의 내러티브 — 타겟의 감정에 닿는 톤으로)
```

## 예시 — Pace-off (러닝 브랜드, 참고용)
- **목표**: 전통 상업 공간을 넘어 진정성·몰입감을 갖춘 문화적 공간으로 전환.
- **타겟층**: 도시형 Gen Z — 해외 패션·문화에 개방적, 분위기/경험 중시, 한정판·협업 선호, 공간을 기록·공유하는 세대.
- **이름 의미**: Pace(속도) + Off(끄다/벗어나다) = "페이스를 낮추다". 타인의 기준에서 벗어나 나만의 리듬을 찾는다는 철학.
- **스토리**: "우리는 모두 다른 호흡의 길이를 가지고 있다. … Pace-off는 속도를 측정하지 않는 트랙이다."

## 사용 메모
- 이 브리프는 `concept-director`의 입력. 채워진 브리프 → "관점 한 문장"으로 응축한다.
- 5번 스토리는 카피라이팅(휴먼터치) 영역, 변주안은 AI 발산 가능.
