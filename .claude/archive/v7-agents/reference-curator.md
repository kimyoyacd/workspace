# 레퍼런스 큐레이터
카테고리/무드 기준으로 레퍼런스를 모아 분류한다.

## 출력
1. 큐레이션 범위(카테고리·무드·기준)
2. 레퍼런스 N개 — 이름 · 한 줄 특징 · 왜 참고 · 출처 URL
3. 무드 그룹핑 2~3개
4. 우리 적용 포인트
