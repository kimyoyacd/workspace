# 시장조사 스킬 (Market Research)

목표: 출처 있는 근거로 경쟁 구도를 지도화하고, 우리가 노릴 빈자리를 찾는다.

## 핵심 원칙
- **출처 필수** — 모든 브랜드 정보·주장에 출처(매체명 + URL). 출처 없으면 "추정"으로 명시.
- **객관 우선** — 유리한 결론 먼저 정하지 않는다. 구도를 그린 뒤 빈자리를 찾는다.
- **축은 소비자 언어로** — X·Y축은 소비자가 느끼는 대립(감성↔기능, 대중↔프리미엄 등)으로.
- 정밀도는 요청에 맞춤: 빠른 스캔이면 5~8개, "제대로/30개"면 카테고리 대표 브랜드 폭넓게.

## 출력 형식
1. **조사 범위** — 카테고리 / 기준("최근 1년 이내") / 브랜드 수 / 기준 시점.
2. **브랜드 분석 테이블** — 이름·소재지·오픈시기·브랜드스토리(요약)·타겟층·기능/목적·출처(URL).
3. **3개 그룹 분류** — 인사이트 공통점으로 묶기. 각 그룹 공통 관점 한 줄. ⭐ 이 3그룹이 브레인스토밍 3방향 후보.
4. **XY 포지셔닝 맵** — 소비자 대립 축 2개 정의 → 브랜드/3그룹을 사분면에 배치. 군집과 공백 짚기.
5. **화이트스페이스** — 비어 있는 사분면 = 우리 자리. "기회인가 함정인가"(수요 없는 빈자리는 함정) 판단.
6. **시사점** — 컨셉 디렉터·발산·견적에 주는 한 줄 함의.

> 상세 프롬프트: `.claude/library/prompts/wide-research.md`가 있으면 그 사양을 따른다.

## 가드레일
- 데이터 부족 시 추정으로 채우지 말고, 무엇을 더 조사해야 하는지 먼저 밝힌다.
- 숫자·점유율·매출 주장은 출처 없이 쓰지 않는다.
