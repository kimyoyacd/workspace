# 시장조사 스킬 (Market Research)

목표: 출처 있는 근거로 경쟁 구도를 지도화하고, 우리가 노릴 빈자리를 찾는다.

## 실행 절차 — `.claude/library/prompts/wide-research.md` 사양 기준

### Step 1 · Wide Research (경쟁사 수집)
- **기본 30개** 브랜드 선정 (최근 1년 이내 활동 중인 국내·해외).
  빠른 스캔 요청 시 5~8개로 축소 가능하되 "축소함" 반드시 명시.
- 각 브랜드를 8속성으로 정리: ①이름 ②로고URL ③소재지 ④오픈시기 ⑤브랜드스토리 ⑥타겟층 ⑦기능 ⑧목적
- 모든 항목에 **출처 URL 필수 + 클릭 가능한 링크(`<a href>`)로 제공**. 없으면 "추정" 표기.
- **이종업계 레퍼런스 2~3개 별도 수집** — 동업계 밖에서 "이 제품이 왜 당신에게 필요한가"를
  설득하는 방식이 탁월한 브랜드. 레이아웃·스토리텔링·비주얼 전략 벤치마크.
  **우선 소스 (아래 순서로 탐색):**
  1. **명품·프리미엄** — Chanel Beauty(텍스처 클로즈업·여백), Bottega Veneta(세계관 우선 최소정보), Aesop(성분→감성 텍스트 전환)
  2. **인테리어·편집숍** — Merci Paris, Muji, 어반리턴 등 "제품을 경험으로 파는" 상세페이지 구조
  3. **글로벌 에이전시 사례** — COLLINS·JKR·Wolff Olins·W+K가 작업한 브랜드 런칭·제품 랜딩페이지
  4. **타 카테고리 테크·F&B·패션** — Nike·Spotify·Maison Kitsuné 등 데이터·감성·세계관 표현법

### Step 2 · 타겟 관심도 × 트렌드 축
- 해당 프로젝트의 **타겟층이 지금 관심 갖는 키워드·트렌드** 3~5개 추출 (SNS·검색량·미디어 기반).
- 트렌드 축과 경쟁사 포지션을 교차 → "타겟이 향하는 방향에 누가 선점했나" 파악.

### Step 3 · 3그룹 분류 + XY 포지셔닝 맵
- 인사이트 공통점으로 3그룹 묶기. ⭐ 이 3그룹 = 브레인스토밍 3방향 후보.
- XY축: 소비자가 느끼는 대립(감성↔기능, 대중↔프리미엄 등) 2개 정의 → 브랜드 배치 → 군집·공백 짚기.
- 화이트스페이스: "기회인가 함정인가"(수요 없는 빈자리는 함정) 판단.

## 출력 순서
1. 조사 범위 (카테고리·브랜드 수·기준 시점)
2. 브랜드 분석 테이블 (8속성 + 출처)
3. 타겟 관심도·트렌드 요약
4. 3그룹 분류 + 각 그룹 공통 관점 한 줄
5. XY 포지셔닝 맵 (텍스트 사분면)
6. 화이트스페이스 판단
7. 시사점 — concept-director·brainstorm으로 넘기는 한 줄 함의

## 가드레일
- 데이터 부족 시 추정으로 채우지 말고 추가 조사 항목을 먼저 밝힌다.
- 숫자·점유율·매출은 출처 없이 쓰지 않는다.
- 객관 우선 — 유리한 결론 먼저 정하지 않는다.

## ★ 검증 게이트 — 통과해야 다음 단계 진행 가능

시장조사 완료 후 아래 7개 항목을 체크. **전부 ✅여야** concept-director·brainstorm으로 넘어간다.
하나라도 ❌면 해당 항목 보완 후 재검증.

```
[ ] 브랜드 수: 30개 이상 수집됐는가 (축소 시 "축소함" 명시했는가)
[ ] 출처: 모든 브랜드에 URL 또는 "추정" 표기가 됐는가
[ ] 8속성: 빠진 항목 없이 채워졌는가
[ ] XY축: 소비자 언어 대립으로 정의됐는가 (업계 용어 아닌가)
[ ] 화이트스페이스: 수요 근거 없이 "빈자리 = 기회"로 단정하지 않았는가
[ ] 3그룹: 각 그룹이 서로 다른 인사이트를 갖는가 (중복 아닌가)
[ ] 시사점: concept-director에 넘길 한 줄 관점이 나왔는가
```

게이트 통과 선언 형식:
> **✅ 검증 게이트 통과 — concept-director / brainstorm으로 진행 가능**
> 또는
> **❌ [항목명] 미충족 — [보완할 내용] 후 재검증 필요**

**라운드 제한: 최대 4회.** 4라운드까지 재검증 후에도 ❌ 항목이 남아있으면
미충족 항목을 리스트로 남기고 다음 단계로 진행한다. 완벽보다 진행 우선.
