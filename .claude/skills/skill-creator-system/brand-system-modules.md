# 브랜드 시스템 모듈 양식 (3차)

> 출처: kyeongahko×Runable 3차 협업. 필요한 모듈만 골라 조립한다.

## 1. 브랜드 보드 (노션 스타일, 한 장)
- 구성: 브랜드명/로고 · 한 줄 컨셉 · 핵심 무드 이미지 3~6 · 핵심 가치 키워드
- 톤: 미니멀·여백 중심 (예: "Havnn" 보드처럼 절제된 정리)

## 2. 브랜드 가치 카드 / 지표
| 가치 | 정의(한 줄) | 행동 지표 | 비주얼 단서 |
|---|---|---|---|
| _가치1_ | | | |
- 가치 3~5개. 각 가치가 비주얼/카피에 어떻게 드러나는지 단서 포함.

## 3. 가치 키워드맵 (마인드맵)
- 중심: 브랜드 핵심 가치 1개
- 1차 가지: 핵심 가치 3~5 → 2차 가지: 연관 키워드/감정/소재
- 용도: 카피·무드·네이밍 발산의 씨앗 (brainstormer 연계)

## 4. 브랜드 카피 / 헤드라인 세트
- 형식: 대문자 임팩트 헤드라인 + 보조 설명 한 줄
- 예시 톤: `QUIET ISLAND` / `SENSORY DESIGN` / `POWER OF RETAIL`
- 세트로 3~6개 → 캠페인/섹션 헤더로 활용

## 5. 기호·연상 패턴
- 브랜드 모티프(소재·형태)를 → 반복 패턴/심볼/텍스처로 전개
- 산출: 패턴 그리드(원/2차 변주), 적용 예(패키지·배경)

## 6. 경쟁사 CI/BI 분석
- 경쟁/레퍼런스 브랜드 로고·아이덴티티 그리드
- 각 항목: 브랜드 · 로고 인상 · 컬러/타이포 특징 · 우리와의 차별점
- 출처 URL·상표 권리 확인. 분석 목적, 모사 금지.
