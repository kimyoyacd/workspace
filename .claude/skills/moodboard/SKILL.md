# 무드보드 스킬 (Moodboard Builder)

목표: 컨셉/관점을 레퍼런스+생성 이미지로 조합한 무드보드로 시각화한다.
기반 방법론: `.claude/library/prompts/MOODBOARD_MASTER_FRAMEWORK.md` (6버킷·퍼스널리티 슬라이더·4단계 파이프라인)

## 4단계 파이프라인

### 1단계: 프리-디렉션 & 퍼스널리티 슬라이더
- 이미지 조사 전, 클라이언트의 주관적 언어를 수치화된 매트릭스로 변환.
  예: `대중성 ───┼─── 희소성`, `진지함 ───┼─── 위트`, `전통적 ───┼─── 반항적`
- 업계 평균(Safe Route) 외에 와일드카드 대안을 최소 1개 준비.

### 2단계: 분석적 레퍼런스 분해
- 알고리즘 트랩 탈출 — 무작위 수집 금지.
- 수집 이미지를 [배색 조합 / 화면 구도 / 질감·소재 / 문화적 맥락] 4축으로 분해 후 재조합.

### 3단계: 내러티브 주입 + 비주얼 에디팅
- 그레이스케일 오버레이로 컬러 충돌 정제 (조형·레이아웃에 시선 집중).
- 브랜드 고유 서사를 비주얼 모티프로 치환.

### 4단계: 프레젠테이션 & Annotation
- 에디토리얼 가중치: 핵심 비주얼 = 가장 큰 면적.
- 정밀 피드백 라벨링: A-1, B-2 형식으로 모든 이미지 유닛 라벨링.

## 6 버킷 프레임워크 (수집·전개 시)
| 버킷 | 목적 |
|---|---|
| Setting (장소·배경) | 공간 구조·톤·질감 |
| Action (인물 행동) | 역동성, 이야기 있는 움직임 |
| Angle (카메라 앵글) | 구도·렌즈 시점 독창성 |
| Lighting (조명·광원) | 비주얼 깊이·감정 |
| Edit (후반 편집) | 컬러 커렉션·레이아웃·그레인 |
| BTS (비하인드) | 숏폼·릴스·TikTok용 현장 생동감 |

## 이원화 (프로젝트 성격에 따라)
- **바이브 보드** — 브랜드 감성·페르소나·추상적 분위기
- **택티컬 보드** — 타이포그래피·UI·패키지 지기구조·실제 인터랙션

## 레퍼런스 소스 (2단계 수집 시 우선 탐색)
무작위 검색 금지. 아래 소스에서 감도 높은 비주얼을 우선 수집한다 (WebSearch·WebFetch 활용).

| 소스 | 용도 |
|---|---|
| cosmos.so | 트렌드 앞서가는 큐레이션 — 최우선 탐색 |
| Pinterest | 키워드별 보드 · 연관 이미지 확장 |
| Behance | 글로벌 에이전시·스튜디오 프로젝트 (COLLINS, Pentagram 등) |
| 글로벌 에이전시 사이트 | 에이전시 포트폴리오 직접 탐색 |
| 감도 높은 편집샵 | SSENSE, 무신사부티크, 아더에러 등 커머스 비주얼 |
| Midjourney 커뮤니티 | AI 생성 비주얼 스타일 레퍼런스 |
| 트렌드 선도 브랜드 | 해당 카테고리 앞서가는 브랜드 공식 채널 |

- 수집한 이미지는 반드시 출처 URL과 함께 기록한다.
- 직접 다운로드가 안 되는 이미지는 URL + 스타일 설명으로 기록하고, 유사 무드는 Higgsfield로 재생성한다.

## 도구
- 레퍼런스: reference-curator 스킬 결과 + 위 레퍼런스 소스 직접 탐색.
- 생성 이미지: ToolSearch로 Higgsfield `generate_image` 불러 사용.

## 출력
- 무드 키워드 + 퍼스널리티 슬라이더 설정값
- 컬러/타입 방향
- 레퍼런스/생성 이미지 배치안 (6버킷 기준)
- 와일드카드 대안 1개
