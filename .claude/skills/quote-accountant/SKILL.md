# 견적 회계사 스킬 (Quote Accountant)

목표: 단가표 기반으로 원가를 계산하고 마진 시나리오를 제시한다.
**최종 견적가와 마진율은 절대 정하지 않는다 — 시나리오만 제시, 결정은 사용자.**

## 입력 (반드시 단가표 사용)
- 노임단가·계산식·매출 규칙: `.claude/library/unit-rates.md` 먼저 읽고 그 값 사용.
- 정보 부족하면 추정하지 말고 무엇이 필요한지 먼저 묻는다.

## ⭐ 우리실 매출 규칙 (반드시 적용)
> 견적 전체 중 **'UI Design' 직군 직접인건비만 우리 실(MX) 매출**.
> ① 프로젝트 총액과 ② 우리실 매출(UI Design분)을 항상 분리 표기.

## 계산식
```
직군별 직접인건비 = 월 노임단가 × M/M × 인원수 × 참여율(%)
프로젝트 총액      = Σ(전 직군 직접인건비) + 경비·이윤(입력 시)
우리실 매출        = UI Design 직군 직접인건비 합계
```

## 출력 형식
1. **직군별 원가표** — 직군 / 등급 / 인원 / M/M / 참여율 / 직접인건비
2. **합계 분리** — ⓐ 프로젝트 총액 / ⓑ 우리실 매출(UI Design분) / ⓒ 우리실 비중(%)
3. **마진 시나리오** — 10/20/30/40% 4종. VAT 별도 명시.
4. **결정 필요** — "최종 견적가·마진율은 실장님이 선택" 명시 + 리스크 한 줄.

## 가드레일
- 최종 견적가 / 마진율 / 단가 협상 카드 / 계약 날인은 사용자 직접.
- VAT 별도/포함 항상 명시.
