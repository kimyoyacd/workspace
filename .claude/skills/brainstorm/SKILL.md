# 브레인스토밍 스킬 (Brainstorm)

목표: 판단하지 말고 최대한 많고 다양한 가능성을 긍정적으로 쏟아낸다.

## 핵심 원칙
- 비판 금지. "안 되는 이유"는 절대 말하지 않는다.
- 양(quantity)이 질을 만든다. 최소 8~12개 이상.
- 발상 각도를 의도적으로 다양화: 안전한 것 / 과감한 것 / 엉뚱한 것을 섞는다.
- 실무 맥락(평면·홈페이지·배너·게임 UI)을 활용하되 거기에 갇히지 않는다.

## 아날로지 발산 (시장조사의 이종업계 원리 번역 확장)
market-research 스킬에서 "원리 번역(아날로지)" 후보가 넘어온 경우, 아래 방식으로 확장한다.

**진행 방식:**
1. 넘어온 원리(예: 음악 크레셴도, 지휘 동작, 항공 UI 공간감)를 그대로 수용
2. "이 원리를 우리 프로젝트에 적용하면?" — 번역·혼합·재해석
3. 원리 하나당 표현 방식 3개 이상 발산
   - 비주얼로 번역하면?
   - 모션·인터랙션으로 번역하면?
   - 카피·네이밍으로 번역하면?

**예시:**
- 음악 크레셴도(점점 크게) → 스크롤할수록 타이포 커짐 / 컬러 채도 높아짐 / 사운드 레이어
- 지휘자 손동작 → 마우스 이동에 반응하는 타이포 / 제스처 트리거 애니메이션
- 항공 퍼스트클래스 공간감 → 여백을 "서비스"로 표현 / 정보 밀도 낮추기를 프리미엄으로 전환

## 출력 형식
1. **빠른 발산** — 아이디어를 짧은 한 줄로 번호 매겨 나열 (8개 이상)
2. **카테고리 묶음** — 비슷한 결끼리 2~4개 그룹으로 정리
3. **확장 추천 3선** — 더 파볼 만한 3개 선택:
   - **핵심 키워드 3~5개** — 그 방향을 압축하는 단어들
   - **한 줄 포지셔닝** — 경쟁/일반 대비 이 방향이 서는 자리
4. **약식 포지셔닝 맵** — 대립 X축·Y축 2개 세우고 3방향을 사분면에 배치 → 비어 있는 자리 짚기.
   (정밀 30개 경쟁사 맵은 market-research 스킬 몫. 그 결과 있으면 우선 인용)
5. **다음 단계 질문** — 방향을 좁히기 위한 질문 1~2개

## 톤
밝고 에너지 있게. "이런 것도 가능해요", "여기서 한 발 더 나가면" 식으로 확장적 어조.
