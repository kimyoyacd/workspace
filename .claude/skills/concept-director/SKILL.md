# 컨셉 디렉터 스킬 — 사전 관점 게이트 (Concept Director)

목표: 어떤 디자인 작업이든 시작하기 전에, 핵심 제품/타이틀을 기준으로 브랜드 관점을 한 문장으로 고정한다.
고급 디자인 에이전시(COLLINS·Wolff Olins·JKR·W+K)의 "관점 → 시스템" 방법론 적용.

## 위치 — 가장 앞단 (게이트)
```
[핵심 제품/타이틀]  →  ★관점 게이트  →  관점 한 문장
                                          │
        ┌──────────────┬──────────────┬──┴───────────┐
        ▼              ▼              ▼              ▼
      사이트        브랜딩          UA          상세페이지
   (모두 같은 관점을 물려받아 각자의 매체 언어로 전개)
```
발산·검증보다 **먼저** 호출. 관점이 없으면 발산도 검증도 기준이 없다.

## 핵심 4원칙
1. **관점 먼저** — 비주얼·실행보다 핵심가치 한 문장이 먼저. 응축 안 되면 아직 컨셉 아님.
2. **자기 렌즈** — 표면 아래 진짜 문제를 보는 우리만의 시선으로 비틀기.
3. **헤리티지 결정** — 무엇을 지키고 무엇을 부술지 명시 (보존 ↔ 파괴 스펙트럼).
4. **행동하는 시스템** — 모션·반응·생성 규칙·AI×휴먼터치 레이어까지 정의.

## 입력
- **반드시**: 핵심 제품 또는 타이틀.
- **있으면 좋음**: 타깃, 풀려는 문제, 헤리티지, 예정된 디자인 타입.

## 출력 형식
1. **관점 한 문장** — 'Sound is Power'처럼 전략적 코어를 한 문장으로. (가장 중요)
2. **근거** — 핵심가치 / 풀려는 진짜 문제 / 대상 한 줄씩.
3. **자기 렌즈** — 어떤 시선으로 비트는가 (1~2줄).
4. **헤리티지 결정** — `지킨다`(무엇·왜) / `바꾼다`(무엇·왜).
5. **디자인 타입별 전개** — 표로, 하나의 관점이 각 매체에서 하는 일(한 줄).
6. **시스템 행동 정의** — 핵심 레이어별 정적 산출 / 모션·반응 규칙 / AI×휴먼터치.
7. **정렬 체크 (Jaguar 가드)** — 핵심가치·대상과 정렬되어 있는가 자가 판정. 깨지면 경고+보정안.
8. **다음 단계** — 이 관점으로 무엇을 발산·검증·제작할지 1줄 연결.

## 가드레일
- 관점이 한 문장으로 응축 안 되면 "아직 컨셉 아님"이라 솔직히 말한다.
- "과감함" 추천 시 핵심가치 정렬을 반드시 함께 확인한다.
