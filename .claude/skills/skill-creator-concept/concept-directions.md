# 컨셉 디렉션 팔레트 (2차)

> 출처: kyeongahko×Manus 2차 협업에서 나온 컨셉 보드 이름들.
> 새 프로젝트마다 이 팔레트를 **출발점**으로 변주·확장한다(그대로 복붙 X). 각 디렉션은 "어떤 문제를 푸는가"로 골라야 한다.

## 무드/컨셉 디렉션 예시
| 디렉션 | 성격 한 줄 | 어울리는 상황 |
|---|---|---|
| **Curated Chaos** | 정제된 무질서 — 큐레이션된 콜라주/레이어 | 다양성·에디토리얼·아카이브 톤 |
| **Immersive Worlds** | 몰입형 세계관 — 풀블리드 비주얼/공간감 | 경험·체험 중심, 게임/공간 브랜드 |
| **Street Pulse** | 거리의 에너지 — 생생한 스트리트/그래피티 | 영 타깃, UA/캠페인, 활동성 |
| **Cycle of Scenes** | 장면의 순환 — 시퀀스·스토리텔링 구성 | 내러티브, 영상·모션 연계 |
| **Raw Meets Refined** | 거침 × 정제 — 러프 텍스처 + 정돈된 타이포 | 프리미엄인데 진정성 강조 |
| **Living Archive** | 살아있는 아카이브 — 자료적·기록적 정리감 | 헤리티지·브랜드 자산 축적 |
| **Brutalist Framework** | 브루탈리즘 — 강한 그리드·고대비·기능미 | B2B·테크·강한 인상 |

## 무드보드 구성 요소 (각 컨셉당)
- 컨셉 이름 + 한 줄 정의 + **푸는 문제**(1차 포지셔닝과 연결)
- 무드 키워드 5~7
- 컬러 팔레트 + 타이포 방향
- 레퍼런스 이미지 그리드 (출처 URL)
- 적용 예시(패키지/웹/배너 중 1~2)

## Client × Mood 매트릭스
- 가로축: 클라 성향(보수 ↔ 실험) / 세로축: 무드(차분 ↔ 강렬)
- 각 컨셉을 사분면에 배치 → 클라 적합도·리스크 한눈 비교
