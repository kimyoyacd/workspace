## S4-A. Before / After / Bridge — 3개 브랜드 공통

### BEFORE (현재 인식)

```
올리브영 색조 PB 3개 브랜드의 상세 페이지는
"노후화된 형식"으로 만들어진 "제품 설명서"다.

- 컷-설명-컷-설명이 반복되는 나열 구조
- 브랜드 무드와 상세 페이지 비주얼의 온도 차이
- 3개 브랜드가 구분되지 않는 동일 포맷
- 발색과 텍스처가 보이지 않는 스와치 나열
- 소비자가 "사고 싶다"는 감정이 아닌
  "이런 제품이 있구나"는 인지로 끝남
```

### AFTER (목표 인식)

```
올리브영 색조 PB 3개 브랜드의 상세 페이지는
"감각이 설계된 경험"이다.

- 도입부 3초에 브랜드 무드가 전달된다
- 컬러와 텍스처가 직접 느껴진다 (감각 시각화)
- 웨이크메이크/컬러그램/필리밀리가 각자의 언어를 가진다
- 스크롤이 하나의 메이크업 여정이 된다
- 소비자가 "이게 나야"를 경험하고 구매로 이어진다
```

### BRIDGE 워드 (후보 + 확정 요청)

```
Bridge 워드: 감각(感覺)

선택 근거:
1. RFP 이면에 있는 진짜 니즈: "색조스럽게 변경해 보는 감각이나 센스"
   → 이선민이 직접 사용한 단어: "감각", "센스"
   
2. 현행 문제의 본질: "감각이 설계되지 않았다"
   → Before 상태를 한 단어로 설명

3. 해결의 방향: "감각을 설계한다"
   → After 상태를 한 단어로 제시

4. 경쟁사 차별화:
   → 다른 업체: "최고의 결과물을 납품하겠습니다"
   → 우리: "감각이 설계된 상세 페이지를 만들겠습니다"
   → "감각 설계"라는 개념 자체가 우리만의 프레임

5. 3개 브랜드 공통 적용 가능:
   → 웨이크메이크: 컬러 감각
   → 컬러그램: 플레이 감각
   → 필리밀리: 터치 감각

대안 후보 (디렉터 의견 반영용):
- 연출(演出): "감각을 연출한다"
- 언어(言語): "컬러가 언어가 된다"
- 리듬(Rhythm): "색조에 리듬을 넣다"
```

## S4-B. Goal 3-Layer — Bridge 워드로 관통

### User Goal (소비자 목표)

```
감각(感覺)이 전달되는 상세 페이지를 통해,
소비자가 제품을 손에 들기 전부터
"이게 나야"를 경험한다.

구체적 행동 변화:
- 스크롤을 멈추고 자세히 본다 (이탈률 감소)
- 컬러를 선택하는 데 확신이 생긴다 (결정 속도 향상)
- 구매 후 "기대했던 그 감각"을 실제로 경험한다 (재구매율 향상)
- SNS에 언박싱/발색 컷을 올리고 싶어진다 (자발적 확산)
```

### Business Goal (비즈니스 목표)

```
감각(感覺) 설계가 내재화된 상세 페이지 제작 체계를 통해,
올리브영 색조 PB의 온라인 전환율을 높이고
글로벌 동시 운영 비용을 줄인다.

구체적 지표:
- 상세 페이지 평균 체류 시간 증가 (현재 대비 +30% 목표)
- 상세 페이지 → 장바구니 전환율 향상
- 글로벌 버전(일본·영어) 동시 납품으로 운영 효율화
- 월 정기 발주 체계 → 브랜드 일관성 유지 + 비용 예측 가능
```

### Brand Goal (브랜드 목표)

```
감각(感覺)이 설계된 상세 페이지를 통해,
웨이크메이크·컬러그램·필리밀리가
"K-뷰티 색조 PB의 비주얼 기준"이 된다.

구체적 포지셔닝:
- "올리브영 PB = 상세 페이지도 프리미엄"이라는 인식 구축
- 롬앤·3CE 등 인디 브랜드 대비 PB의 한계 극복
- 일본·미국 현지 소비자에게 K-뷰티 감각 기준 제시
- 2026 K-뷰티 2.0 트렌드: "강한 브랜딩" 방향 선점
```

## Bridge 워드 "감각(感覺)"이 3-Layer를 관통하는 구조

```
User Goal:    소비자가 감각을 경험한다 (감각의 수신)
              ↕
Business Goal: 감각 설계 체계로 전환율을 높인다 (감각의 경제화)
              ↕
Brand Goal:   브랜드가 감각의 기준이 된다 (감각의 권위화)

→ 3개 레이어 모두 "감각"으로 연결됨
→ 제안서 전체를 "감각 설계 파트너십" 서사로 통일 가능
```

## S4 → S6 컨셉 3안 네이밍 정제

S3에서 도출한 컨셉 3안 후보를 Bridge 워드로 정제:

| 안 | 컨셉명 (초안) | Bridge 연결 | 타깃 브랜드 |
|---|------------|-----------|-----------|
| **1안** | **Color Speaks** | "컬러가 감각을 말한다" | 웨이크메이크 (과제 시안) |
| **2안** | **Mood First** | "기분이 곧 감각이다" | 컬러그램 |
| **3안** | **Sensation System** | "감각을 체계로 설계한다" | 3개 브랜드 공통 |

### 3안이 "같은 방향의 3가지 입구"인 이유

```
공통 방향: "감각이 설계된 상세 페이지"

1안 Color Speaks: 컬러 자체에서 감각이 전달되는 방식
                  → 비주얼 중심, 화보급 도입부, 발색 시퀀스

2안 Mood First:   소비자의 기분이 감각의 출발점인 방식
                  → 서사 중심, 무드 선택 → 컬러 큐레이션

3안 Sensation System: 감각을 만드는 제작 시스템이 차별점인 방식
                       → 체계 중심, AI+모듈화로 월 납품 시스템

→ 3안이 같은 "감각 설계"를 향하되, 각각 다른 강점을 강조
```
