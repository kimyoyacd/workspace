# 50. 용어 사전 — "이 말이 대체 뭐야?"

> 클로드를 쓰다 보면 스킬·마크다운·깃허브·MCP 같은 낯선 말이 튀어나와요. **하나도 몰라도 일은 시작할 수 있어요.** 이 사전은 "겁먹지 말라"고 있는 거예요.
> 먼저 아래 **딱 3줄**만 보고, 필요할 때 찾아 읽으면 돼요.

> 1. **지금 꼭 알아야 할 건 거의 없어요.** 챗에서 우리말로 부탁하는 것만으로 대부분 돼요.
> 2. **아래 말들은 "일하다 지나가며 보게 될 간판"** 이에요. 뜻만 알면 안 무서워요.
> 3. **★ = 알면 편함 / (몰라도 됨) = 지금은 신경 끄기.**

---

## A. 파일·문서

### 마크다운 (.md) ★
- **한 줄**: 글을 아주 간단한 기호로 예쁘게 쓰는 **가벼운 문서**예요. (워드의 아주 가벼운 버전)
- **왜 써요**: 어디서나 열리고, 누가 뭘 고쳤는지 보기 좋아서 같이 쓰는 문서에 딱이에요. 이 온보딩 문서들(`00_기초개념.md` 등)이 다 마크다운이에요.
- **알아둘 기호**: `#` 제목, `**굵게**`, `-` 목록, `|`로 표. 이게 다예요.
- **비유**: 제목·굵게·목록이 되는 메모장.

### HTML (몰라도 됨)
- **한 줄**: 웹페이지를 만드는 방식이에요. `onboarding.html`이 그 예 — 브라우저에서 예쁘게 열려요.
- 결과를 "예쁘게 전달"할 때 클로드가 대신 만들어줘요. 직접 손으로 쓸 일은 없어요.

---

## B. 깃허브(GitHub) — 팀 공동 창고

> **먼저 안심하세요**: **대부분은 깃허브를 직접 안 만져도 돼요.** "우리 결과물이 여기 보관되고 링크로 공유된다" 정도만 알면 돼요.

### 깃허브 (GitHub) ★
- **한 줄**: 문서·파일을 팀이 같이 보관하고, **누가 언제 뭘 바꿨는지 다 기록**하는 인터넷 창고예요.
- **비유**: 구글 드라이브 + "되돌리기"가 아주 센 것.

### 저장소 (Repository / Repo)
- **한 줄**: 프로젝트 폴더 전체예요. 우리 `workspace`가 저장소 하나예요.

### 커밋 (Commit)
- **한 줄**: "저장 + 뭘 바꿨는지 한 줄 메모"를 한 번 찍는 거예요.
- **비유**: 게임 세이브. 언제든 그 지점으로 돌아갈 수 있어요.

### 브랜치 (Branch)
- **한 줄**: 원본을 안 건드리고 **따로 떼서 작업하는 복사본**이에요. 실험하다 잘못돼도 원본은 안전해요.

### PR (Pull Request, 풀 리퀘스트)
- **한 줄**: "내가 바꾼 걸 원본에 **합쳐도 돼요?**" 하고 확인을 요청하는 거예요.
- **비유**: "이 수정본 봐주세요" 하고 올리는 결재.

## C. 클로드에서 자주 듣는 말

### 프로젝트 / 세션 / 모델 / 컨텍스트 / 토큰
- 이건 [`00_기초개념.md`](./00_기초개념.md)에서 자세히 다뤄요. 한 줄 요약:
  - **세션** = 대화 하나 = 기억 하나 (옆 대화는 못 봐요)
  - **프로젝트** = 관련 대화·파일을 모으는 폴더
  - **모델** = 두뇌 등급 (보통 Sonnet / 어려우면 Opus)
  - **컨텍스트** = 지금 들고 있는 대화 전부 / **토큰** = 글자 세는 단위

### 스킬 (Skill) ★
- **한 줄**: 어떤 일을 "이렇게 해"라고 **미리 정리해 둔 설명서 묶음**이에요. 클로드가 필요할 때 알아서 꺼내 써요.
- **왜 좋아요**: 매번 똑같이 길게 설명 안 해도 돼요. 예를 들어 "슬라이드 만들기 스킬"을 만들어두면, 그냥 "슬라이드 만들어줘" 하면 그 규칙대로 나와요.
- **비유**: 자주 하는 요리의 **레시피 카드**. 부를 때마다 재료·순서를 다시 안 불러줘도 돼요.

### 에이전트 (Agent) ★
- **한 줄**: 특정 역할만 잘하는 **클로드 분신**이에요. 부르면 그 시선으로 일해요.
- **예**: "견적 담당", "트렌드 조사 담당", "시안 봐주는 담당" 같은 분신.
- **스킬과 차이**: 스킬 = *하는 방법* / 에이전트 = *그 방법 든 담당자*. (처음엔 이 차이까지 몰라도 돼요)

### 아티팩트 (Artifact) ★
- **한 줄**: 클로드가 만든 결과(웹페이지·문서)를 **링크로 바로 볼 수 있게 내보낸** 거예요.
- 이 온보딩의 HTML 페이지가 아티팩트로 나와서 링크로 공유돼요.

## D. 연결·도구

### MCP / 커넥터 (Connector) ★
- **한 줄**: 클로드를 **다른 도구에 연결하는 다리**예요. 연결하면 그 안 자료를 읽거나 쓸 수 있어요.
- **예**: 노션(Notion), 지메일(Gmail), 구글 드라이브, 피그마(Figma), 캘린더 등.
- **비유**: 클로드에게 "노션 열쇠", "드라이브 열쇠"를 쥐여주는 거예요. (연결은 보통 관리자가 해줘요)
- **주의**: 연결된 자료는 비밀·개인정보일 수 있어요 → 다룰 때 보안 규칙(`30`, `99`)을 지켜요.

### Claude Code / 터미널 (몰라도 됨)
- **한 줄**: 까만 화면에 명령을 쳐서 파일·자동화를 직접 다루는 **개발자용 화면**이에요.
- **대부분은 지금 몰라도 돼요.** 파일을 많이 만들거나 자동화가 필요해지면 그때 배우면 돼요.
