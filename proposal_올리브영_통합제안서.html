트렌드 팩트
2026년 K-뷰티는 반사광이 강한 글라스 스킨을 넘어 '블룸 스킨'으로 전환 중. 색조 트렌드도 소프트-포커스 블러쉬, 블러드 립, 새틴 피니시로 이동하며 피부와 자연스럽게 융합되는 룩이 주도. 피치·로즈·베리 밀크·뮤트 에이프리콧 톤이 핵심 컬러.

Mitch Paone · DIA Studio

COLLINS 방법론
COLLINS는 Dropbox의 브랜드 가치를 $6B에서 $19B으로 끌어올린 리브랜딩 에이전시. 핵심 방법론은 "가치 제안 재작성" — 제품이 하는 일이 아니라, 제품이 가져다주는 변화를 전면에 배치.
Spotify의 Flex Deck 시스템 — 프로그래밍 가능한 슬라이드 시스템으로 수천 개의 광고주별 맞춤 프레젠테이션 자동 생성.
→ 올리브영 적용
기존: "이 제품은 7가지 컬러 + 고발색 + 지속력 12시간"
재작성: "오늘의 기분을 7가지 스펙트럼으로 표현하세요. 아침부터 밤까지, 당신의 컬러가 변하지 않습니다."
Flex Deck 방식으로 SKU별 데이터 기반 상세페이지 자동 조합 — 템플릿 × 데이터 = N개 페이지
