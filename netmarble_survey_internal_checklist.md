# 만족도 조사 대응 — 내부 준비 체크리스트 (내부용)

> 고객사 공유용 문서와 별도로 관리하는 내부 준비 자료입니다. 고객사에 그대로 전달하지 않습니다.

## 1. PPT / 자료 준비

- [ ] 만족도 조사에서 개선사항 / 요구사항 / 긍정 점수 / 주요 점수만 추려 정리 (원본은 과도하게 재가공하지 않고 캡처 삽입)
- [ ] 필요한 이미지 Figma 업로드 요청 → Figma 자료 받아 PPT에 연결·정돈
- [ ] 서브컬처향 타이포그래피 레퍼런스 타입별(1번/2번/A·B·C) 수집 → 캡처 이미지로 PPT 삽입
- [ ] AI 기반 서브컬처 무드보드 A/B/C/D 타입 러프 제작 (완성 시안이 아닌 방향성 확인용)
- [ ] 기존 구두 제안 사례 정리 — "안 한 게 아니다"를 보여줄 수 있게 제안안 / 시도 방향 / 선택되지 않은 이유 포함
- [ ] 담당자 선에서 누락·미보고된 제안이 있는지 확인 (회의에서는 사실 위주로만 전달, 추정 언급 금지)

## 2. 디자인 / 폰트

- [ ] 오리진 / 몬길 통합 대응 방향 정리 (타이틀별 톤 차이 개별 확인)
- [ ] 해당 타이틀의 서비스 국가/지역 확인 (오리진은 일본 등 다국어 대응으로 특히 신중)
- [ ] 폰트 라이선스 리스크 및 사용 가능 폰트 범위 정리 (한글은 내부 판단 가능, 일본어·외국어는 확인 필요)
- [ ] 산돌 등 다국어 폰트 구독/구매 가능성 확인
- [ ] 레이아웃 사례 라이브러리 구축 계획 수립

## 3. 영상 / AI / IP

- [ ] 영상 피드백("레이아웃 비슷, 트랜지션 뻔함")에 대한 상세 질문 리스트 작성 — 팀장이 실무자 의견을 전달받아 말하는 구조일 수 있으므로 질문을 미리 준비
- [ ] 과거 메신저 / 논의 자료(1~2년 전) 존재 여부 확인
- [ ] IP 제한 프로젝트(킹스로드 등)의 사용 가능 소재 범위 정리
- [ ] AI 사용 가능 / 불가 범위 확인 — 비캐릭터 요소(배경·이펙트·카메라 무빙) 우회안은 꼼수로 보일 수 있으므로 반드시 사전 협의 후 진행
- [ ] 사이즈 베리에이션 가능 범위 및 검수 기준 정리
- [ ] 액션 영상 요청의 구체 의미 확인 (브랜딩 무드 vs 캐릭터 액션 구분)
- [ ] 드래곤볼 전투 영상 등 AI 액션 레퍼런스 확인 — 레퍼런스 가능 ≠ 해당 IP 적용 가능, 라이선스·IP·AI 조건 별도 확인

## 4. 선제 기획 / AI 리포트

- [ ] 6월 기준 대기 / 리서치 / 아이데이션 가능 구간 확인
- [ ] AI 리포트에 넣을 선제 기획 사례 정리 (제작물 / SB / 콘셉트 보드 / 크리에이티브 레이어 샘플 선별)
- [ ] 선제안에 필요한 고객사 데이터 항목 정리 (기존 성과, 타이틀별 방향성, 선호 데이터)
- [ ] ⚠️ "대기 TO 활용" 표현 조정 — 디자인·영상 파트는 실제 대기가 거의 없음. "업무 효율화를 위한 AI 리서치·아이데이션 병행" 톤으로 통일하고, 대기 TO가 없을 때도 추가 제작을 약속하는 표현은 금지
- [ ] 리포트 구조: 이번 달 리서치·아이데이션 구간 → 진행한 선제 기획 → 산출물 예시 → 고객사 확인 필요 사항

## 5. 프로세스 기준 정리

- [ ] 영상 직행 가능 수정 범위 / 디자인 검수 필요 범위 내부 기준 확정 후 공유 (담당자별 임의 처리 방지)
- [ ] 기존에 영상에서 바로 처리했던 사례 정리
- [ ] 짧은 영상 소재 제작 기간 단축 사례 정리 (당일 출고 사례 포함)
- [ ] 고객사에 확인할 "간단 수정" 기준 질문 준비

## 6. 커뮤니케이션 톤 가이드 (회의 시 유의)

- "우리는 할 수 있다"가 아니라 **"정확도를 높이기 위해 방향 정렬 시간이 필요하다"** 는 방향으로 설명
- 무드보드 단계는 "정답 시안"이 아니라 **감도를 빠르게 맞추기 위한 도구**임을 명확히
- 클라이언트 레퍼런스가 실제 서브컬처와 다를 수 있음 → 지적이 아니라 "기준을 함께 맞추자"는 제안으로 표현
- 담당자 미보고 추정, AI 항목의 모순(AI 금지 프로젝트에서 AI 액션 영상 요청) 등은 직접 언급하지 않고 질문으로 확인
- 추가 산출물은 확정 시안이 아닌 **방향성 탐색 / 선제안**임을 항상 전제
