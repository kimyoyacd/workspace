왼쪽: 현재 상세 페이지 도식화 (문제 포인트 화살표 표시)

→ "어떤 감각인가"가 "어떤 성분인가"보다 먼저 온다.

INSIGHT 3
글로벌 PB 성장세 (일본+125%, 미국+162%)를
지탱하는 건 상세 페이지의 언어를 초월한 "감각 전달력"이다.
→ 텍스트 없이도 사고 싶은 상세 페이지가 글로벌 경쟁력.

[비주얼 디렉션]
3개 인사이트를 세로 카드로 배치

### 2-1 · 웨이크메이크의 과제

```
[헤드라인]
웨이크메이크가 풀어야 할 것:
"퍼스널 컬러 전문성을 상세 페이지 안에서 직접 경험하게 하라."

[현재 상태]
✓ 소프트 블러링 아이 팔레트 — 3년 연속 어워즈
✓ 16가지 컬러 / 4가지 텍스처 타입
✓ 르세라핌 허윤진 화보 컷 보유
✓ 일본 큐텐 아이섀도우 카테고리 1위

[설계되지 않은 것]
"다채로운 컬러 레인지를 프로페셔널 무드로
 꽉 채워야 한다는 것" — 고주현 발언 기반

→ 16가지 컬러가 나열로 그쳐있다
→ 퍼스널 컬러 매핑이 상세 페이지에 없다
→ 텍스처 차이가 시각화되지 않는다

[우리의 설계 방향]
Color Speaks 컨셉:
퍼스널 컬러 매핑 × 텍스처 ID × 발색 시퀀스

[비주얼 디렉션]
좌: 현재 팔레트 상세 페이지 도식 (문제 포인트 3개 화살표)
우: 설계 방향 와이어프레임 스케치
담당자 발언(고주현) 인용구 박스 삽입
```

### 2-2 · 컬러그램의 과제

```
[헤드라인]
컬러그램이 풀어야 할 것:
"SKU가 많아도 도입부에서 멈추게 하고,
 IP 콜라보까지 시스템으로 대응해야 한다."

[현재 상태]
✓ 잔망루피 등 IP 콜라보 한정판 적극 운영
✓ 다채로운 컬러 라인업 보유
✓ MZ세대 공감대 높은 브랜드 무드

[설계되지 않은 것]
"트렌디하고 정리된 구성" — 윤진희 발언 기반

→ 일반 라인 상세 페이지 퀄리티 < IP 콜라보 상세 페이지
→ 콜라보마다 상세 페이지 톤&매너 달라져 브랜드 일관성 저하
→ 많은 SKU가 정리된 구성이 아닌 나열로 처리됨

[우리의 설계 방향]
Mood First 컨셉:
무드 게이트 × 컬러 큐레이션 × IP 통합 포맷 시스템

[비주얼 디렉션]
잔망루피 콜라보 vs 일반 제품 상세 페이지 품질 차이 도식화
담당자 발언(윤진희) 인용구 박스 삽입
```

PRECISION 코드 × 도구→결과 시퀀스 × 색조 감성 연결

[비주얼 디렉션]
도구 스펙 나열 현재 도식 vs 도구→결과 시퀀스 방향 대비

   → 자체 워딩 가이드 보유 (A-4 참조)

③ AI 활용 적극 허용
   "너무 열려 있다" — 이선민 발언
   → 촬영 소스 기반 AI 연출 컷 포함 운영

④ 월 정기 납품 체계
   SKU 신규 출시, 시즌 리뉴얼 대응
   → AI 생산 파이프라인으로 납품 사이클 최적화

[비주얼 디렉션]
