### 1-3 · Competitive Audit — 상세페이지 구조 진단

```
[헤드라인]
현재 올리브영 색조 PB 상세 페이지에는 5가지 개선 기회가 있습니다.
(Baymard Institute 이커머스 UX 연구 기준)

[현행 구조 도식]
┌─────────────────────────────┐
│  화이트 배경 제품 패키지 컷   │ ← 3초 후킹 실패
│  스와치 이미지 나열 (정적)    │ ← 텍스처 미구분
│  모델 착용 컷 1~2장          │ ← 퍼스널컬러 미연결
│  성분/스펙 텍스트             │ ← 서사 없는 나열
│  수상 배지                   │
└─────────────────────────────┘

[5가지 개선 기회 — 실증 근거 기반]

① 도입부 후킹 부재 (ATF Critical Gap)
   현재: 화이트 배경 제품 컷으로 시작
   경쟁사: 3CE 풀블리드+대형 타이포 / 롬앤 무드 배경+GIF
   근거: "고객이 3초 안에 핵심 파악 못하면 이탈"
         — 2026 상세페이지 트렌드 (draph.art)
   영향: 뷰티 리테일 트래픽 82%가 모바일 → 첫 화면 = 전부

② 발색·텍스처 정적 표현 (Sensory Gap)
   현재: 팔 스와치 사진 1~2장, 매트/시머/글리터 구분 없음
   경쟁사: 롬앤 GIF 텍스처 / NARS 인터랙티브 발색 전환
   근거: "After Feel이 구매 결정 1위" — K-뷰티 2026 트렌드
         리뷰어가 자체적으로 텍스처 분류하여 리뷰 (maedeia.com)
   영향: 공식 페이지의 역할을 리뷰가 대신 수행 중

③ 퍼스널컬러 매핑 부재 (Personalization Gap)
   현재: 16색 팔레트에 퍼스널컬러 정보 없음
   경쟁사: Fenty 쉐이드 파인더 / 화해 소비자 자체 태그
   근거: 화해·글로우픽에서 소비자가 "봄웜 추천" 자체 분류
         올리브영 "바이브 큐레이션" 정책과 불일치 (CJ뉴스룸)
   영향: "나에게 맞는 색이 뭐지?" → 상세페이지에서 답 없음

④ 브랜드간 비주얼 무차별 (Brand Identity Gap)
   현재: 3개 브랜드 동일 포맷, 동일 톤앤매너
   근거: 컬러그램 IP 콜라보 vs 일반 라인 품질 격차 극심
         담당자 윤진희: "트렌디하고 정리된 구성" 요구
   영향: 브랜드 충성도 형성 불가, 가격 비교로만 전환

⑤ 2026 트렌드 대응률 19%
   7대 트렌드 중 AI 이미지·초개인화·GIF 영상 = 모두 미적용
   발색 비교(F1)·WebP 변환(F6)만 부분 대응
   근거: 올리브영 테크블로그 기술 인프라는 준비됨
         → 콘텐츠 레이어의 감각 설계만 빈칸

[비주얼 디렉션]
좌측: 현행 상세페이지 섹션 구조 도식 (5개 문제 포인트 화살표)
우측: 경쟁사 상세페이지 섹션 구조 (3CE/롬앤/NARS 3분할)
하단: 5개 Gap을 레이더 차트로 시각화 (현재 vs 경쟁사 vs 목표)
```

### 1-4 · 핵심 인사이트

```
[헤드라인]
"색조 상세 페이지의 경쟁력은 정보량이 아니라 감각 밀도입니다."

[인사이트 3개]

INSIGHT 1 — 3초의 법칙
소비자는 상세 페이지에서 평균 3~5초 안에
"살 것인가"를 결정한다.
→ 3초 안에 감각이 전달되지 않으면 스크롤은 끝난다.
  출처: Baymard Institute 이커머스 UX 연구
  근거: "고객이 3초 안에 핵심을 파악하지 못하면 이탈" (2026 PDP 트렌드)
  현재: 올리브영 PB 상세페이지 도입부 = 화이트 배경 제품 컷

INSIGHT 2 — 감각이 구매를 결정한다
K-뷰티 2026 핵심 트렌드: Sensory Textures
"After Feel"이 구매 결정 1위 요인으로 부상.
→ "어떤 감각인가"가 "어떤 성분인가"보다 먼저 온다.
  근거: 화해·글로우픽에서 소비자가 텍스처·퍼스널컬러를 자체 분류
        → 리뷰어가 공식 페이지의 역할을 대행하는 구조적 문제

INSIGHT 3 — 감각은 글로벌 언어다
글로벌 PB 성장세 (일본+125%, 미국+162%)를
지탱하는 건 상세 페이지의 언어를 초월한 "감각 전달력"이다.
→ 텍스트 없이도 사고 싶은 상세 페이지가 글로벌 경쟁력.
  근거: 뷰티 리테일 82%가 모바일 (Salsify)
        리뷰 11~30건 보유 시 전환율 +68% (Salsify Beauty PDP Report)
  시사: 상세페이지 자체의 감각 밀도가 높으면 리뷰 의존도 감소

[비주얼 디렉션]
3개 인사이트를 세로 카드로 배치
각 카드에 숫자(01/02/03) + 한 줄 제목 + 설명 2줄 + 출처 각주
배경: 그라데이션 (브랜드 키 컬러 → 화이트)

### 2-1 · 웨이크메이크의 과제

```

[현재 상세페이지 구조 분석]

  현행 섹션               문제점                  경쟁사 Best Practice
  ─────────────────────────────────────────────────────────────────
  패키지 풀컷 (화이트)    후킹 없음               3CE: 풀블리드+여백+타이포
  16구 팔레트 탑뷰        컬러명 오버레이 없음      NARS: 컬러명 직접 삽입
  스와치 이미지 1~2장     텍스처 구분 불가          롬앤: GIF 질감 클로즈업
  모델 착용 컷            퍼스널컬러 미연결         Fenty: 쉐이드 파인더
  성분/수상 텍스트        감각 전달 실패           CT: 전문가 보이스

[소비자 Pain Point — 실증 근거]
  ① 화해·글로우픽 리뷰에서 소비자가 "봄웜 추천" "가을웜 뮤트" 자체 태그
     → 공식 페이지가 제공하지 않는 정보를 소비자가 직접 생산 중
  ② 리뷰어(maedeia.com)가 "매트 3개, 시머 5개, 글리터 4개" 자체 분류
     → 텍스처 ID를 리뷰어가 대신 정리하는 상황
  ③ "실물이 어떤 느낌인지 손으로 느껴질 것 같다" 경험 미달
     → After Feel 트렌드 대응 실패

[담당자 발언]
  "다채로운 컬러 레인지를 프로페셔널 무드로
