# S3 AI-Assisted Ideation — 올리브영 색조 PB 상세 페이지

> Stage: S3 AI-Assisted Ideation (Playbook v1.1)
> 기반: S2 브랜드 리서치 결과 + RFP + 설명회 녹취록
> 목적: "감각이 설계되지 않았다"를 푸는 컨셉 방향 20가지 발산
> → S4·S6에서 3안으로 수렴

---

## 이데이션 인풋 요약

```
해결해야 할 핵심 문제:
"노후화된 상세 페이지를 요즘 색조스럽게 변경해 보는 그런 감각이나 센스"
(이선민 직접 발언)

3개 브랜드의 공통 문제:
- P1 감각 설계 부재 (브랜드 무드 ≠ 상세 페이지 비주얼)
- P2 발색 비가시화 (스와치만 있고 실제 텍스처 없음)
- P3 서사 부재 (컷-설명 나열, 스토리 없음)
- P4 브랜드 구분 없음 (3개 브랜드가 동일 포맷)
- P5 모바일 미흡 (PC 기준 레이아웃)

브랜드별 고유 무드:
- 웨이크메이크: 컬러 전문성, 퍼스널 컬러, 생생한 발색
- 컬러그램: 플레이풀, MZ 감성, IP 콜라보, 소장 가치
- 필리밀리: 스마트 툴, 전문성, 일상 편의
```

## 컨셉 방향 20가지 — 발산

### 카테고리 A: 감각 시각화 (Sensation Visualization)

---

#### A-1. 텍스처 ID 시스템
**핵심 아이디어:** 모든 제품에 "텍스처 ID 카드"를 부여. 매트/시머/글리터/새틴/젤리 등 5가지 텍스처 타입을 아이콘+컬러코드로 시각화. 상세 페이지 첫 화면에서 텍스처 ID를 보여주면 소비자가 제품을 손에 들기 전부터 "어떤 느낌인지" 알 수 있다.

**웨이크메이크 적용:** 아이팔레트 16구 각각에 텍스처 ID 부여 → 16가지 감각의 지도
**컬러그램 적용:** 틴트 제형별 텍스처 ID → "글리치 핑크 = 젤리 타입 ID-03"
**필리밀리 적용:** 브러시 털 밀도/형태를 텍스처 ID처럼 시각화

**강점:** 브랜드 공통 시스템, 재사용 가능, 글로벌 통용

---

#### A-2. 발색 클로즈업 시퀀스 (Swatch to Skin Journey)
**핵심 아이디어:** 컬러 스와치 → 손등 발색 → 실제 눈/입술 적용 → 완성 룩 순서로 4단계 발색 여정을 보여준다. 롬앤의 강점을 흡수하되 퍼스널 컬러 필터를 추가.

**현재와의 차이:** 지금은 스와치만 있음. 이 방식은 "내 피부에 어떻게 보일까"를 직접 보여줌
**AI 활용:** 퍼스널 컬러 4계절 타입별 발색 시뮬레이션 이미지 AI 생성

---

#### A-3. 무드 팔레트 비주얼 (Mood-Color Mapping)
**핵심 아이디어:** 컬러를 기능으로 설명하지 않고 "무드"로 설명. 컬러그램의 "바이브 큐레이션" 전략 직결. 상세 페이지 첫 섹션이 제품 이미지가 아닌 "오늘 어떤 무드인가요?" 비주얼로 시작.

**3가지 무드 코드 예시:**
- Daily Mood: 자연스러운, 맑은, 가볍게
- Vibe-Up Mood: 생기있는, 눈길을 끄는, 존재감
- Night Mood: 깊은, 세련된, 은은하게 빛나는

---

#### A-4. 소재 클로즈업 극대화 (Material Fetish)
**핵심 아이디어:** 제품 표면, 파우더 텍스처, 틴트의 물성을 극단적으로 클로즈업. 손가락으로 터치하는 순간, 브러시가 파우더 표면을 지나는 장면 등 ASMR급 비주얼로 "감각적 욕망"을 자극.

**레이아웃:** 전체 폭 풀블리드 클로즈업 이미지, 텍스트 최소화
**AI 활용:** 실물 촬영 + AI 업스케일링으로 피부 질감 수준의 클로즈업 생성

---

#### A-5. Before/After 발색 레이어 (Layer by Layer)
**핵심 아이디어:** 메이크업을 레이어별로 쌓아가는 과정 시각화. 베이스 → 포인트 1 → 포인트 2 → 하이라이터 → 완성. 각 단계가 하나의 섹션이 되는 상세 페이지.

**필리밀리 특화:** "이 브러시로 레이어 1을 어떻게 올리는가" → 도구와 결과 연결

---

### 카테고리 B: 서사 구조 (Narrative Architecture)

---

#### B-6. Hook First — 도입부 3초 전략
**핵심 아이디어:** 상세 페이지 진입 후 3초 안에 "이 제품이 왜 나에게 필요한가"를 전달. 풀블리드 모델 화보 + 한 줄 카피 + 제품명으로 구성된 도입부. 이후 스크롤 내려가면서 이유가 펼쳐진다.

**카피 방향 예시 (웨이크메이크):**
"당신의 퍼스널 컬러는 이미 정해져 있다. 문제는, 그것을 증명하는 팔레트였을 뿐."

**모바일 최적화:** 세로 풀블리드 이미지 = 한 화면이 곧 첫 인상

---

#### B-7. 컬러 스토리텔링 (One Color, One Story)
**핵심 아이디어:** 컬러 하나에 이름·무드·계절·감정 한 줄을 붙인다. 롬앤 방식 심화. "fig afternoon"처럼 컬러명 자체가 상세 페이지의 카피가 되는 구조.

**컬러그램 적용 예시:**
- #14 잔망 핑크: "오늘 한 번쯤은, 과감하게"
- #07 땅콩 베이지: "아무것도 안 한 것 같은데, 다 한 것 같은"

---

#### B-8. 퍼스널 컬러 매핑 서사 (Find Your Color)
**핵심 아이디어:** 상세 페이지 안에 "내 퍼스널 컬러 타입은?" 비주얼 가이드를 넣는다. 봄웜/여름쿨/가을웜/겨울쿨 타입별로 추천 컬러를 보여주는 섹션. 소비자가 상세 페이지를 읽으면서 "이게 나야"를 경험.

**웨이크메이크 특화:** 퍼스널 컬러 전문 브랜드 포지션 직접 구현

---

#### B-9. "오늘의 룩" 시퀀스 (Today's Look)
**핵심 아이디어:** 상세 페이지가 "하나의 메이크업 튜토리얼"이 되는 구조. 클리오 방식 흡수. Step 1 → Step 2 → Step 3 → 완성 룩. 각 스텝에서 이 제품이 어떤 역할을 하는지 보여준다.

**필리밀리 특화:** 도구 + 제품 함께 보여줌 (필리밀리 브러시 + 웨이크메이크 섀도우)
**크로스 브랜드 기회:** 3개 브랜드 제품을 함께 사용하는 "올리브영 PB 완성 룩" 시퀀스

---

#### B-10. 역순 서사 (Result First)
**핵심 아이디어:** 완성된 메이크업 룩부터 먼저 보여주고, 역으로 "이 결과를 만든 제품은?" 순서로 상세 페이지를 구성. "이 룩이 탐난다 → 그래서 이 제품이 필요하다" 욕망 먼저, 이유는 나중.

**3CE 방식 흡수 + 강화:** 화보급 완성 룩 이미지 → 컬러 이름 → 단계별 역추적

---

### 카테고리 C: 시스템 디자인 (System & Structure)

---

#### C-11. 브랜드별 고유 감각 언어 코드 (Brand Sensation Code)
**핵심 아이디어:** 3개 브랜드가 각각 다른 "감각 코드"를 가지도록 시스템 설계.

```
웨이크메이크 감각 코드: SPECTRUM
→ 컬러 스펙트럼 전개가 핵심 비주얼 언어
→ 레이아웃: 수평 컬러 그라데이션 + 퍼스널 컬러 매핑

컬러그램 감각 코드: PLAY
→ 율동감 있는 레이아웃, 예상을 벗어나는 구성
→ 레이아웃: 비대칭 그리드 + 캐릭터 요소 + 팝업 느낌

필리밀리 감각 코드: PRECISION
→ 정밀하고 정돈된, 전문가의 도구 느낌
→ 레이아웃: 수직 정렬, 클린 여백, 기능 다이어그램
```

---

#### C-12. 컬러 선택 인터랙티브 섹션 시뮬레이션 (Color Selector UI)
**핵심 아이디어:** 상세 페이지 내 컬러 선택 섹션을 "인터랙티브 UI처럼" 디자인. 실제 클릭은 안 되지만, 컬러 칩을 탭/클릭하면 해당 컬러의 발색 이미지가 바뀌는 것처럼 보이는 정적 비주얼 레이아웃.

**NARS 방식 흡수:** 컬러 선택이 곧 시각적 경험이 되는 구조

---

#### C-13. 성분 시각화 시스템 (Ingredient Visualization)
**핵심 아이디어:** 성분을 텍스트 나열이 아닌 인포그래픽으로. "피부 장벽 강화 성분 3종", "퍼스널 컬러 발색 지속력 성분" 등 소비자 언어로 성분 편집.

**2026 트렌드 대응:** 스킨케어링 메이크업 트렌드 → 색조에도 성분 설명 필요

---

#### C-14. 글로벌 동시 설계 레이아웃 (Global-Ready Layout)
**핵심 아이디어:** 처음부터 한국어/영어/일본어 3개 버전이 동시에 나오도록 레이아웃을 설계. 텍스트 영역을 모듈화하여 언어만 교체해도 비주얼 무드가 유지되는 구조.

**설명회 기회 포착:** 윤진희 발언 "글로벌 버전도 있으면 좋겠다" → 제안서에서 우리 강점으로

---

#### C-15. 소셜 증명 통합 섹션 (Social Proof Integration)
**핵심 아이디어:** 인스타그램/틱톡 스타일의 비주얼을 상세 페이지에 임베드 형태로 디자인. "인플루언서 발색 컷" 섹션을 상세 페이지 하단에 격자형으로 배치. 실제 SNS 연결이 아닌 디자인으로 구현.

**설명회 근거:** 이선민 "인플루언서 컷" 언급 → 상세 페이지에 통합

---

### 카테고리 D: AI 활용 (AI-Powered)

---

#### D-16. AI 퍼스널 컬러 발색 시뮬레이터 (AI Swatch Simulation)
**핵심 아이디어:** AI로 4가지 퍼스널 컬러 타입(봄웜/여름쿨/가을웜/겨울쿨)의 피부 톤에 동일 제품이 어떻게 발색되는지 시뮬레이션 이미지를 생성. 상세 페이지에 "내 타입 선택 → 발색 미리보기" 섹션 삽입.

**웨이크메이크 강점 직결:** 퍼스널 컬러 전문 브랜드의 증명 수단
**AI 실물 삽입:** 과제 시안에서 직접 구현해 "우리는 AI를 말만 하지 않는다" 증명

---

#### D-17. AI 연출컷 생성 (AI Lifestyle Shots)
**핵심 아이디어:** 실제 촬영 예산 없이도, AI로 제품이 등장하는 라이프스타일 컷 생성. 햇빛 아래 테이블 위의 컬러그램 틴트, 드레싱 테이블 위의 필리밀리 브러시 세트 등.

**설명회 근거:** 이선민 "AI 활용 너무 열려 있다" → 실물 삽입으로 증명
**제안서 전략:** 과제 시안에 AI 생성 연출컷 실제 삽입

---

#### D-18. 컬러 무드 AI 생성 배경 (AI Mood Background)
**핵심 아이디어:** 제품 컬러와 동일한 톤의 배경을 AI로 생성. "잔망 핑크 틴트" → 핑크 빛 꽃잎이 흩날리는 배경. 촬영 세트 비용 없이 무드 있는 상세 페이지 비주얼 구현.

**컬러그램 특화:** IP 콜라보 시즌 한정판 상세 페이지에 즉각 적용 가능

---

#### D-19. AI 메이크업 Before/After (Virtual Try-On Visual)
**핵심 아이디어:** AI로 동일 모델의 민낯 → 메이크업 완성 Before/After를 생성. 실제 촬영 없이도 제품의 변신 효과를 극적으로 보여주는 상세 페이지 섹션.

**주의:** 실제 제품과 AI 시뮬레이션임을 명기 필요 (소비자 신뢰)

---

#### D-20. 상세 페이지 모듈 자동화 시스템 (AI-Powered Page Assembly)
**핵심 아이디어:** 새 제품 출시 시 AI가 기본 정보(제품명, 컬러, 텍스처, 성분)를 입력받아 상세 페이지 초안을 자동 생성. 디자이너는 수정·정제만. 매달 반복되는 상세 페이지 제작 공정을 60% 단축.

**설명회 근거:** 매달 정기 발주 예정 → AI 자동화 시스템이 경쟁우위
**제안서 전략:** Part 4 Execution에서 이 시스템을 "월별 납품 체계"로 제시
