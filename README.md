# 커리어·인생 운영 시스템 (Career & Life OS)

> 디자인/콘텐츠/크리에이티브 분야 중간~시니어급을 위한, Claude와 함께 운영하는 살아있는 성장 시스템.

이 저장소는 한 번 읽고 끝나는 조언 모음이 아닙니다. **시간이 지날수록 데이터가 쌓이고, 방향이 또렷해지는 운영 체계**입니다.

## 시작하는 법

1. **`01-profile.md`를 채운다** — 분석의 모든 출발점. 비어 있으면 조언도 일반론에 그칩니다.
2. **`02-direction.md`를 함께 읽고 토론한다** — 어느 트랙으로 갈지가 모든 결정의 기준점.
3. **매주 일요일/매월 말, 회고를 남긴다** — 5분이면 충분. 쌓이면 패턴이 보입니다.

## Claude와 일하는 법

- "이번 주 회고하자" → 주간 템플릿으로 대화 시작
- "방향 다시 점검하자" → `02-direction.md` 기반 재분석
- 막히는 결정이 있을 때 → 그냥 물어보세요. 이 시스템의 맥락을 갖고 답합니다.
