## 1. 회사 소개 & 핵심 역량

### 구성
1. **한눈에 보는 수행사** — 설립·인원·주요 클라이언트 1페이지 인포그래픽
2. **왜 우리인가 — 3가지 강점 카드**
   - **색조 감각**: 뷰티/색조 도메인 상세 페이지·비주얼 콘텐츠 수행 이력
   - **End-to-End**: 기획→촬영→디자인→모션(GIF)→번역 디자인 원스톱
   - **속도 & 유연성**: 30일 미만 리드타임 대응 체계, 베리에이션·배너 빠른 전개
3. **수행 실적 하이라이트** — 대표 3건 썸네일 + 한 줄 성과

> 사례 참조:
> - KT "수행 장점" (연도별 실전 경험 리스트)
> - 코웨이 BEREX (스크립트형으로 PM-디자인-모션 역할 분담 명시)

## 2. RFP 이해 & 프로젝트 개요

### RFP 핵심 요약 (=우리가 정확히 이해하고 있다는 신뢰 시그널)

| 핵심 업무 | 상세 페이지 전반(기획+촬영+디자인) / 디자인 온리 / 촬영 / 배너 / DA / 영문 번역 디자인 |

### 설명회에서 확인한 핵심 포인트 (차별화 무기)
- RA(규제심사) 기간 1~2주 → 내용 수정 대응 필수 (화장품법·광고법 워딩 검수)
- 발색 컷·모델 컷은 올리브영 콘텐츠팀이 제공 → 제품 연출 컷·GIF는 수행사 촬영
- AI 활용에 **적극 긍정적** → "AI로 연출 컷 자연스럽게 해주시면 환영"
- IP 콜라보 연 2~3회 (컬러그램) → 콜라보 디자인 대응력 필요
- 인플루언서 컷 수급 → 브랜드 무드와 충돌 없는 레이아웃 설계 중요
- 촬영 소품·모델료·헤어메이크업 실비는 별도 청구 가능

> 사례 참조: 42dot Design Strategy "Introduction" (프로젝트 정의·배경 정리 방식)

## 3. 색조 상세 페이지 시장 분석

### 3-1. 올리브영 상세 페이지 트렌드 리서치

설명회에서 언급된 레퍼런스 브랜드를 축으로 분석:

| 브랜드 | 특징 | 학습 포인트 |
|--------|------|------------|
| **3CE** | 여백 없이 꽉 채운 비주얼, 컬러 블록 레이아웃 | 색조 감각의 기준선 |
| **투쿨포스쿨** | 프레임 다양성, 인포그래픽 활용 | 정보+감성 균형 |
| **로지킨** | 인플루언서 컷 자연스러운 통합 | 모델 컷·브랜드 톤 조화 |
| **VDL** | 정리된 컷 구성, 과하지 않은 디자인 | 깔끔한 정보 전달 |
| **나르카** | 3D·GIF 활용, 시각화 중심 | 사용법·효능 직관 전달 |
| **메디큐브** | 효능 명확한 비주얼 증거 | 기능성 설득 구조 |

### 3-2. Competitive Audit — 현행 상세 페이지 진단

대상: 웨이크메이크·컬러그램·필리밀리 기존 온에어 상세 페이지
- **Features** (유지할 점): 제품 USP 전달, 브랜드 키 컬러 일관성
- **Problems** (개선 기회): 노후화된 레이아웃, 정보 밀도 vs 스크롤 피로, 발색 컷 활용도, 모바일 최적화

> 사례 참조:
> - 42dot Design Strategy: Research → Audit → Competitive → C&O → Approach 전체 흐름
> - Clay SDIJ: 사용자 관점 진단 (여기서는 "구매 고객 관점 진단"으로 변환)

## 4. 브랜드별 크리에이티브 전략

### 총괄 전략 프레임 — 3축

```
① Hook First     — 상단 3초 안에 스크롤을 멈추는 키 비주얼
② Color Speaks   — 발색·텍스처가 직접 말하는 구조 (카피 최소화, 비주얼 극대화)
③ Scroll to Cart — 정보 → 설득 → 전환까지 끊기지 않는 스토리 라인
```

> 사례 참조: 42dot Recruit "3축 UX 전략 (Exploration·Personalization·Communication)"의 프레임 구조 차용

---

### 4-1. 웨이크메이크 — "컬러로 나를 표현하는 프로 메이크업"

| 요소 | 전략 |
|------|------|
| **톤앤매너** | 힙하고 세련된 프로페셔널, 피나/페리페라처럼 너무 귀엽지 않게 |
| **레이아웃** | 여백 최소, 꽉 채운 구성 (3CE·투쿨포스쿨 참조) |
| **컬러 활용** | 다양한 컬러 레인지를 레이아웃 자체에 반영 — 컬러 블록 섹션 전환 |
| **프레임** | 단조로운 "컷-설명-컷-설명" 지양 → 다채로운 프레임 변주 |
| **핵심 디자인 요소** | 텍스처 컷 크리에이티브 배치, 눕기 컷 프레임 다양화, 인포그래픽 활용 |

### 4-2. 컬러그램 — "1그램의 컬러로 만나는 새로운 나"

| 요소 | 전략 |
|------|------|
| **톤앤매너** | 트렌디·영(14~32 타겟), 톡톡 튀지만 정리된 |
| **상단 전략** | SKU 많은 브랜드 → 상단 후킹으로 스크롤 유도 (포인트 비주얼 선행) |
| **IP 콜라보** | 소스 기반 자유 디자인 → 프로모션·기획전·브랜드관까지 확장 가능한 시스템 |
| **인플루언서 컷** | 브랜드 무드 충돌 없이 자연스러운 통합 레이아웃 (VDL·로지킨 참조) |
| **온고잉 vs 콜라보** | 두 가지 디자인 시스템: 상시 제품용 + 콜라보 시즌용 |

### 4-3. 필리밀리 — "다채롭게 채우고 가볍게 느끼자"

| 요소 | 전략 |
|------|------|
| **톤앤매너** | 깔끔하되 시각화 강화 (나르카 3D·GIF 참조) |
| **핵심 과제** | 사용법(HOW-TO) 직관적 전달 — VOC 핵심 요구 |
| **BnA 구조** | Before & After가 명확한 컷, 커팅해도 독립적으로 작동하는 모듈형 구성 |
| **효능 증거** | 메디큐브식 "이 제품이 뭘 해주는지" 명확한 비주얼 블록 |
| **브러시→속눈썹 확장** | 다양한 툴 카테고리에 유연하게 적용되는 상세 페이지 프레임 |

## 5. 수행 범위 & 업무 프로세스

### 5-1. 업무 범위 매핑

RFP 단가표 항목을 정확히 반영:

```
┌─────────────────────────────────────────────────────┐
│  A. 상세 페이지 제작 전반 (기획 + 촬영 + 디자인)     │
│     ├─ A1. 메인 상세 페이지 (신규 라인)              │
│     ├─ A2. 베리에이션 상세 페이지 (라인업 확장)       │
│     ├─ A3. 배너 제작 — 짧은 버전                     │
│     └─ A4. 배너 제작 — 긴 버전                       │
│                                                      │
│  B. 디자인 온리 (기획·촬영 소스 제공받아 디자인만)    │
│     ├─ B1. 상세 페이지 디자인 (메인)                 │
│     ├─ B2. 상세 페이지 디자인 (베리에이션)            │
│     ├─ B3. 배너 디자인                               │
│     └─ B4. DA 소재 제작                              │
│                                                      │
│  C. 촬영                                             │
│     ├─ C1. 기획 + 촬영                               │
│     ├─ C2. 촬영만 (기획안 제공 시)                    │
│     └─ C3. 하우투(HOW-TO) 영상 제작                  │
│                                                      │
│  D. 번역 디자인 (국문 → 영문 적용)                   │
│                                                      │
│  E. 라이센스 (국내 온·오프 + 글로벌 온라인 포함)     │
│                                                      │
│  F. 기타 — 촬영 실비 (소품·모델·헤어메이크업 별도)   │
└─────────────────────────────────────────────────────┘
```

### 5-2. 제작 프로세스 (표준 워크플로우)

```
[D-30] 제작 요청 수령
    ↓
[D-28] 제품 분석 · 경쟁 레퍼런스 · 브랜드 가이드 확인
    ↓
[D-25] 스토리라인 기획 · 키 비주얼 방향 · 촬영 플랜 · 카피 초안
    ↓
[D-22] 기획안 제출 → 색조 마케팅팀 검토·피드백
    ↓
[D-18] 촬영 진행 (제품 연출·GIF·필요 시 모델)
       + 올리브영 제공 소스(발색 컷·모델 컷) 수령
    ↓
[D-14] 디자인 1차 시안 (전체 레이아웃 + 인포그래픽)
    ↓
[D-11] 피드백 반영 → 디자인 2차
    ↓
[D-7~D-0] RA 검수 기간 (화장품법·광고법 워딩 수정 대응)
    ↓
[D-Day] 최종 납품 + 영문 번역 디자인 동시 진행
    ↓
[D+3] 영문 버전 최종 납품
```

> 사례 참조:
> - KT "수행 범위 End-to-End 3-Layer" → 여기서는 기획(전략) / 제작(촬영+디자인) / 납품(검수+번역) 3단
> - Clay SDIJ "단계별 리뷰" → 기획안 → 1차 시안 → RA 검수 3단 리뷰 구조

## 6. 인력 투입 계획

### 6-1. 투입 인력 구성

| 역할 | 인원 | 주요 업무 | 투입률 |
|------|------|----------|--------|
| **PM / 기획** | 1명 | 제작 요청 수령, 스토리라인 기획, 카피, 일정 관리, RA 대응 | 100% |
| **메인 디자이너** | 1명 | 키 비주얼, 메인 상세 페이지, 인포그래픽, 모션(GIF) | 100% |
| **서브 디자이너** | 1명 | 베리에이션, 배너, DA 소재, 영문 적용 | 80% |
| **포토그래퍼** (내부/외주) | 1명 | 제품 연출, 하우투 영상 촬영 | 건별 |
| **AI 크리에이터** (겸임 가능) | — | AI 연출 컷 생성, 소스 보정 | 건별 |

- 각 인력별: 이름 / 경력 연차 / 핵심 수행 프로젝트 3건 / 포트폴리오 링크
- **외주 인력 포함**: 프로필 + 포트폴리오 필수 (설명회 요구사항)

> 사례 참조:

## 7. 품질 관리 & 커뮤니케이션

### 7-1. 품질 관리 체계

| 단계 | 체크포인트 | 관리 방법 |
|------|-----------|-----------|
| 기획 | 브랜드 가이드 준수, USP 정확성 | 기획안 체크리스트 |
| 촬영 | 컬러 정확도, 올리브영 규격(W1020) | 촬영 가이드 + 사전 테스트 컷 |
| 디자인 | 브랜드 톤앤매너, 레이아웃 다양성 | 내부 크리에이티브 리뷰 |
| 검수 대응 | RA 수정(화장품법·광고법) | 법규 워딩 가이드 자체 보유 |

## 10. 과제 전략

### 과제 요건 (설명회 확인)
- **대상**: 웨이크메이크 / 컬러그램 / 필리밀리 중 택 1
- **범위**: 상세 페이지 전체가 아닌 **도입부 + 포인트 1** (전체의 약 1/3)
- **규격**: W1020, 높이 자유
- **촬영**: 별도 촬영 없이 제공 소스(구글 드라이브 링크)만 사용
- **권장**: 웨이크메이크 또는 컬러그램 (필리밀리는 "툴"이라 색조 감각 보여주기 어려움)

### 과제 브랜드 선택 가이드

| 브랜드 | 장점 | 리스크 |
|--------|------|--------|
| **웨이크메이크 (추천)** | 색조 감각·디자인 역량 극대화 가능, 다채로운 컬러 활용 | 디자인 요소 많이 필요 → 공수 높음 |
| **컬러그램** | 트렌디한 감각 어필, 상단 후킹 차별화 기회 | SKU 많은 브랜드 특성 이해 필요 |
| **필리밀리** | 깔끔한 시각화에 자신 있다면 차별화 | 색조 감각 평가에 불리할 수 있음 |

### 과제 구성 제안 (웨이크메이크 기준)

```
[도입부] — 1~2단락
  ├─ 키 비주얼 히어로 이미지 (제공 소스 + AI 연출 활용)
  ├─ 브랜드 메시지 + 제품명 + 한 줄 카피
  └─ 컬러 팔레트 쇼케이스 (다양한 컬러 레인지 한눈에)

[포인트 1] — 1단락
  ├─ 핵심 USP 인포그래픽 (텍스처·발색 비주얼 중심)
  ├─ 발색 컷 크리에이티브 프레임 (획일적 배치 지양)
  └─ 사용감/효능 비주얼 블록
```

**차별화 포인트**:
- 프레임 다양성: "컷-설명" 반복 지양, 레이아웃 변주
- 컬러 블록 전환: 섹션마다 브랜드 키 컬러 기반 배경색 변화
- AI 활용: 제공 소스로 AI 연출 컷 생성 → "촬영 없이도 이 수준" 증명
- 여백 없이 꽉 찬 구성: 3CE·투쿨포스쿨 레퍼런스 반영
