## S1 산출물 — One-Pager Brief

### 표면 요청 vs 진짜 요청

| | 내용 |
|---|---|
| **표면 요청** | "색조 PB 3개 브랜드 상세 페이지 제작해줄 협력사를 뽑겠다" |
| **진짜 요청** | "노후화된 상세 페이지를, 요즘 색조 감각으로 리빌드할 수 있는 팀인지 증명해라" |

설명회에서 이선민 마케터가 직접 말한 핵심:
> "저희가 넣었던 거는 조금 노후화된 상세 페이지를 넣은 부분 있어요.
> 그 노후화된 부분을 요즘 요즘 색조스럽게 조금 변경해 보는 그런 감각이나 센스를 좀 보면 어떨까"

→ **과제가 곧 제안서다.** 문서 설득보다 과제 디자인의 "감각 증명"이 당락을 가른다.

### 의사결정자 가설

| 인물 | 역할 | 관심사 |
|------|------|--------|
| **구매팀 김민지** | 계약·단가·정산 | 임의 예가 부합, 최저 단가 준용 수용 여부 |
| **색조 마케팅팀 이선민** | 실무 운영·평가 주도 | 디자인 감각, 색조 이해도, 브랜드 핏, AI 활용 역량 |
| **웨이크메이크 담당 고주현** | 브랜드 무드 판단 | 힙하고 세련된 프로페셔널, 다채로운 레이아웃 |
| **컬러그램/필리밀리 담당 윤진희** | 브랜드 무드 판단 | 트렌디·정리된 구성, IP 콜라보 대응력 |

### 성공 기준 3줄

1. 과제 시안에서 "이 팀은 색조를 안다"는 감각이 즉시 느껴진다
2. 기획→촬영→디자인→번역을 원스톱으로 돌릴 수 있는 체계가 보인다
3. 단가가 임의 예가 범위 안에 있다

## S1 추가 산출물 — The Real Question 슬라이드 초안

### 긴장 구조 — 3쌍

```
가진 것 (OY 색조 PB)              ×    일어나지 않는 것 (상세 페이지)
─────────────────────────          ─────────────────────────────
① 웨이크메이크·컬러그램·필리밀리       상세 페이지가 브랜드 무드를
   3개 브랜드, 올리브영 매출 상위        제대로 전달하지 못하고 있다
                                        → "노후화"

② 발색 컷·모델 컷·인플루언서 컷        이 소스들이 상세 페이지 안에서
   풍부한 비주얼 소스 확보                감각적으로 조합되지 않고 있다
                                        → "획일적 컷-설명 반복"

③ 글로벌 확장(일본·미국) 가속,          국문·영문 상세 페이지를
   올리브영 글로벌 몰 성장               동시에 감각적으로 운영할
                                        체계가 없다
```

### The Real Question

> **"왜 올리브영 색조 PB의 브랜드 자산은, 상세 페이지에서 작동하지 않는가?"**

### The Answer (25자 이내)

> **"감각이 설계되지 않았기 때문이다."**

→ 이 한 문장이 제안서 전체의 핵심 주장. 우리는 그 감각을 설계할 수 있는 팀이다.

## S4 초안 — Brand Platform Reframing (디렉터 검수 대기)

### Before / After / Bridge

| | |
|---|---|
| **Before** | "올리브영 색조 PB 상세 페이지 = 제품 정보 나열 페이지" |
| **After** | "상세 페이지 = 브랜드가 고객과 만나는 첫 번째 감각 경험" |

| **Reframe Statement** | "상세 페이지는 정보가 아니라 감각이다. 스크롤하는 3초 안에 브랜드의 결이 손끝에 닿아야 한다." |

## S4-B — Project Goal 3-Layer (Bridge: 감각)

| 레이어 | 목표 | 측정 방향 |
|--------|------|-----------|
| **User Goal** | 상세 페이지에서 브랜드의 감각이 즉시 느껴진다 — 스크롤을 멈추고, 발색을 상상하고, 장바구니에 담는다 | 상세 페이지 체류 시간, 스크롤 깊이 (OY 내부 데이터 기반 공동 설정) |
| **Business Goal** | 감각적 상세 페이지가 구매 전환으로 연결된다 — 정보 나열 대비 전환율 상승 | 상세→장바구니 전환율, 상세→구매 전환율 (OY 내부 데이터 기반) |
| **Brand Goal** | "올리브영 색조 PB = 상세 페이지까지 감각적인 브랜드"라는 포지션 확보 — 자사몰·글로벌 몰에서도 일관된 브랜드 경험 | 글로벌 몰 상세 페이지 통일성 (정성 평가) |

> KPI 수치는 기입하지 않는다. "OY 내부 데이터 기반 공동 설정" 전제.

## 과제 전략 (별도 제출 — 그러나 사실상 이것이 당락)

### 과제 = S6의 축소 실행본

설명회 코멘트 근거:
> "어쨌든 그걸 보고 어쨌든 저희랑 핏이 계속 맞아서 잘 할 수 있는지에 대해서 조금 보고 싶어서 요청드리는 부분" — 이선민

| 항목 | 내용 |
|------|------|
| **브랜드 선택** | **웨이크메이크** (색조 감각 극대화 가능) |
| **범위** | 도입부 + 포인트 1 (상세 페이지 1/3) |
| **규격** | W1020, 높이 자유 |
| **소스** | 제공 소스만 사용 (별도 촬영 없음) |
| **AI 활용** | 적극 — 제공 소스 기반 AI 연출 컷 포함 (설명회 "너무 열려 있다") |

### 과제 구성 — Hook First + Color Speaks 적용

```
[도입부 — Hook First]
  ├─ 키 비주얼 히어로: 제품 + 브랜드 키 컬러 풀블리드
  ├─ 브랜드 메시지 + 제품명 + 1줄 카피
  └─ 컬러 팔레트 쇼케이스 (다양한 컬러 레인지 한눈에)

[포인트 1 — Color Speaks]
  ├─ 핵심 USP → 발색·텍스처 비주얼 중심 (카피 최소)
  ├─ 발색 컷 프레임 변주 (획일적 배치 지양)
  ├─ AI 연출 컷 (제공 소스 기반)
  └─ 사용감/효능 인포그래픽 블록
```
