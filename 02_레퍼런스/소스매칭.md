# 소스 매칭 — 올리브영 색조 PB 상세페이지 입찰

> RFP_분석.md 기반. 제출물 4종(단가표·포트폴리오·인력투입계획서·과제) 중
> 포트폴리오와 과제에 활용할 내부 자산 + 외부 레퍼런스 매칭.

## 1. 내부 사례 매칭 (Drive @제안샘플 기준)

직접적인 뷰티/색조 상세페이지 사례는 미보유 → **인접 역량으로 구성**:

| 포트폴리오 섹션 | 활용 사례 | 어필 포인트 |
|---|---|---|
| 브랜드 톤앤매너 구현력 | FANLIGHT BI Renewal, 11ST BX, np_bx | 브랜드 아이덴티티 → 비주얼 시스템 전환 역량 |
| 감각적 비주얼·F&B/라이프스타일 | 제주맥주 Website Prototype | 제품 중심 비주얼 스토리텔링 |
| 콘텐츠 스토리라인 기획 | 42dot Renewal(스토리텔링), KT(콘텐츠 전략) | 기획→촬영→디자인 통합 수행 |
| 대형사 협업 신뢰도 | KT, 카카오, 11번가, 코웨이, CJ 계열 경험 | "주요 브랜드 협업 경험" 평가항목 직격 |
| 디바이스·디테일 품질 | 코웨이 BEREX GUI | 정밀 비주얼 품질 관리 |
| **AI 활용 제작물 (별도 섹션 권장)** | AI 목업·모티브 스튜디오 실험 (Notion 1주차 'AI로 목업 이미지 제작' 등) | 담당자가 명시적으로 환영 — 차별화 카드 |

운영 역량 어필: 월 10건+ 물량의 안정 운영 → KT/42dot의 운영방안 모듈(프로세스·QC·리스크 대응) 재사용.

## 2. 외부 레퍼런스 — 색조 상세페이지 벤치마크

담당자가 직접 언급한 레퍼런스 (과제 제작 시 최우선 분석 대상):

| 브랜드 | 참고 포인트 | 경로 |
|---|---|---|
| **3CE** | 웨이크메이크 지향 — 꽉 찬 구성, 색조 표현 | 올리브영몰 / stylenanda 상세페이지 |
| **투쿨포스쿨** | 웨이크메이크 지향 — 다채로운 프레임 | 올리브영몰 상세페이지 |
| **로지킨, VDL** | 컬러그램 지향 — 인플루언서 컷과 무드 조화, 정리된 구성 | 올리브영몰 상세페이지 |
| **나르카** | 필리밀리 지향 — 3D, 시각화된 사용법 | 올리브영몰 상세페이지 |
| **메디큐브** | 필리밀리 지향 — 효능 명확한 컷 구성 | 올리브영몰 상세페이지 |
| 페리페라 | 웨이크메이크가 **지양**하는 무드 (키치·귀여움) | 반면교사 |

추가 수집 채널:
- Behance: `cosmetic detail page`, `beauty product page korea` 검색
- Pinterest: 상세페이지 / 뷰티 디테일페이지 보드
- 올리브영몰 색조 랭킹 상위 브랜드 상세페이지 (롬앤, 클리오, 데이지크 등)

- 발주처 제공: 구글 드라이브 (메일 배포 예정 — **수신 확인 필수**)
- 기존 온에어 상세페이지: 올리브영몰에서 과제 품목 검색

  - 컬러그램 음영 창조 라이너
