# 색조 상세페이지 디자인 무드

## 무드 정의

> **"노후화된 페이지를 '요즘 색조스럽게'."**
> 발주처가 과제로 보고 싶은 것은 단 하나 — 같은 USP·같은 카피로
> 디자인만으로 시대감을 바꿔내는 색조 감각.
> 평가자의 표현: "컷 하나-설명 하나-컷 하나-설명 하나"의 1차원 배치를 벗어나
> **프레임을 다채롭게, 여백 없이 꽉 차게, 그러나 정돈되게.**

---

## 1. 공통 규격 (과제 필수)

```
캔버스 폭: 1020px (올리브영 규격, 고정)
높이: 자유
범위: 상단 임팩트 도입부 + 정보성 POINT 1 (전체의 1/3 수준)
제출: PDF
```

---

## 2. 브랜드별 무드 — 웨이크메이크 (1순위 권장)

> "컬러로 나를 표현하는 데일리 프로 메이크업"
> **힙하고 세련된 프로페셔널.** 키치·귀여움(페리페라류) 금지.

### 컬러 시스템

```
--wm-bg-deep:    #111111   /* 도입부 다크 배경 — 프로페셔널 무드의 축 */
--wm-bg-light:   #F5F3F0   /* 정보 섹션 배경: 웜 뉴트럴 */
--wm-ink:        #1A1A1A   /* 본문 텍스트 */
--wm-ink-invert: #FFFFFF   /* 다크 배경 위 텍스트 */
--wm-accent:     제품 키 컬러에서 추출 (팔레트 도미넌트 셰이드)
--wm-line:       rgba(0,0,0,0.12)
```

- **제품 컬러가 곧 액센트** — 별도 브랜드 컬러를 만들지 말고, 해당 호수의 발색 톤(예: 소프트 블러링 = 뮤트 핑크브라운 계열)을 배경 그라데이션·타이포 포인트로 끌어온다
- 도입부는 다크 또는 딥톤 배경으로 임팩트 → POINT 1부터 라이트 배경 전환 (명도 대비로 섹션 리듬)
- 채도 높은 원색 금지 — "프로 메이크업" 무드는 뮤트·딥 톤이 기준

### 타이포그래피

```css
/* 헤드라인 — 도입부 */
font-family: 'Pretendard', sans-serif;
font-weight: 800; letter-spacing: -0.04em;
/* 영문 포인트 — 제품명·BLURRING 등 */
font-family: 'Helvetica Now', 'Inter', sans-serif;
text-transform: uppercase; letter-spacing: 0.06em;
```

| 역할 | 크기(1020px 기준) | 비고 |
|---|---|---|
| 도입 헤드카피 | 64–80px | 2줄 이내, 발색 키워드 강조 |
| 섹션 타이틀(POINT 1) | 40–48px | 영문 레이블 + 국문 조합 |
| 본문 | 26–30px | 모바일 가독 기준 |
| 캡션·호수명 | 20–22px | All-caps 영문 + 자간 넓게 |

### 레이아웃 패턴

```
[도입부]
풀블리드 KV (다크/딥톤) → 제품 비스듬히 크게, 그림자·반사 활용
헤드카피 오버레이 → 16색 팔레트면 컬러칩을 그리드가 아닌 리듬감 있는 배열로

[POINT 1]
좌우 분할·대각 배치·오버랩 프레임 혼용 — 동일 패턴 2회 연속 금지
텍스처 컷(스와이프·크러쉬)은 프레임 밖으로 흘리거나 타이포와 겹쳐서
```

- **꽉 채우되 정돈**: 여백으로 고급스러움을 내지 말고, 밀도 + 정렬로 세련됨을 낸다 (3CE·투쿨포스쿨 방식)
- 텍스처 컷·누끼를 "획일적으로 붙이지 말 것" — 회전, 크롭, 프레임 침범으로 변화
- GIF 포인트: 발색 그라데이션, 팔레트 오픈, 텍스처 스와이프 중 1곳

## 3. 브랜드별 무드 — 컬러그램 (대안)

> "단 1그램의 컬러로 만나는 새로운 나" — **플레이풀, 14~32세 트렌드.**
> 단, SKU가 많아 **상단 후킹 → 정리된 전개**가 생명.

### 컬러 시스템

```
--cg-bg:      #FFFFFF / #FAF7F4   /* 라이트 베이스 — 제품 컬러가 주인공 */
--cg-accent:  제품 셰이드 (음영 라이너 = 브라운 그라데이션 20/30/70%)
--cg-point:   비비드 1색만 (스티커·뱃지용)
```

- 음영 창조 라이너처럼 "단계(20%→70%)"가 USP인 제품은 **그라데이션 바·퍼센트 타이포**를 핵심 그래픽 장치로
- 스티커·라벨·손글씨 요소는 1섹션 1개까지 — 플레이풀하되 산만 금지 (로지킨·VDL의 "정리된 컷" 기준)

### 레이아웃 패턴

- 상단 후킹: 제품 한 컷 + 한 문장 (스크롤 끊김 방지가 최우선, SKU 나열은 후반으로)
- 인플루언서/모델 컷은 브랜드 무드와 충돌하지 않게 — 보정 톤을 제품 컬러와 맞추고, 프레임·여백 처리로 통일
- 호수별 비교는 표가 아니라 **컬러칩 + 발색 swatch 정렬**로

## 4. 공통 컴포넌트 스펙

### USP 강조 (KPI 배너 대응)
```
[ 큰 숫자/키워드 ]  예: "1g" "16 COLORS" "20%→70%"
숫자 64px+ 단독 강조, 설명은 그 아래 26px
색조에선 수치보다 컬러·발색이 KPI — 숫자 대신 컬러칩 배열도 가능
```

### 발색컷 영역 (브랜드 제공 소스)
- 발색컷은 **무보정·무합성 원본 사용** (AI 생성·과도한 보정 절대 금지 — 컴플라이언스)
- 발색컷 주변에만 충분한 여백 — 색 정보가 정확히 읽히게

### GIF/모션
- 1과제 내 1~2개 — 텍스처 스와이프, 발색 전환, 사용법 루프
- 효능·발색을 과장하는 모션 금지 (RA 검수 기준)

## 5. Do / Don't

| Do | Don't |
|---|---|
| 제품 발색 톤을 페이지 전체 팔레트로 확장 | 브랜드와 무관한 장식 컬러 추가 |
| 프레임 변화 (오버랩·대각·크롭) | 컷-설명-컷-설명 1차원 반복 |
| 다크 도입부 → 라이트 전개의 명도 리듬 | 전체 다크 (정보 가독성 저하) |
| AI 연출컷 활용 (배경·소품·무드컷) | **발색컷 AI 생성/보정** |
| 제공 카피·USP 유지 + 디자인으로 승부 | 카피 대폭 변경 (화장품법 리스크) |
| 영문 레이블 절제 사용 (힙 무드) | 과도한 영문 도배 (가독성·RA 리스크) |

## 6. 과제 품목 메모

| 브랜드 | 품목 | 디자인 출발점 |
|---|---|---|
| 웨이크메이크 | 소프트 블러링 아이팔레트 (16색, 뮤트 톤) | "블러링" = 소프트 포커스 그래픽 모티프, 16색 칩 리듬 배열 |
| 컬러그램 | 음영 창조 라이너 (20/30/70% 음영) | 퍼센트 단계 그라데이션을 핵심 비주얼 장치로 |
| 필리밀리 | 쿠션 팡팡 퍼프 | 비권장 (툴 브랜드 — 색조 감각 평가 불리, 담당자 언급) |
